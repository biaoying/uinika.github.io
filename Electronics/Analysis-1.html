电路分析涉及的理论与定理 -『线性电路篇』

发表于 2020-09-19 分类于 Electronics 阅读次数： 本文字数： 75k 阅读时长 ≈ 1:08

电路分析系列文章主要讨论电路当中的电磁现象及其发生过程，其理论与工程实践紧密相连，分析过程中需要注意电路模型与实际电路、测量值与计算值的差别与联系。此外，由于每个电路模型都具有特定的工作条件，因此需要根据不同的工作场景选择不同的电路模型。『线性电路篇』主要介绍由线性元件、独立源、线性受控源所构成电路的基本知识，是后续动态电路分析、模拟电子技术的先导内容。

本文主要包括了基本概念与两类约束、电路的等效变换、电阻电路的分析方法、常用电路定理等线性电路分析方向的内容，同时也介绍了非线性电阻电路的基本分析。由于电路分析技术本质是数学在电路当中的应用，因此学习过程需要注重将电路分析方法与数学知识相结合。全文撰写过程当中，参考并借鉴了国外大学所普遍采用的《Electric Circuits》第 10 版教材。

电路模型

电路是为了特定需求，将实际元件连接起来所构成的电流通路。一个完整的电路通常由电源（电池）和负载（灯泡）以及连接用的导线与开关组成。

实际电路当中的电子元件在工作时，其电磁特性较为复杂，例如灯泡在电源作用下会发光消耗电能（电阻特性），与此同时还会在周围产生磁场（电感特性），这样的特性可以采用一个电阻元件 \(R\) 和电感元件 \(L\) 的串联来表征。但如果仅仅考虑其消耗电能的主要特性，而忽略其它次要性质，那么就可以仅采一个电阻元件来进行表征。

类似的，蓄电池也可以采用一个电压源和电阻的串联来进行表征，连接的导线用理线导线（线段）来表示，这样就可以得到如下的电路模型：

之所以要引入电路模型的概念，是因为实际元器件在电压和电流的作用下，往往涵盖了消耗电能、储存电能、储存磁场能这样一些基本的特性。在一定条件下，可以将实际器件进行理想化，将各个特性分开来进行考虑，分别采用理想电阻元件、理想电容元件、理想电感元件来进行表征；将这些理想元件连接起来，就得到了理想电路模型。

综上所述，电路模型是在集总参数假设条件下，对实际电路的科学抽象与近似描述，是电气系统的近似数学模型。其中，集总参数假设是指实际电路尺寸远远小于其工作信号的波长；满足集总参数假设的理想电路元件就称为集总参数元件，其具有较为单一的电磁特性：

根据电路物理尺寸是否远小于其传输信号的波长，可以将其分为集总参数电路和分布参数电路两种类型。

【例题】工作频率为50Hz的电网传输线是否满足集总参数假设？

【解答】其对应波长为：\(\lambda = \frac{波速}{频率} = \frac{3 \times 10^8}{f} = 6000km\)，因此对于物理尺寸小于6000km的传输线，是可以满足集总参数假设的。

【例题】电视信号传输线的工作频率可以达到 \(10^8Hz\)数量级，如果某个无线电视频道的传输信号工作频率约为 \(200MHz\)，那么 \(0.5m\) 的传输线路是否满足集总参数假设？

【解答】通过计算可以得到其对应波长为 \(1.5m\)，因此 \(0.5m\) 的传输线路并未远远小于这个波长，因此不属于集总参数电路，而属于分布参数电路。

基本物理量

电荷

电荷分为正负电荷两种极性，采用英文字母Q/q表示，基本单位为库仑C，电荷的定向运动形成电流。

电荷量是一个离散的量，其值为电子所携带电荷量 \(1.6022 \times 10^{-19}\) 库仑的整数倍。

电流

电流是单位时间内通过导体横截面的电荷量，用英文字母I/i表示，基本单位为安培A。

\[ i = \frac{dq}{dt} = \frac{库仑}{秒} \]

直流电流 DC 的大小和方向都不随时间变化，用大写字母 I 表示；而交流电流 AC 是方向随时间变化的电流，采用小写字母 i 表示。其中较为常见的是正弦交流电，即电流随着时间按照正弦规律变化的电流：

电路当中，习惯性将正电荷移动的方向规定为电流的实际方向，电流方向可以采用箭头和双下标两种方式进行标注：

对于较为复杂不易确定电流方向的电路，或者流过元件的电流为方向不断变化的交流时，电路分析引入了参考方向的概念。参考方向是一个人为任意指定的方向，例如：指定如下两个元件的参考方向都是从a 指向b，然后对电路进行分析求解；如果求得电流 \(i>0\)，说明电流的实际方向与参考方向一致；如果求得电流 \(i<0\)，则说明电流实际方向与参考方向相反；

【例题】（1）已知 \(i=1A\)，试标记出电流的实际方向？（2）已知 \(i=2 \sin (100 \pi t + \frac{3}{2} \pi)\) 安培，求解 \(t=2\) 秒时的实际电流方向？

【解答】（1）由于 \(i\) 为正值，所以电流的实际方向是由a指向b的。（2）因为 \(t=2\) 秒，\(i=2 \sin (100 \pi \times 2 + \frac{3}{2} \pi) = -2\) 安培，所以电流的实际方向与参考方向相反，即由b指向a。

电压

电路当中a、b两点之间的电压，是指单位正电荷从a点移动到b点电场力所做的功，通常用字母U(直流)/u(交流)表示，单位为伏特V：

\[ u_{ab} = \frac{dw}{dq} = \frac{焦耳}{库仑} \]

电压的实际方向是电位降低的方向，即从高电位指向低电位。而电压的参考方向与电流类似，也可以通过人为进行设定，根据该电压参考方向进行分析计算之后，如果计算的结果电压 \(u>0\)，就说明实际电压方向与参考方向一致：

【例题】已知a点到b点的电位降是6V，即 \(U_{ab}=6V\)，当取参考方向不同时，电压 \(U\) 的值分别为多少？

【解答】如果指定的参考方向由a指向b，由于参考方向与实际方向相同，因此 \(U_1 = 6V\)。如果这里设定的参考方向由b指向a，则由于参考方向与实际方向相反，此时的 \(U_2 = -6V\)。

电位

电场力将单位正电荷从电路某点移动至参考点（人为选定，电位为0伏，用符号 \(\bot\) 表示）所作的功就定义为该点的电位，通常采用英文字母\(V\) 表示，其单位为伏特V。

下图当中，可以任意选择a、b、c、d为参考点，假设选择了d点作为参考点，那么a点的电位 \(V_a\) 就是 \(U_{ad}\)，b点的电位 \(V_b\) 就是 \(U_{bd}\)，c点的电位 \(V_c\) 就是 \(U_{cd}\)。

如果 \(U_{bd}\) 是从 b 点到达 d 点的电位降，则可以得到如下等式：

\[ U_{bd} = U_{bc} + U_{cd} \implies U_{bc} = U_{bd} - U_{cd} = V_b - V_c \]

通过上式可以知道，电路当中任意两点的电压，其实就是这两点之间电位的差值，因此电位都是相对于参考点而言的。

【例题】已知 \(U_{s1}=6V\) 和 \(U_{s2}=3V\)，试计算 \(U_{ab}\) 处的电压值？

【解答】上图左右两个电路唯一的区别在于，所选取参考点的位置不同。

对于左图而言，c点的电位为零参考点，即 \(V_c = 0V\)；由于a点的电位比c点高出6V，因此a点的电位 \(V_a = 6V\)；相应的b点的电位为\(V_b=-3V\)，由此可以计算得到 \(U_{ab}\) 是a点和b点的电位差：

\[ U_{ab} = V_a - V_b = 6V - (-3V) = 9V \]

对于右图而言，b点为零电位参考点，即 \(V_b = 0V\)；由于c点的电位比b点高3V，因此c点的电位 \(V_c = 3V\)；而a点电位在c点基础上，再高出一个6V，因此 \(V_a=9V\)；此时的 \(U_{ab}\) 仍然是a点与b点的电位差：

\[ U_{ab} = V_a - V_b = 9V - 0V = 9V \]

通过上面这个例题可以发现，电路当中各点电位与参考点的选取有关。但是最终求得的 \(U_{ab}\) 都等于9V，也就说明电压与参考点的选取无关。

之前讨论电压和电流这两个基本物理量的时候，分别引入了参考方向的概念。电压参考方向和电流参考方向都是任意假定，并且两者相互独立。但是对于电路当中的相同元件而言，电压与电流的参考方向无外乎两种情况：

一种是电流的流向与电位降的方向相同，称为关联参考方向；另一种是电流流向与电位升方向相同，称为非关联参考方向。

【例题】判断下图当中的元件A和元件B的电压和电流是否为关联参考方向？

【解答】对于元件A，电流流向与电位降方向相同，所以是关联参考方向。而对于元件B，电流方向与电位升方向一致，所以是非关联参考方向。

功率

电流是指单位时间内，从a点移动到b点的正电荷数量，即 \(i = \frac{dq}{dt}\)。电压是指单位时间以内，单位正电荷从a点移动到b点所做的功，即 \(u=\frac{dw}{dq}\)。由此可以发现，电压和电流都是正电荷从a点移动到b点方向，它们的实际方向都是一致的。

将电压和电流结合起来，就可以引入一个新的基本变量功率，即单位时间内电场力所做的功，采用英文字母p表示，基本单位为瓦特W：

\[ p(t) = \frac{dw}{dt} = \frac{dw}{dq} \times \frac{dq}{dt} = ui \]

无论参考方向是否关联，只要功率的计算结果大于零，就意味该元件属于吸收（消耗）功率的负载，否则元件就是发出（产生）功率的电源：

  • \(p>0\) 元件是吸收（消耗）功率的负载；
  • \(p<0\) 元件是发出（产生）功率的电源；

【例题】 设 \(I=1A\)、\(U_1=10V\)、\(U_2=6V\)、\(U_3=4V\)，试计算元件 A、B、C 的功率？

【解答】

  • 元件 A 处于非关联方向，\(P_1=-U_1I=-10×1=-10W\)，由于 \(P_1<0\)，所以是产生 \(10W\) 功率，属于电源；
  • 元件 B 处于关联方向，\(P_2=U_2I=6×1=6W\)，由于 \(P_2>0\)，所以是吸收 \(6W\) 功率，属于负载；
  • 元件 C 处于关联方向，\(P_3=U_3I =4×1=4W\)，由于 \(P_3>0\)，所以是吸收 \(4W\) 功率，属于负载；

通过观察可以发现，上述元件产生的功率为 \(10W\)，而吸收的功率也是 \(6W+4W=10W\)，说明该电路产生的总功率等于吸收的总功率，这就是任何电路都会遵循的功率守恒原则。

能量

电场力在一段时间内所做的功就称为能量，其基本单位为焦耳（牛顿/米）J。已知电压的计算公式为 \(u = \frac{dw}{dq}\)，由此可以推导得到 \(dw = u \times d_q\)，对这个等式左右两侧进行积分，就可以得到能量 \(w\) 是 \(u\) 对电荷量 \(q\) 的一个积分。又由于 \(d_q = i \times d_t\)，所以能量的公式可以记为电压 \(u\) 和电流 \(i\) 对时间 \(t\) 的积分。

\[ w(t) = \int\limits u(t) \cdot dq(t) = u \cdot i \cdot d_t \]

电力系统当中，通常使用千瓦时（即1度市电）这个能量单位，其换算关系如下所示：

\[ 1 度电 = 1kW \cdot h = 1000W \times 3600S = 3.6 \times 10^6 J \]

电阻

理想电阻元件是从实际电阻器抽象而来的理想化模型，其主要特性是消耗电能，然后以热能的方式释放掉。其电压电流关系，可以采用 u-i 平面上的一条曲线来描述，这条曲线就称为伏安特性曲线。

如果一个电阻元件的伏安特性曲线是过原点的一条直线，就称为线性电阻元件，用英文字母 R 来表示，基本单位为欧姆Ω，其电路符号如下图所示：

伏安特性曲线的斜率等于电阻值，这就是欧姆定律，它体现了电阻对电流的阻碍作用，采用数学表达式可以表示如下：

\[ R = \frac{u}{i} \implies u=Ri \]

当电流流过电阻，电阻就会消耗能量，因此沿电流流动的方向就会出现电压降，该电压降的大小等于 \(u=Ri\)，即电流与电压降方向一致，电阻上的电压 \(u\)、电流 \(i\) 处于关联参考方向：

如果线性电阻上的电压 \(u\) 和电流 \(i\) 为非关联参考方向，此时电阻两端的电压 \(u=-Ri\)。

电阻元件除使用电阻R作为参量进行表征以外，还可以使用电导来进行表征。电导采用英文字母 G 来表示，单位为西门子S，其值为电阻的倒数：

\[ G = \frac{1}{R} \implies i = \frac{u}{R} = G \cdot u \]

接下来讨论线性电阻的功率，当电阻元件的电流和电压处于关联参考方向时，此时功率的计算公式为：

\[ p = ui = (R \cdot i) \cdot i = R \cdot i^2 \geq 0 \]

当电阻元件的电流、电压处于非关联参考方向时，此时功率计算公式如下所示：

\[ p = -ui = -(R \cdot i) \cdot i = R \cdot i^2 \geq 0 \]

由于线性电阻元件的阻值 R 为一个正实数，所以功率 \(p \geq 0\) 恒为一个非负值，因此正电阻可以视为一个无源元件。接下来，讨论电阻吸收的能量，在关联参考方向下，能量是关于功率对时间t的积分，由于其计算结果大于零，因此线性电阻元件是一个耗能元件。

\[ w = \int^{t}_{-\infty} uidt = R \int^{t}_{-\infty} i^2dt = G \int^{t}_{-\infty} u^2dt \geq 0 \]

【例题】阻值为2Ω的电阻，其电压电流处于关联参考方向，已知电阻上电压为 \(u(t) = 4 \cos t\) 伏，求流过该电阻的电流 \(i(t)\) 以及消耗的功率 \(p(t)\)？

【解答】由于参考方向是关联的，根据欧姆定律，电流 \(i(t) = \frac{u(t)}{R} = \frac{4 \cos t}{2Ω} = 2 \cos t\) 安。而功率 \(p(t) = Ri^2 = 2Ω (2 \cos t)^2 = 8 \cos^2t\) 瓦。

上面讨论的线性电阻元件，其伏安特性曲线是一条经过原点的直线，被称为线性时不变电阻。除此之外，伏安特性曲线的斜率随时间变化而变化的电阻，则称为线性时变电阻。

电容

电容器由 2 块面积为 \(A\) 的金属极板和介电常数为 \(\varepsilon\) 的中间介质构成，在 2 块金属极板上外加电压源 \(U\)，就会在两块金属极板上存储等量异种电荷 \(+q\) 和 \(-q\)，从而在中间介质上建立了一个电场。当外部电压源撤走以后，两块金属极板上存储的电荷会因为电场力互相吸引，但是由于中间介质绝缘所以不能被中和，金属极板之间存储的电荷量会长久保存下去，因而电容器实际上是一个存储电荷的元件。

电容 C 的大小由电容极板的表面积 \(A\)、极板之间的间隔距离 \(d\)、介电常数 \(\varepsilon\) 三者共同决定。

\[ C = \frac{\varepsilon A}{d} = \frac{介电常数 \times 极板表面积}{极板间隔距离} \]

理想电容元件的定义为任意时刻 \(t\)，一个二端元件所贮存的电荷 \(q\) 与两端电压 \(u\) 可以使用 u-q 平面上的一条曲线进行描述。

\[ C = \frac{q}{u} = \frac{电荷}{电压} \]

如果这条曲线为过原点的一条直线，就称它为线性时不变电容（下图左），否则就为非线性时不变电容（下图右）。

伏安关系-微分

前面对于电容的分析，主要基于其储存电荷量q和施加电压u的角度，但是电路分析当中更加关心的是伏安关系。在电压、电流关联参考方向下，可以得到如下推导过程：

\[ \begin{cases} C = \frac{q}{u} \\ i=\frac{dq}{dt} \end{cases} \implies i = C\frac{du}{dt} \]

通过上面的推导可以得出结论：流经电容的电流与该时刻电压的变化量成正比，即电压 \(u\) 发生变化时才能产生电流 \(i\)，如果电压 \(u\) 不发生变化，则电流 \(i\) 等于零，此时电容元件在直流电路当中相当于开路，电容元件具有隔断直流的作用。

伏安关系-积分

已知电流 \(i = C\frac{du}{dt}\)，那么电压 \(u\) 就可以记为电流 \(i\) 对时间 \(t\) 的积分。

\[ u(t) = \frac{1}{C} \int^{t}_{-\infty} idt \]

上式说明 \(t\) 时刻电压取决于之前电流值，因此电容元件具有记忆性，是一种记忆元件。假设当前起始时间为 \(t_0\)，那么 \(u(t)\) 就可以记为 \(C\) 分之1负无穷到 \(t_0\) 的 \(i\) 对时间 \(t\) 的积分加上 \(C\) 分之1的 \(t_0\) 到 \(t\) 的 \(i\) 对时间 \(t\) 的积分。

\[ u(t) = \frac{1}{C} \int^{t}_{-\infty} idt + \frac{1}{C} \int^{t}_{t_0} idt \]

由于前面已知 \(u(t_0) = \frac{1}{C} \int^{t}_{-\infty} idt\)，所以上式最终可以写作如下形式：

\[ u(t) = u(t_0) + \frac{1}{C} \int^{t}_{t_0} idt \]

如果起始时刻为零时刻，则上式可以继续写作下面的格式：

\[ u(t)=u(0) + \frac{1}{C} \int^{t}_{0} idt \]

根据上面这一系列的推导，可以知道电容是一种记忆元件，其电压具有连续性。

储能

接下来讨论电容的储能，电容的电压 \(u\) 和电流 \(i\) 在关联参考方向下可以得到如下等式：

\[ \begin{cases} p = ui \\ i = C\frac{du}{dt} \end{cases} \implies p = C \cdot u\frac{du}{dt} \]

由于能量是关于功率的积分，那么可以继续推导出下面的等式：

\[ W_c = \int^{t}_{-\infty} C \cdot u \frac{du}{dt} \cdot dt = \frac{1}{2} C \cdot u^2(t) - \frac{1}{2} C \cdot u^2(-\infty) \]

\(u(-\infty)\) 时刻的电压可以认为等于0，因此 \(t\) 时刻的储能 \(W_c\) 应等于：

\[ W_c = \frac{1}{2} C \cdot u^2 (t) \]

上述等式说明 \(t\) 时刻的储能，与此时电压 \(u\) 的平方成正比，是一个非零值，电容是一个储存电场能的元件。接下来讨论从 \(t0\) 到 \(t\) 时间过程当中，电容储能的变化量：

\[ \Delta W_c = \int^{u(t)}_{u(t_0)} C \cdot u(t) \cdot {du}(t) = \frac{1}{2} Cu^2(t) - \frac{1}{2} Cu^2(t_0) \]

如果从 \(t0\) 到 \(t\) 时刻，电容两端电压增大，这意味着电容储存的能量也在增大，电容处于充电状态。相反的，如果电容 \(u(t)\) 时刻电压小于 \(u(t_0)\)，那么电容在这个过程当中存储的能量在减少，处于放电状态。电容在充放电过程当中，即不消耗能量也不产生能量，因此它是一个无源元件。

综上所述，这里对上述所分析的电容元件性质做一个小结：

  1. 电容的伏安关系为 \(i = C \cdot \frac{du}{dt}\)；
  2. 电容元件对于直流而言是开路的；
  3. 电容元件的充放电都需要经历一个时间过程，因此电容上的电压不会突然发生改变；
  4. 理想电容不消耗能量，是一个储能元件，所储存的能量为 \(W_c = \frac{1}{2} C \cdot u^2(t)\)；

伏安关系-微分

当流过线圈的电流发生变化，那么线圈当中的磁通链也会发生变化，根据电磁感应定律，此时线圈两端就会产生感应电压。在电感的电压、电流处于关联参考方向时，电压 \(u\) 就是磁通链 \(\Psi\) 对时间 \(t\) 的微分：

\[ \begin{cases} L = \frac{\Psi}{i} \\ u = \frac{d \Psi}{dt} \end{cases} \implies u = L \frac{di}{dt} \]

上面等式说明在任何时刻 \(t\)，电感元件的电压 \(u\) 取决于该时刻电流 \(i\) 的变化率，而与该时刻的电流 \(i\) 本身无关。也就是说如果电流 \(i\) 是一个不变化的常量，那么电压 \(u\) 等于0，即电感元件对于直流量相当于短路。

伏安关系-积分

如果 \(i\) 是变化的，就会产生感应电压 \(u\)，又由于电压 \(u = L \frac{di}{dt}\)，因此电流 \(i\) 就是电压 \(u\) 关于时间 \(t\) 的积分。

\[ i(t) = \frac{1}{L} \int^{t}_{-\infty} u(t) \cdot dt \]

这说明任意时刻的电流 \(i\) 记载了 \(t\) 时刻以前所有的电压值，因此电感是一个记忆元件。假设当前起始时刻为 \(t_0\)，此时 \(i_t\) 可以写作 \(L\) 分之1负无穷到 \(t_0\) 时刻 \(u\) 对时间 \(t\) 的积分，加上 \(L\) 分之1的 \(t_0\) 到 \(t\) 时刻 \(u\) 对时间 \(t\) 的积分。

\[ i(t) = \frac{1}{L} \int^{t_0}_{-\infty} u(t) \cdot dt + \frac{1}{L} \int^{t}_{t_0} u(t) \cdot dt \]

由于 \(i(t_0) = \frac{1}{L} \int^{t_0}_{-\infty} u(t) \cdot dt\)，因此上面的等式可以写作下面形式：

\[ i(t) = i(t_0) + \frac{1}{L} \int^{t}_{t_0} u(t) \cdot dt \]

如果起始时刻为零时刻，那么上式可以进一步列写为：

\[ i(t) = i(0) + \frac{1}{L} \int^{t}_{t_0} u(t) \cdot dt \]

根据电流 \(i\) 和电压 \(u\) 的积分关系，可以知道电流 \(i\) 是处处连续的，因此电感的电流具有连续性。

储能

已知能量是功率对时间的积分，在电压电流关联参考方向下 \(p=ui\)，而根据 \(u = L \frac{di}{dt}\)，

\[ \begin{cases} p=ui \\ u = L \frac{di}{dt} \end{cases} \implies p = L \cdot i \cdot \frac{di}{dt} \]

能量 \(w\) 就是功率 \(p\) 对时间 \(t\) 的积分，可以写作负无穷到 \(t\) 时刻 \(L \cdot i\) 乘以 \(di\) 比 \(dt\) 再乘以 \(dt\)，通过计算可以得到如下推导：

\[ W_L = \int^{t}_{-\infty} Li \frac{di}{dt} dt = \frac{1}{2} Li^2(t) - \frac{1}{2} Li^2(-\infty) \]

上面等式最左侧的负无穷时刻电流 \(i\) 可以视为0，那么就可以得到关于能量 \(W_L\) 的公式：

\[ W_L = \frac{1}{2}Li^2(t) \]

也就是说任意时刻电感储存的能量，与该时刻电流 \(i\) 的平方成正比，其值大于等于零，因此电感是一个储能元件。接下来，考虑时间 \(t_0\) 到 \(t\) 时刻，电感储能的变化量：

\[ \Delta W_L = \int^{i(t)}_{i(t_0)} Li(t) \cdot di(t) = \frac{1}{2} Li^2(t) - \frac{1}{2} Li^2(t_0) \]

如果 \(i(t) > i(t_0)\)，则上式说明在 \(t_0\) 到 \(t\) 时刻的时间范围内，电感储存的能量在增加，否则说明电感元件在释放能量。由此可知 \(t_0\) 到 \(t\) 时刻，电感元件即不产生能量，它也不消耗能量，因此电感是一个无源元件。

最后，我们对电感元件的性质进行一个简单的小结：

  • 电感元件的伏安关系在关联参考方向下 \(u = L \frac{di}{dt}\)，即变化的电流才会产生电压，而对于直流而言相当于短路；
  • 理想电感元件不会消耗能量，是一个储能元件 \(W_L = \frac{1}{2}Li^2(t)\)；
  • 由于能量的增加和释放都需要一定时间，因此电感上的电流处处连续，不能发生突变；

伏安关系总结

电路分析当中涉及的 3 种无源电路元件电阻、电容、电感到这里就介绍完毕了，可以发现每种元件的电压、电流都拥有各自相应的元件伏安关系。

  • 电阻元件的伏安关系 \(u = R \cdot i\) 符合欧姆定律，有电压就会有电流；
  • 电容元件的伏安关系 \(i = C \cdot \frac{du}{dt}\) 是变化的电压产生电流；
  • 电感元件的伏安关系 \(u = L \cdot \frac{di}{dt}\) 是变化的电流产生电压；

开短路问题

理想电压源 \(U_s\) 与电阻 \(R_L\) 构成一个回路，如果开路就意味着电流 \(I = 0\)；如果短路则意味着端口电压 \(U = 0\)，这与电压源的特性相矛盾，因而将独立电压源短路没有意义。

理想电流源 \(I_s\) 与电阻 \(R_L\) 构成一个回路，当电流源两端短路时，端电压 \(U = 0, I = I_s\)；如果开路则意味着电流源的端口电流 \(I_s = 0\)，这与电流源的特性相矛盾，因此将独立电流源开路没有意义。

功率

观察上述四种受控源的电路符号，会发现它们的控制端钮不是开路就是短路（对应控制量为开路电压或短路电流），同时电压和电流都是关联参考方向。因而在关联参考方向下，受控源的功率可以记为:

\[ p = u_1 i_1 + u_2 i_2 = u_2 i_2 \]

基尔霍夫电流定律 KCL

基尔霍夫电流定律：集总参数电路当中的任意结点，在任意一个时刻，流入该结点的电流代数和为零。

\[ \sum_{k=1}^{n} i_k = 0 \]

上式当中，可以假设流入结点的电流为正+，流出结点的电流为负-，当然也可以作出相反的假设。

【例题】求解流过下图当中结点a的电流的代数和？

【解答】根据 KCL 可知流过结点a的电流代数和为零 \(-i_1 + i_2 + i_3 - i_4 = 0\)，将方程变换后得到 \(i_2 + i_3 = i_1 + i_4\)，可以发现方程中 \(i_2\) 和 \(i_3\) 是流入结点a的电流，\(i_1\) 和 \(i_4\) 是流出结点a的电流。

对于上面示例中的结点a而言，流入电流之和等于流出电流之和，这样就可以得到基尔霍夫电流定律的第 2 种表达：任意一个时刻，流入任意一个结点的电流之和等于从该结点流出的电流之和。

注意：基尔霍夫电流定律本质上是一种电路的电流约束，而且这样的约束是线性的，电路分析当中称之为线性相关。

等效变换的概念

按照元件的参数特性，电路可以分为时变电路和非时变电路。按照输入输出关系，则可以分为线性电路和非线性电路。如果电路当中全部有源元件和无源元件都属于线性时不变元件，那么该电路就称为线性时不变电路。如果这个线性时不变电路当中的所有无源元件都是电阻，那么这个电路就称为线性电阻电路。线性电阻电路的独立源可以是直流也可以是交流，如果所有的独立源都是直流，则该电路就是直流电阻电路。如果没有特殊说明，本文讨论的都是直流电阻电路。

根据基尔霍夫电流定律，一个端子流入的电流，等于另一个端子流出的电流。

有了二端网络的概念，就可以将上面电路视为两个二端网络 \(N_1\) 与 \(N_2\) 的联结，2 个网络都包含有独立源，这种包含有独立源的二端网络就称为有源二端网络，否则称之为无源二端网络。二端网络电路分析当中，端口电流和电压的约束关系称为二端网络的伏安关系。

将电路视为二端网络 \(N_1\) 和 \(N_2\) 的联结之后，可以将 \(N_2\) 部分电路等效为一个电压源和电阻的串联，这个串联电路可以记为 \(N_3\)。换而言之，就是使用 \(N_3\) 代替了 \(N_2\)，使得 \(a\) 和 \(b\) 两端的伏安关系与 \(a^{'}\) 和 \(b^{'}\) 两端的伏安关系完全相同，那么电路分析当中就认为两者是等效的。

如果二端网络 \(N_1\) 的伏安关系和另一个二端网络 \(N^{'}\) 的伏安关系完全相同，则可以认为二端网络 \(N_1\) 和 \(N^{'}\) 等效。

注意：二端网络的等效是对于网络外部等效，而非对于网络内部的等效。

电阻的串并联等效

电阻的串联

电阻的串联是指 \(N\) 个电阻首尾相连，中间无任何分支；串联电路当中，各个电阻上通过的电流 \(i\) 相同，总电压或端口电压 \(u\) 等于各个电阻上的电压之和 \(u = u_1 + u_2 + ... + u_n\)，根据欧姆定律可以进一步表示为：\(u = (R_1 + R_2 + ... + R_n)i\)。

\(N\) 个电阻的串联是一个无源二端网络，而无源二端网络总可以等效为一个等效电阻 \(R_{eq}\)。根据欧姆定律，\(R_{eq}\) 的伏安关系为 \(u = R_{eq}i\)。

如果上述两个电路等效，那么就意味着其伏安关系相同，由此可以得到：

\[ R_{eq} = R_1 + R_2 + ... + R_n = \sum_{i=1}^{n} R_i \]

每个电阻上的电压 \(u_k\)，等于电流 \(i\) 乘以该电阻 \(R_k\)，又由于 \(u = R_{eq}i\)，联立可以得到如下方程组：

\[ u_k = \begin{cases} u_k = R_k i = \\ i=\frac{u}{R_{eq}} \end{cases} \implies u_k = \frac{R_k}{R_{eq}} u \]

可以看到，\(N\) 个电阻串联是在对端口总电压 \(u\) 进行分压，而且串联之后的等效电阻值 \(R_{eq}\) 大于任意一个串联电路上的电阻。

电阻的并联

电阻的并联是指 \(N\) 个电阻首尾分别连接在一起形成公共端，串联电路上每个电阻的电压 \(u\) 都相等，端口电流 i 等于各个电阻上的电流之和 \(i = i_1 + i_2 + ... i_n\)，根据欧姆定律可以记为 \(i = {\frac{u}{R_1} + \frac{u}{R_2} + ... \frac{u}{R_n}}\)。除此之外，还可以将电阻 \(R_n\) 以电导 \(G_n\) 的形式来进行表示 \(i = (G_1 + G_2 + ... + G_n)u\)。

\(N\) 个电阻并联同样是一个无源二端网络，也同样可以等效为一个电阻 \(R_{eq}\)。根据欧姆定律，\(R_{eq}\) 的伏安关系为 \(i = \frac{u}{R_{eq}} = u \cdot G_{eq}\)。

根据等效的定义，上述两个二端网络具有完全相同的伏安关系，这意味 \(N\) 个电阻并联的等效电阻的倒数等于每个并联电阻的倒数之和：

\[ \frac{1}{R_{eq}} = \frac{1}{R_1} + \frac{1}{R_2} + ... + \frac{1}{R_n} = \sum_{i=1}^{n} \frac{1}{R_i} \]

上述方程也可以列写为等效电导 \(G_{eq}\) 的形式：

\[ G_{eq} = G_1 + G_2 + ... + G_n \]

从上述公式可以知道，\(N\) 个电阻并联之后的等效电阻 \(R_{eq}\) 小于任意一个并联电路上的电阻阻值。

电阻并联以后，流过每个电阻的电流 \(i_k\)，会对端口总电流 \(i\) 形成分流：

\[ \frac{i_k}{i} = \frac{u/R_k}{u/R_{eq}} = \frac{G_k}{G_{eq}} \implies i_k = \frac{G_k}{G_{eq}} \cdot i \]

比如有 2 个电阻并联时，每个电阻所分得的电流系数等于：分母为两个电阻之和，分子为另外一条支路的电阻。

\[ \begin{cases} i_1 = \frac{G_1}{G_{eq}} \cdot i = \frac{1/R_1}{1/R_1 + 1/R_2} \cdot i = \frac{R_2}{R_1 + R_2} \cdot i \\ i_2 = \frac{G_1}{G_{eq}} \cdot i = \frac{1/R_2}{1/R_1 + 1/R_2} \cdot i = \frac{R_1}{R_1 + R_2} \cdot i \end{cases} \]

电流表量程切换

【例题】电流表的表头等效电阻 \(R_g=1KΩ\)，满刻度量程 \(Ig=100μA\)，如果需要测量 \(1mA\)、\(10mA\)、\(100mA\) 的电流，分流电阻 \(R1\)、\(R2\)、\(R3\) 的阻值应该分别是多少？

【解答】满刻度量程指的是表头电流 \(I_g\) 为满刻度 \(100μA\) 的时候，被测量电流 \(I\) 能够达到 \(1mA\)、\(10mA\)、\(100mA\)。

首先，当接线柱 5 连接到接线柱 1 的时候，电阻 \(R_g\) 与 \(R_3\) 和 \(R_2\) 串联，从而对总电流 \(I_1\) 进行分流：

\[ I_g = \frac{R_1}{R_1 + R_2 + R_3 + R_g} I_1 \implies I_1 = \frac{R_1 + R_2 + R_3 + R_g}{R_1} I_g \]

然后，当接线柱 5 连接到接线柱 2 的时候，电阻 \(R_g\) 与 \(R_3\) 相串联，\(R_1\) 和 \(R_2\) 相串联，然后对总电流 \(I_2\) 进行分流：

\[ I_g = \frac{R_1 + R_2}{R_1 + R_2 + R_3 + R_g} I_2 \implies I_2 = \frac{R_1 + R_2 + R_3 + R_g}{R_1 + R_2} I_g \]

同理，当接线柱 5 连接到接线柱 3 的时候，电阻 \(R_1\)、\(R_2\)、\(R_3\) 先串联再与 \(R_g\) 并联，然后对总电流 \(I_3\) 进行分流：

\[ I_g = \frac{R_1 + R_2 + R_3}{R_1 + R_2 + R_3 + R_g} I_3 \implies I_3 = \frac{R_1 + R_2 + R_3 + R_g}{R_1 + R_2 + R_3} I_g \]

接下来，就可以开始对 \(R_1\)、\(R_2\)、\(R_3\) 的阻值分别进行求解：

当接线柱 5 连接到接线柱 3 的时候，满量程为 \(1mA\)：

\[ I_3 = \frac{R_1 + R_2 + R_3 + R_g}{R_1 + R_2 + R_3} I_g \implies 1mA = \frac{R_1 + R_2 + R_3 + 1000Ω}{R_1 + R_2 + R_3} 100 \mu A \implies R_1 + R_2 + R_3 = 111.1Ω \]

当接线柱 5 连接到接线柱 2 的时候，满量程为 \(10mA\)：

\[ I_2 = \frac{R_1 + R_2 + R_3 + R_g}{R_1 + R_2} I_g \implies 10mA = \frac{R_1 + R_2 + R_3 + 1000Ω}{R_1 + R_2} 100 \mu A \implies R_1 + R_2 = 11.1Ω \]

联立上述两个推导结果方程可以得到：

\[ \begin{cases} R_1 + R_2 + R_3 = 111.1Ω \\ R_1 + R_2 = 11.1Ω \end{cases} \implies R_3 = 100Ω \]

当接线柱 5 连接到接线柱 1 的时候，满量程为 \(100mA\)：

\[ I_1 = \frac{R_1 + R_2 + R_3 + R_g}{R_1} I_g \implies 100mA = \frac{R_1 + R_2 + R_3 + 1000Ω}{R_1} 100 \mu A \implies R_1 = 1.1Ω \]

联立接线柱 2 的方程，就可以得到如下的推导过程：

\[ \begin{cases} R_1 + R_2 = 11.1Ω \\ R_1 = 1.1Ω \end{cases} \implies R_2 = 10Ω \]

最后，就可以求得电流表里分流电阻的阻值分别为：\(R1=1.1Ω\)、\(R2=10Ω\)、\(R3=100Ω\)。

为了与之前得到的三角形伏安关系表达式保持一致，需要将电流 \(i_{1Y}\)、\(i_{2Y}\)、\(i_{3Y}\) 作为待求变量，分别用电压 \(u_{12Y}\)、\(u_{23Y}\)、\(u_{31Y}\) 进行表示。此时 3 个变量对应 2 个表达式，还需要再补充一个 KCL 方程：

\[ i_{1Y} + i_{2Y} + i_{3Y} = 0 \implies i_{1Y} + i_{2Y} = -i_{3Y} \]

将上式中的 \(i_{1Y} + i_{2Y} = -i_{3Y}\) 代入至 \(u_{23Y} = R_2 i_{2Y} - R_3 i_{3Y}\)，可以得到 \(u_{23Y} = (R_2 + R_3) i_{2Y} + R_3 i_{1Y}\)，将这个式子与之前列写的 \(u_{12Y} = R_1 i_{1Y} - R_2 i_{2Y}\) 联立，然后在两个等式的左右侧分别乘以 \(-R_3\) 和 \(R_1\)，从而可以得到如下推导过程：

\[ \begin{cases} u_{12Y} = R_1 i_{1Y} - R_2 i_{2Y} \\ u_{23Y} = (R_2 + R_3) i_{2Y} + R_3 i_{1Y} \end{cases} \implies \begin{cases} u_{12Y} \times (-R_3) & = [R_1 i_{1Y} - R_2 i_{2Y}] \times (-R_3) \\ u_{23Y} \times (R_1) & = [(R_2 + R_3) i_{2Y} + R_3 i_{1Y}] \times (R_1) \end{cases} \]

\[ \implies \begin{cases} -R_3 u_{12Y} & = -R_1 R_3 i_{1Y} + R_2 R_3 i_{2Y} \\ R_1 u_{23Y} & = R_1 (R_2 + R_3) i_{2Y} + R_1 R_3 i_{1Y} \end{cases} \]

将上面这个联立方程组的等号左右侧分别相加，最终得到如下等式：

\[ R_1 u_{23Y} - R_3 u_{12Y} = R_2 R_3 i_{2Y} + R_1 (R_2+R_3) i_{2Y} \]

根据上面这个表达式，就可以将 \(i_{2Y}\) 求解出来：

\[ i_{2Y} = \frac{u_{23Y} R_1 - u_{12Y} R_3}{R_1R_2 + R_2R_3 + R_3R_1} \]

再根据前面已经列写的方程 \(u_{12Y} = R_1 i_{1Y} - R_2 i_{2Y}\)，以及 KVL 方程 \(u_{12Y} + u_{23Y} + u_{31Y} = 0\)，就可以求得 \(i_{1Y}\) 的表达式：

\[ i_{1Y} = \frac{u_{12Y} R_3 - u_{31Y} R_2}{R_1R_2 + R_2R_3 + R_3R_1} \]

上述步骤求解得到 \(i_{1Y}\) 和 \(i_{2Y}\) 的表达式之后，再根据 KCL 方程 \(i_{1Y} + i_{2Y} + i_{3Y}\)，以及 KVL 方程 \(u_{12Y} + u_{23Y} + u_{31Y} = 0\)，得到 \(i_{3Y}\) 的表达式：

\[ i_{3Y} = \frac{u_{31Y} R_2 - u_{23Y} R_1}{R_1R_2 + R_2R_3 + R_3R_1} \]

接下来，将经过上述推导得到的星形、三角形联结的伏安关系列写在一起进行比较分析：

\[ \Delta = \begin{cases} i_{1\Delta} = \frac{u_{12\Delta}}{R_{12}} - \frac{u_{31\Delta}}{R_{31}} \\ i_{2\Delta} = \frac{u_{23\Delta}}{R_{23}} - \frac{u_{12\Delta}}{R_{12}} \\ i_{3\Delta} = \frac{u_{31\Delta}}{R_{31}} - \frac{u_{23\Delta}}{R_{23}} \end{cases} \qquad Y = \begin{cases} i_{1Y} = \frac{u_{12Y} R_3 - u_{31Y} R_2}{R_1R_2 + R_2R_3 + R_3R_1} \\ i_{2Y} = \frac{u_{23Y} R_1 - u_{12Y} R_3}{R_1R_2 + R_2R_3 + R_3R_1} \\ i_{3Y} = \frac{u_{31Y} R_2 - u_{23Y} R_1}{R_1R_2 + R_2R_3 + R_3R_1} \end{cases} \]

结合等效的定义，可以分别列写出关于 \(R_{12}\)、\(R_{23}\)、\(R_{31}\) 的表达式：

\[ \begin{cases} R_{12} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_3} \\ R_{23} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_1} \\ R_{31} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_2} \end{cases} \]

最后，通过星形联结的电阻，得到转换为三角形联结之后的电阻。观察上面方程组的 3 个表达式，可以总结得到如下的规律：

\[ \Delta 型电阻 R_{mn} = \frac{Y 型电路电阻两两乘积之和}{Y 型不相邻电阻} \]

同样的，要实现三角形到星形联结的等效变换，根据等效的概念，可以通过推导得到下面方程组：

\[ \begin{cases} R_{1} = \frac{R_{12} R_{31}}{R_{12} + R_{23} + R_{31}} \\ R_{2} = \frac{R_{12} R_{23}}{R_{12} + R_{23} + R_{31}} \\ R_{3} = \frac{R_{23} R_{31}}{R_{12} + R_{23} + R_{31}} \end{cases} \]

根据上述方程组，同样可以总结得到星形联结转换为三角形联结时电阻等效的规律：

\[ Y 型电阻 R_{i} = \frac{\Delta 型所在端钮 i 上的两个电阻乘积}{\Delta 型三个电阻之和} \]

接下来，讨论一种较为特殊的情况。当三角形或星形联结当中 3 个电阻的阻值都相等时。如下图所示，三角形内部的 3 条线分别代表的是星形连接中的 3 个电阻 \(R_1\)、\(R_2\)、\(R_3\)，而三角形本身的 3 条边则分别代表的是三角形连接当中的 3 个电阻 \(R_{12}\)、\(R_{23}\)、\(R_{31}\)。

如果三角形联结当中的 3 个电阻相等 \(R_{12} = R_{23} = R_{31} = R\)，那么等效变换以后的星形联结 \(R_1\)、\(R_2\)、\(R_3\) 仍然相等，并且 \(R_i = \frac{R}{3}\)。同样的，如果已知星形联结当中的 3 个电阻相等 \(R_{1} = R_{2}= R_{3} = R\)，那么等效变换以后的三角形联结 \(R_1\)、\(R_2\)、\(R_3\) 同样相等，并且 \(R_{mn} = 3R\)。

\[ \begin{cases} 如果 R_{12} = R_{23} = R_{31} = R & \implies R_i = \frac{R}{3} \\ 如果 R_{1} = R_{2}= R_{3} = R & \implies R_i = 3R \end{cases} \]

由此可以观察到，在 \(Y\) 和 \(\Delta\) 联结变换的情况下，具有外大内小的特点：

\[ R_{\Delta} = 3 R_Y \]

电容的串并联等效

前面的内容讨论了电阻元件的串并联，以及电阻星形、三角形联结的等效变换，接下来讨论另外两种无源元件电容和电感的串并联联结。

电容串联等效

下图是 \(n\) 个电容的串联联结，流过每一个电容的电流都为 \(i\)；根据 KVL 定律，端口电压 \(u\) 为 \(N\) 个电容上电压的代数和：

\[ u = u_1 + u_2 + ... + u_n \]

已知电容元件的伏安关系为 \(i = C \frac{du}{dt}\)，即电容上的电压是关于电流的积分，从而可以得到这样的推导：\(C_1\) 分之一负无穷到 \(t\) 时刻 \(i\) 对时间 \(t\) 的积分，加上 \(C_2\) 分之一负无穷到 \(t\) 时刻 \(i\) 对时间 \(t\) 的积分，一直加到 \(C_n\) 分之一负无穷到 \(t\) 时刻 \(i\) 对时间 \(t\) 的积分：

\[ u = \frac{1}{C_1} \int^{t}_{-\infty} i(t)dt + \frac{1}{C_2} \int^{t}_{-\infty} i(t)dt + ... + \frac{1}{C_n} \int^{t}_{-\infty} i(t)dt \]

由于电容串联时，流过每个电容的电流 \(i\) 都是相同的，因而，可以进一步整理为：

\[ u = (\frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}) \int^{t}_{-\infty} i(t)dt \]

如果将 \(\frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}\) 记为 \(\frac{1}{C_{eq}}\)，则可以最终得到下面的等式：

\[ u = \frac{1}{C_{eq}} \int^{t}_{-\infty} i(t)dt \]

当 \(\frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}\) 等于 \(\frac{1}{C_{eq}}\) 的时候，两个二端元件都具有相同的伏安关系，因而可以认为是等效的。

由此可以得出结论：\(n\) 个电容串联之后容值的倒数，等于各个电容的倒数之和：

\[ \frac{1}{C_{eq}} = \frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n} = \sum_{k=1}^n \frac{1}{C_k} \]

电容并联等效

下图是 \(n\) 个电容的并联，每个电容都具有相同的端电压 \(u\)，根据 KCL，端口上的电流是 \(n\) 个电容上电流的代数和：

\[ i = i_1 + i_2 + ... + i_n \]

由于电流 \(i = C \frac{du}{dt}\)，代入上式可以得到：

\[ i = C_1 \frac{du(t)}{dt} + C_2 \frac{du(t)}{dt} + ... + C_n \frac{du(t)}{dt} \]

由于每一个电容上的电压 \(u\) 都是相同的，因此还可以表示为：

\[ i = (C_1 + C_2 + ... + C_n) \frac{du(t)}{dt} \]

如果现在有一个电容 \(C_{eq}\)，那么当 \(C_{eq} = C_1 + C_2 + ... + C_n\) 的时候，其伏安关系可以记为：

\[ i = C_{eq} \frac{du(t)}{dt} \]

说明 2 个二端元件都具有相同的伏安关系，因此也可以认为是等效的：

由此可知，\(n\) 个并联电容的等效电容值等于各个电容值之和：

\[ C_{eq} = C_1 + C_2 + ... + C_n \]

\[ \frac{1}{L_{eq}} = \frac{1}{L_1} + \frac{1}{L_2 } + ... + \frac{1}{L_n} \]

\[ u = u_{s1} = u_{s2} = u \]

只有电压值相等且极性一致的情况下，电压源的并联才会符合 KVL 定律。

\[ i_{s1} = i_{s2} \]

实际电源的等效变换

实际电压源

实际电压源的电路模型是由电压源和电阻的串联组成的，对于这个二端网络，其伏安关系可以记为：

\[ u = u_s - R_s \cdot i \]

这样的伏安关系，可以采用横坐标为电流 \(i\) ，纵坐标为电压 \(u\) 的平面直角坐标系描述如下：

当 \(i = 0\) 的时候，即二端网络的开路电压为 \(u_s\)；当 \(u = 0\) 的时候，即二端网络的短路电流为 \(\frac{u_s}{R_s}\)。两点确定一条直线，这条直线就是该二端网络的伏安关系曲线，这条曲线描述的是：实际电源当中的 \(R_s\) 不能够忽略，当其向外提供电能的时候，端口电压 \(u\) 减少的情况。

实际电流源

实际电流源的电路模型是由电流源和电阻并联组成，这个二端网络的伏安关系可以记为：

\[ i = i_s - \frac{u}{R_s^{'}} \implies u = {R_s^{'}} \cdot i_s - {R_s^{'}} \cdot i \]

这样的伏安关系，同样可以采用横坐标为电流 \(i\) ，纵坐标为电压 \(u\) 的平面直角坐标系进行描述：

当 \(i = 0\) 的时候，端口上的开路电压 \({R_s}^{'} \cdot i_s\)；当 \(u = 0\) 的时候，端口上的短路电流为 \(i_s\)。两点确定一条直线，即二端网络的伏安关系曲线，它描述的是实际电流源由于内阻的存在，导致对外提供的端口电流 \(i\) 减少的情况。

实际电压源和电流源的电路都无法再进一步化简，但是在二端网络伏安关系完全相同的情况下，两个电路可以进行等效互换，

\[ \begin{cases} u = u_s - R_s \cdot i \\ i = {R_s^{'}} \cdot i_s - {R_s^{'}} \cdot i \end{cases} \implies \begin{cases} R_s = {R_s}^{'} \\ u_s = R_s \cdot i_s \end{cases} \]

电流源电阻并联电路和电压源电阻串联电路的等效互换，对于电路的分析十分有帮助。值得注意的是，进行两种电路的等效互换时，注意电流源和电压源的参考方向：电流源电流的参考方向是从电压源的低电位流向高电位。

独立方程求解法

电路分析的基本依据是两类约束：元件约束（伏安关系 VAR）、拓扑约束（基尔霍夫电流电压定律 KCL KVL）。如图所示电路拥有 4 个结点（1、2、3、4）和 6 条支路（\(I_1\)、\(I_2\)、\(I_3\)、\(I_4\)、\(I_5\)、\(I_6\)）。

独立的 KCL 方程

首先，假设流入结点的电流为正+流出为负-，列写上述结点的 KCL 方程：

\[ \begin{cases} 结点 1 \implies -I_1 - I_2 - I_4 = 0 \\ 结点 2 \implies I_2 - I_3 - I_5 = 0 \\ 结点 3 \implies I_1 + I_3 - I_6 = 0 \\ 结点 4 \implies I_4 + I_5 + I_6 = 0 \end{cases} \]

由于每条支路都与两个结点相连接，支路电流从一个结点流出，又从另外一个结点流入。因而如果将上述 4 个方程相加，等式的两边必然都等于0。这样上述 4 个 KCL 方程当中的任意一个可以由另外三个推导出来，例如将上述方程组的前三个方程相加，数学化简最后得到的就是第四个方程，因此这四个方程都不是独立的。

如果将这里的第四个方程去掉，所得到的前三个方程，此时任何一个方程都不可能由另外两个推导出来，因而成为一组独立的 KCL 方程组。

\[ \begin{cases} 结点 1 \implies -I_1 - I_2 - I_4 = 0 \\ 结点 2 \implies I_2 - I_3 - I_5 = 0 \\ 结点 3 \implies I_1 + I_3 - I_6 = 0 \end{cases} \]

具有 \(n\) 个结点的电路，其独立的 KCL 方程数为 \(n-1\) 个；任选 \(n-1\) 个结点列写 KCL 方程，就可以得到独立的 KCL 方程组，这里的 \(n-1\) 个结点就称为独立结点。

独立的 KVL 方程

首先，标记出上面电路所有支路上的电压 \(U_1\)、\(U_2\)、\(U_3\)、\(U_4\)、\(U_5\)、\(U_6\)：

由于顺时针方向上电压降的代数和为零，可以列写回路上的 KVL 方程：

\[ \begin{cases} 回路 1-3-2-1 \implies U_1 - U_3 - U_2 = 0 \\ 回路 1-2-4-1 \implies U_2 + U_5 - U_4 = 0 \\ 回路 2-3-4-2 \implies U_3 + U_6 - U_5 = 0 \\ 回路 1-3-4-1 \implies U_1 + U_6 - U_4= 0 \end{cases} \]

同样将上述方程组的前三个方程左右相加，可以得到第四个方程，由于任意一个方程都可以由另外三个推导得出，所以这组方程不是独立的。如果将第四个方程移除，这样任意一个方程都不可能由另外两个推导得到，从而成为一组独立的 KVL 方程组，这样的三个回路就称为独立回路。

\[ \begin{cases} 回路 1-3-2-1 \implies U_1 - U_3 - U_2 = 0 \\ 回路 1-2-4-1 \implies U_2 + U_5 - U_4 = 0 \\ 回路 2-3-4-2 \implies U_3 + U_6 - U_5 = 0 \end{cases} \]

平面电路的全部网孔就是一组独立回路，对于具有 \(b\) 条支路 \(n\) 个结点的平面电路而言，其网孔数量为 \(b-n+1\) 个，任选 \(b-n+1\) 个网孔列写 KVL 方程，就可以得到一个独立的 KVL 方程组。

2b 法

如果当前电路拥有 \(b\) 条支路 \(n\) 个结点，那么可以列写的独立 KCL 方程数量为：

\[ 支路数量 b - 结点数量 n + 1 \implies \begin{cases} -I_1 - I_2 - I_4 = 0 \\ I_2 - I_3 - I_5 = 0 \\ I_1 + I_3 - I_6 = 0 \end{cases} \]

与此同时，可以列写出的独立 KVL 方程数量为：

\[ 结点数量 n - 1 \implies \begin{cases} U_1 - U_3 - U_2 = 0 \\ U_2 + U_5 - U_4 = 0 \\ U_3 + U_6 - U_5 = 0 \end{cases} \]

如果以支路电压和支路电流作为待求变量，那么待求的变量数量有 \(2b\) 个，而独立的 KCL 和 KVL 方程总数为 \((b - n + 1)+ (n - 1) = b\) 个，并不能求解出全部的支路电压与电流。由于电路分析主要依据的是两类约束，前面已经根据 KCL 和 KVL 拓扑约束列写了方程组，所以接下来可以再根据 VCR 元件约束列写如下的支路电压表达式：

\[ 支路数量 b \implies \begin{cases} U_1 = I_1 R_1 + U_{s2} \\ U_2 = I_2 R_2 \\ U_3 = I_3 R_3 \\ U_4 = U_{s1} + I_4 R_4 \\ U_5 = I_5 R_5 \\ U_6 = U_{s3} \end{cases} \]

这样根据 VCR 元件约束就可以得到 \(b\) 条支路上的 \(b\) 个方程，加上基于 KCL 和 KVL 拓扑约束得到的 \((b - n + 1)+ (n - 1) = b\) 个方程，一共可以联立 \(2b\) 个方程，从而能够求解出全部的支路电压和支路电流，这样的方法就称为 2b 法。

支路电流法

对于如下这个拥有 \(6\) 条支路 \(4\) 个结点的电路，前面小节采用了 2b 法进行求解，这里进行一下简单的回顾。

首先，选取 \(n\) 个结点中的任意 \(n-1\) 个结点，列出独立的 KCL 方程；然后，选取 \(b-n+1\) 个独立回路，通常选择网孔，列出独立的 KVL 方程；最后，由于 \(b\) 条支路电流与 \(b\) 条支路电压为当前待求变量，\(2b\) 个变量需要对应 \(2b\) 个方程，因此还需要补充 \(b\) 个方程，这里采用欧姆定律将支路电压用支路电流进行表示。

\[ \begin{cases} -I_1 - I_2 - I_4 = 0 \\ I_2 - I_3 - I_5 = 0 \\ I_1 + I_3 - I_6 = 0 \end{cases} + \begin{cases} U_1 - U_3 - U_2 = 0 \\ U_2 + U_5 - U_4 = 0 \\ U_3 + U_6 - U_5 = 0 \end{cases} + \begin{cases} U_1 = I_1 R_1 + U_{s2} \\ U_2 = I_2 R_2 \\ U_3 = I_3 R_3 \\ U_4 = U_{s1} + I_4 R_4 \\ U_5 = I_5 R_5 \\ U_6 = U_{s3} \end{cases} \]

由上面的三组方程，可以得到 \(2b\) 个方程，求解这些方程就可以得到 \(b\) 个支路电压与 \(b\) 个支路电流，从而完成 \(2b\) 个变量的求解。接下来，将上述方程中 \(U_1\)、\(U_2\)、\(U_3\)、\(U_4\)、\(U_5\)、\(U_6\) 的表达式，代入到 KVL 方程，就可以得到 3 个网孔用支路电流所表示的 KVL 方程：

\[ \begin{cases} I_1 R_1 + U_{s2} - I_3 R_3 - I_2 R_2 = 0 \\ I_2 R_2 + I_5 R_5 - I_4 R_4 - U_{s1} = 0 \\ I_3 R_3 + U_{s3} - I_5 R_5 = 0 \end{cases} \]

选择顺时针方向上电压降的代数和为零，上面 3 个独立的 KVL 方程，加上之前得到的 \(n-1\) 个独立 KCL 方程，可以得到如下 \(b\) 个方程，进而完成对 \(b\) 个支路电流变量的求解：

\[ \begin{cases} -I_1 - I_2 - I_4 = 0 \\ I_2 - I_3 - I_5 = 0 \\ I_1 + I_3 - I_6 = 0 \end{cases} + \begin{cases} I_1 R_1 + U_{s2} - I_3 R_3 - I_2 R_2 = 0 \\ I_2 R_2 + I_5 R_5 - I_4 R_4 - U_{s1} = 0 \\ I_3 R_3 + U_{s3} - I_5 R_5 = 0 \end{cases} \]

这种以支路电流作为求解变量列写方程，从而全面求解电路的方法称为支路电流法；接下来，重新梳理一下对于 \(b\) 条支路 \(n\) 个结点的电路，采用支路电流法的求解步骤：

  1. 首先，要在电路上标注其支路电流及其参考方向，然后列写 \(n-1\) 个独立结点的 KCL 方程（上面电路选择了结点 1、2、3 列写 KCL 方程）。 \[ \begin{cases} -I_1 - I_2 - I_4 = 0 \\ I_2 - I_3 - I_5 = 0 \\ I_1 + I_3 - I_6 = 0 \end{cases} \]
  2. 然后，选择 \(b-n+1\) 个独立回路（通常直接选定网孔），并指定其绕行方向，然后列写独立的 KVL 方程，采用支路电流来表示支路电压。 \[ \begin{cases} I_1 R_1 + U_{s2} - I_3 R_3 - I_2 R_2 = 0 \\ I_2 R_2 + I_5 R_5 - I_4 R_4 - U_{s1} = 0 \\ I_3 R_3 + U_{s3} - I_5 R_5 = 0 \end{cases} \]
  3. 最后，联立两个方程组就可以得到 \(b\) 个方程，从而得以求解所有支路电流的值。 \[ \begin{cases} -I_1 - I_2 - I_4 = 0 \\ I_2 - I_3 - I_5 = 0 \\ I_1 + I_3 - I_6 = 0 \\ I_1 R_1 + U_{s2} - I_3 R_3 - I_2 R_2 = 0 \\ I_2 R_2 + I_5 R_5 - I_4 R_4 - U_{s1} = 0 \\ I_3 R_3 + U_{s3} - I_5 R_5 = 0 \end{cases} \]

网孔电流法

上一节采用支路电流法列写独立的 KCL 以及 KVL 方程对电桥电路进行求解，虽然只需求解一个支路电流，但是列写的方程数量却多达 6 个。面对类似问题，可以采用一组数量更少的电流、电压变量进行求解，这些变量通常需要具备如下条件：

  • 完备性：这组变量求解以后，利用它可以获得电路当中所有支路电压和电流；
  • 独立性：如果是电流变量，那么应该自动满足 KCL；如果是电压变量，那么应该自动满足 KVL；

网孔电流是一种沿着网孔流动的假想电流，其参考方向可以任意指定，通常选择与回路绕行方向一致。下面电路包含有 3 条支路和 2 个结点，并且选定了 \(i_{m1}\) 和 \(i_{m2}\) 两个网孔电流：

对于上面电路当中的任何一个结点，网孔电流即流入这个结点，也流出这个结点，因此网孔电流自动满足 KCL。所以利用网孔分析法对电路进行分析时，实际列写的是 KVL 方程：

\[ \begin{cases} 网孔 1 \implies R_1i_1 - R_2i_2 + u_{s2} - u_{s1} = 0 \\ 网孔 2 \implies R_2i_2 - R_3i_3 - u_{s2} = 0 \end{cases} \]

接下来，将支路电流用网孔电流代替表示：

\[ \begin{cases} i_1 = i_{m1} \\ i_3 = i_{m2} \\ i_2 = i_{m2} - i_{m1} \end{cases} \]

然后，将采用网孔电流表示的 \(i_1\)、\(i_2\)、\(i_3\) 代入上面的 KVL 方程：

\[ \begin{cases} 网孔 1 \implies R_1i_{m1} - R_2(i_{m2} - i_{m1}) + u_{s2} - u_{s1} = 0 \\ 网孔 2 \implies R_2(i_{m2} - i_{m1}) - R_3i_{m2} - u_{s2} = 0 \end{cases} \]

继续对这个采用网孔电流表示的 KVL 方程组进行数学整理：

\[ \begin{cases} 网孔 1 \implies (R_1 + R_2)i_{m1} - R_2i_{m2} = u_{s1} - u_{s2} \\ 网孔 2 \implies -R_2i_{m1} + (R_2 + R_3)i_{m2} = u_{s2} \end{cases} \]

观察上述方程组，会发现 2 个等式具备如下特点：

  • 网孔 1 当中， \(i_{m1}\) 的系数 \(R_1 + R_2\) 是网孔 1 当中所有电阻的和，称之为网孔 1 的自电阻，记为 \(R_{11}\)；而 \(i_{m2}\) 的系数为 网孔 1 和 2 公共支路上的电阻 \(-R_2\)，称为这两个网孔的互电阻，记为 \(R_{12}\)；等号右侧的 \(u_{s1} - u_{s2}\) 表示的是沿着网孔电流参考方向，电压源电压升的代数和，记为 \(u_{sm1}\)。
  • 网孔 2 当中， \(i_{m2}\) 的系数 \(R_2 + R_3\) 是网孔 2 当中所有电阻的和，称之为网孔 2 的自电阻，记为 \(R_{22}\)；而 \(i_{m1}\) 的系数为 网孔 1 和 2 公共支路上的电阻 \(-R_2\)，称为这两个网孔的互电阻，记为 \(R_{21}\)。等号右侧的 \(u_{s2}\) 表示的是沿着网孔电流参考方向，电压源电压升的代数和，记为 \(u_{sm2}\)。

进行上述设定以后，网孔 1 和 2 的 KVL 方程就可以记为：

\[ \begin{cases} 网孔 1 \implies R_{11}i_{m1} + R_{12}i_{m2} = u_{sm1} \\ 网孔 2 \implies R_{21}i_{m1} + R_{22}i_{m2} = u_{sm2} \end{cases} \]

这里的 \(R_{11}\) 和 \(R_{22}\) 分别是网孔 1 和 2 的自电阻，而 \(R_{12}\) 和 \(R_{21}\) 是网孔公共支路上的互电阻（两者的值相等）。如果两个网孔电流的方向相反，则 \(R_{12}\) 和 \(R_{21}\) 为负值。方程右侧的 \(u_{sm1}\) 和 \(u_{sm2}\) 分别是网孔 1 和 2 在沿着网孔电流参考方向的绕向上，电压源电压升的代数和。

将上述结论推广到包含有 \(l\) 个网孔，且仅含有电阻、独立电压源的电路：

\[ \begin{cases} R_{11}i_{m1} + R_{12}i_{m2} + \cdots + R_{1l}i_{ml} & = u_{sm1} \\ R_{21}i_{m1} + R_{22}i_{m2} + \cdots + R_{2l}i_{ml} & = u_{sm2} \\ R_{l1}i_{m1} + R_{l2}i_{m2} + \cdots + R_{ll}i_{ml} & = u_{sml} \end{cases} \]

  • 形如 \(R_{kk}\) 指的是第 k 个网孔的自电阻（\(k=1,2,...,l\)），其值总是为正+；
  • 形如 \(R_{jk}\) 指的是第 j 个和第 k 个网孔的互电阻（\(j \neq k\)），流过互电阻的两个网孔电流方向相同时为正+相反时为负-，如果公共支路上没有电阻，则直接取0值；
  • 形如 \(u_{smk}\) 指的是第 k 个网孔当中所有电压源电压升的代数和。

注意：如果设定所有网孔的电流方向同时为顺时针或逆时针，此时流过互电阻的两个网孔电流方向一定是相反的，此时互电阻恒为负值-。

【例题】桥式电路当中，\(U_s=12V\)，\(R_1 = R_2 = R_4 = 5Ω\)，\(R3 = 10Ω\)，中间位置的支路是一个检流计，其电阻 \(R_G = 10Ω\)，求解检流计当中的电流 \(I_G\) ？

【解答】这里采用网孔电流法来求解前一小节的例题。

首先，设定网孔电流 \(I_{m1}\)、\(I_{m2}\)、\(I_{m3}\)，并选择其参考方向为顺时针方向，列写 KVL 方程：

\[ \begin{cases} 网孔 1 \implies R_{11}I_{m1} + R_{12}I_{m2} + R_{13}I_{m3} = 0 \\ 网孔 2 \implies R_{21}I_{m1} + R_{22}I_{m2} + R_{23}I_{m3} = 0 \\ 网孔 3 \implies R_{31}I_{m1} + R_{32}I_{m2} + R_{33}I_{m3} = U_s \end{cases} \]

接下来，对上面方程组当中自电阻（\(R_{11}\)、\(R_{22}\)、\(R_{33}\)）和互电阻（\(R_{12}\)、\(R_{13}\)、\(R_{21}\)、\(R_{23}\)、\(R_{31}\)、\(R_{32}\)）的值进行求解：

  • \(R_{11}\) 是第 1 个网孔自电阻：\(R_{11} = R_1 + R_G + R_3 = 5Ω + 10Ω + 10Ω = 25Ω\)；
  • \(R_{22}\) 是第 2 个网孔自电阻：\(R_{22} = R_G + R_2 + R_4 = 10Ω + 5Ω + 5Ω = 20Ω\)；
  • \(R_{33}\) 是第 3 个网孔自电阻：\(R_{33} = R_3 + R_4 = 10Ω + 5Ω = 15Ω\)；

由于当前网孔电流都选择为顺时针方向，所以全部互电阻都将为负值-：

  • 第 1 和 2 公共支路上的互电阻：\(-R_G = R_{12} = R_{21} = -10Ω\)；
  • 第 1 和 3 公共支路上的互电阻：\(-R_3 = R_{13} = R_{31} = -10Ω\)；
  • 第 2 和 3 公共支路上的互电阻：\(-R_4 = R_{23} = R_{32} = -5Ω\)；

求解得到所有自电阻与互电阻的值之后，将其代入至前面列写的网孔电流方程组并进行求解：

\[ \begin{cases} 网孔 1 \implies 25I_{m1} - 10I_{m2} - 10I_{m3} = 0 \\ 网孔 2 \implies -10I_{m1} + 20I_{m2} - 5I_{m3} = 0 \\ 网孔 3 \implies -10I_{m1} - 5I_{m2} + 15I_{m3} = 12 \end{cases} \implies \begin{cases} I_{m1} = 1.26A \\ I_{m2} = 1.13A \\ I_{m3} = 2.02A \end{cases} \]

由于最终需要求解的是检流计 \(R_G\) 上的电流 \(I_G\)，即网孔电流 \(I_{m1}\) 和 \(I_{m2}\) 的代数和：

\[ I_G = I_{m1} - I_{m2} = 0.13A \]

采用网孔电流法列写的方程数量，是支路电流法的一半，计算起来更为简便，这里再总结一下网孔电流法的求解步骤：

  1. 首先，选定 \(l = b-n+1\) 个网孔，标明各个网孔电流及其方向；
  2. 然后，以网孔电流作为未知量，对 \(l\) 个网孔列写 KVL 方程；
  3. 最后，求解这些 KVL 方程，得到网孔电流的值，再进一步求解各条支路的电压电流；

结点电压法

下图电路拥有 \(n=3\) 个结点，独立结点数量为 \(n-1=2\)，选择结点 3 为零电位参考结点，结点 1 和 2 与参考结点的电压降称为结点电压，分别记为 \(U_{N1}\) 和 \(U_{N2}\)。

任意选择一个零电位参考点，结点电压就是指定结点与参考点之间的电压降，也即结点电位，其方向由独立结点指向参考结点。

结点电压变量是否能够对电路进行全面求解，取决于其是否满足完备性和独立性两个条件。该电路的支路电压一共拥有 3 个，分别将其标注为 \(U_1\)、\(U_2\)、\(U_3\)：

由于 \(U_1 = U_{N1}\)、\(U_2 = U_{N2}\)、\(U_3 = U_{N1} - U_{N2}\)，因而只要求解出结点电压 \(U_{N1}\) 和 \(U_{N2}\) ，就可以相应的求出所有的支路电压和支路电流，所以它具有完备性。

对于上面的网孔，顺时针方向绕行一圈的代数和等于 \(U_3 + U_2 - U_1\)，代入 \(U_{N1}\) 和 \(U_{N2}\) 以后可以得到 \((U_{N1} - U_{N2}) + U_{N2} - U_{N1} = 0\)，能够自动满足 KVL 方程，因而也具备独立性。

因为这组结点电压变量可以同时满足完备性和独立性，通过它们可以对电路进行全面的求解。这种以结点电压为未知变量，列写 KCL 方程分析电路的方法就称为结点电压法。

  1. 选定参考结点，并且标注其它 \(n-1\) 个独立结点的电压；
  2. 然后基于独立结点列写 KCL 方程；

\[ \begin{cases} 结点 1 \implies I_1 + I_{S2} + I_2 + I_3 = I_{S1} + I_{S3} \\ 结点 2 \implies I_4 + I_{S3} = I_3 \end{cases} \implies \begin{cases} I_1 + I_2 + I_3 = I_{S1} - I_{S2} + I_{S3} \\ -I_3 + I_4 = - I_{S3} \end{cases} \]

将支路电流 \(I_1\)、\(I_2\)、\(I_3\)、\(I_4\) 采用结点电压进行表示：

\[ \begin{cases} I_1 = G_1U_{N1} \\ I_2 = G_2U_{N1} \\ I_3 = G_3(U_{N1} - U_{N2}) \\ I_4 = G_4U_{N2} \end{cases} \]

将这些方程代入上面列写的结点 KCL 方程并且整理：

\[ \begin{cases} 结点 1 \implies G_1U_{N1} + G_2U_{N1} + G_3(U_{N1} - U_{N2}) = I_{S1} - I_{S2} - I_{S3} \\ 结点 2 \implies -G_3(U_{N1} - U_{N2}) + G_4U_{N2} = -I_{S3} \end{cases} \implies \begin{cases} (G_1 + G_2 + G_3)U_{N1} - G_3U_{N2} = I_{S1} - I_{S2} - I_{S3} \\ -G_3U_{N1} + (G_3 + G_4)U_{N2} = -I_{S3} \end{cases} \]

上面方程左侧的 \(G_1 + G_2 + G_3 = G_{11}\) 是与结点 1 相连的全部电导之和，称为结点 1 的自电导；同样的，\(G_3 + G_4 = G_{22}\) 是与结点 2 相连的全部电导之和，称为结点 2 的自电导；$ -G_3 = G_{21} = G_{12}$ 是结点 1 和 2 之间所有支路上电导之和的负值，称为结点 1 和 2 之间的互电导。方程右侧的 \(I_{S1} - I_{S2} + I_{S3} = I_{S11}\) 是流入结点 1 的电流源电流的代数和；同理，\(I_{S22} = -I_{S3}\) 是流入结点 1 的电流源电流的代数和；电流源的电流流入结点时取正号+，流出时取负号-。最后，将上面的方程组简记为如下形式：

\[ \begin{cases} 结点 1 \implies G_{11}U_{N1} + G_{12}U_{N2} = I_{S11} \\ 结点 2 \implies G_{21}U_{N1} + G_{22}U_{N2} = I_{S22} \end{cases} \]

将结点电压法推广至含有 n 个结点，且仅含有电阻和独立电流源的电路。对于 n 个结点的电路，其独立结点个数为 \(n-1\) 个，因而可以列写 \(n-1\) 个方程：

\[ \begin{cases} G_{11}U_{N1} + G_{12}U_{N2} + \cdots + G_{1(n-1)}U_{N(n-1)} = I_{S11} \\ G_{21}U_{N1} + G_{22}U_{N2} + \cdots + G_{2(n-1)}U_{N(n-1)} = I_{S22} \\ \cdots \qquad \cdots \qquad \cdots \qquad \cdots \qquad \cdots \qquad \cdots \\ G_{(n-1)1}U_{N1} + G_{(n-2)2}U_{N2} + \cdots + G_{(n-1)(n-1)}U_{N(n-1)} = I_{S(n-1)(n-1)} \end{cases} \]

  • 方程左侧的 \(G_{11}\)、\(G_{22}\)、\(G_{(n-1)(n-1)}\) 记作 \(G_{kk}\)，即第 \(k\) 个结点上所有支路的电导之和，称为结点 \(k\) 的自电导，自电导总是为正+；
  • 方程左侧的 \(G_{12}\)、\(G_{21}\)、\(G_{(n-1)1}\)、\(G_{(n-2)2}\)、\(G_{1(n-1)}\)、\(G_{2(n-1)}\) 记作 \(G_{jk} (j \neq k)\) ，即第 \(k\) 和 \(j\) 个结点之间所有公共支路上电导之和的负值，称为结点 \(k\) 与 \(j\) 之间的互电导，互电导总是为负-；
  • 方程右侧的 \(I_{S11}\)、\(I_{S22}\)、\(I_{S(n-1)(n-1)}\) 指的是流入结点 \(k\) 的所有电流源电流的代数和；

采用结点电压法进行电路分析时，列写的方程遵循这样的规律：所在结点的结点电压乘以与该结点相连的所有支路上的电导之和，减去相临结点的电压与公共支路上电导之和的乘积，等于流入该结点的所有电流源电流的代数和。

\[ $G_{12} = -frac{1}{R_2}$ \]

包含无伴电压源

【例题】如图所示的电桥电路，\(U_S=12V\)，中间的支路是一个检流计，求解检流计上的电流 \(I_G\) 的值？

结点 1 的电压 \(U_{N1}\) 乘以与该结点相连支路的电导之和 \((\frac{1}{5} + \frac{1}{10})U_{N_1}\)，减去相邻结点 \(U_{N2}\) 和 \(U_{N4}\) 的电压乘以公共支路上的电导（结点 4 是零电位参考点无需纳入方程），即 \(\frac{1}{10}U_{N_2}\)。等号右侧是流入该结点的电流之和（如果包含有电流源），目前与结点 1 相连接的没有电流源，只有一个电压源 \(U_S\)，因此需要为其标注一个新的电流变量 \(I\)：

\[ (\frac{1}{5} + \frac{1}{10})U_{N_1} - \frac{1}{10}U_{N_2} = I \]

结点 2 的电压 \(U_{N2}\) 乘以与该结点相相连的所有电导之和，减去相邻结点的电压 \(U_{N1}\) 和 \(U_{N3}\) 乘以各自支路上的电导：

结点 3 的电压 \(U_{N3}\) 乘以的自电导为 \(\frac{1}{R_2} + \frac{1}{R_4}\)，减去相邻结点 2 的电压 \(U_{N2}\) 乘以公共支路上的电导。方程右侧由于 \(U_S\) 所表示的支路电流 \(I\) 当前是流出结点 3 的，所以等于 \(-I\)：

\[ -\frac{1}{5} U_{N2} + (\frac{1}{5} + \frac{1}{5}) U_{N3} = -I \]

联立上述方程，并且进行数学整理，得到如下方程组：

\[ \begin{cases} 结点 1 \implies 0.3U_{N1} - 0.1U_{N2} = I \\ 结点 2 \implies -0.1U_{N1} + 0.4U_{N2} - 0.2U_{N3} = 0 \\ 结点 3 \implies -0.2U_{N2} + 0.4U_{N3} = -I \end{cases} \]

\[ U_{N1} - U_{N3} = U_S \]

\[ \begin{cases} U_{N1} = 6.32V \\ U_{N2} = -1.26V \\ U_{N3} = -5.68 \\ I_G = \frac{-U_{N2}}{R_G} = 0.126A \end{cases} \]

结点 3 的电位为0，就意味结点 1 的电压 \(U_{N1} = U_S = 12V\)，这样就无需再对结点 1 列写 KCL 方程，只需要列写结点 2 和 4 的方程：

\[ \begin{cases} 结点 2 \implies (\frac{1}{10} + \frac{1}{10} + \frac{1}{5}) U_{N2} - \frac{1}{10} U_{N1} - \frac{1}{10} U_{N4} = 0 \\ 结点 4 \implies (\frac{1}{5} + \frac{1}{10} + \frac{1}{5}) U_{N4} - \frac{1}{5} U_{N1} - \frac{1}{10} U_{N2} = 0 \end{cases} \]

将 \(U_{N1} = 12V\) 代入上面的方程组，即可以求解出结点 2 和 4 的电压 \(U_{N2}\) 和 \(U_{N4}\)：

\[ \begin{cases} U_{N2} = 4.42V \\ U_{N4} = 5.68V \end{cases} \]

最后，就可以求得检流计上的电流 \(I_G\)：

\[ I_G = \frac{U_{N4} - U_{N2}}{R_G} = \frac{5.68-4.42}{10} = 0.126A \]

可以发现，检流计上电流 \(I_G\) 的值与选择结点 4 作为参考点时的结果一致，对比可以发现方法 2 的求解要更加简洁，说明当电路中出现无伴电压源时，为了便于计算，应该选择与无伴电压源相连接的结点作为参考结点。

对偶原理

结点电压法：以结点电压作为待求变量，列写的是 KCL 方程：

\[ \begin{cases} G_{11}U_{N1} + G_{12}U_{N2} + \cdots + G_{1(n-1)}U_{N(n-1)} = I_{s11} \\ G_{21}U_{N1} + G_{22}U_{N2} + \cdots + G_{2(n-1)}U_{N(n-1)} = I_{s22} \\ \cdots \qquad \cdots \qquad \cdots \qquad \cdots \\ G_{(n-1)1}U_{N1} + G_{(n-2)2}U_{N2} + \cdots + G_{(n-1)(n-1)}U_{N(n-1)} = I_{s(n-1)(n-1)} \\ \end{cases} \]

网孔电流法：以网孔电流作为待求变量，列写的是 KVL 方程：

\[ \begin{cases} R_{11}I_{m1} + R_{12}I_{m2} + \cdots + R_{1l}I_{ml} = I_{sm1} \\ R_{21}I_{m1} + R_{22}I_{m2} + \cdots + R_{21}I_{m1} = I_{sm2} \\ \cdots \qquad \cdots \qquad \cdots \qquad \cdots \\ R_{l1}I_{m1} + R_{12}I_{m2} + \cdots + R_{ll}I_{m1} = I_{sm1} \\ \end{cases} \]

将上述方程组当中的电导G和电压R互换、电压U和电流I互换，就可以将结点电压方程转换为网孔电流方程，或者将网孔电流方程转换为结点电压方程，这种性质称为对偶性；这些可以互换的变量，称为对偶量或者对偶元素。

电容元件：伏安关系为 \(i = C\frac{du}{dt}\)，储存的是电场能 \(W_C = \frac{1}{2}CU^2\)；

电感元件：伏安关系为 \(u = L\frac{di}{dt}\)，储存的是磁场能 \(W_S = \frac{1}{2}Li^2\)；

将电容和电感公式中的电压U和电流I互换、电容C和电感L互换，就可以由电容元件的伏安关系、电场能表达式转换为电感元件的伏安关系、电场能表达式，或者由电感元件的伏安关系、电场能表达式转换为电容元件的伏安关系、电场能表达式，说明电感元件和电容元件也具有对偶性，电压U和电流I和电容C和电感L称为对偶元素。

\[ \begin{aligned} i = C\frac{du}{dt} & \Longleftrightarrow u = L\frac{di}{dt} \\ W_C = \frac{1}{2}CU^2 & \Longleftrightarrow W_S = \frac{1}{2}Li^2 \end{aligned} \]

电路当中的常用对偶元素如下表所示：

\(电阻R \iff 电导G\) \(电感L \iff 电容C\) \(电压U \iff 电流I\)
\(串联 \iff 并联\) \(开路 \iff 短路\) \(电压源 \iff 电流源\)
\(结点 \iff 网孔\) \(KVL \iff KCL\)  

将两个不同电路当中具有对偶性的元素进行互换，其对应的电路方程具有相同的解，这种可以互换的性质称为对偶原理。如果两个电路能够用对偶量可以相互交换的特征方程进行描述，就称这两个电路互相具有对偶性。值得注意的是，对偶原理只适用于平面电路。

叠加定理

线性电路具有线性性质，线性性质具体指的就是齐次性和可加性。下图的线性电阻元件，电流参考方向和电压参考方向如图所示，根据欧姆定律可知 \(u = Ri\)。

对于同样的线性电阻 \(R\)，流过的电流为 \(k\) 倍的 \(i\)，此时电阻元件两端的电压记为 \(u^{'}\)，根据欧姆定律 \(u^{'} = R \cdot ki = k \cdot Ri = ku\)。这说明对于相同的电路，如果其输入（激励）增大了 \(k\) 倍，相应的输出（响应）也同样增大 \(k\) 倍，这就是电路的齐次性。

对于下面的线性电阻元件 \(R\)，如果流过的电流为 \(i_1\) 两端电压为 \(u_1\)，根据欧姆定律可得 \(u_1 = R i_1\)。

如果流过的电流为 \(i_2\) 两端电压为 \(u_2\)，根据欧姆定律可得 \(u_2 = R i_2\)。

当下面元件 \(R\) 上流过的电流为 \(i_1\) 和 \(i_2\) 之和，电压为 \(u^{''}\)，由欧姆定律可知 \(u^{''} = R(i_1 + i_2) = Ri_1 + Ri_2 = u_1 + u_2\)。这说明对于线性电路，多个输入量作用的响应等于每一个输入量单独作用时响应的叠加，这就是电路的可加性。

【例题】如下图所示的线性电路，当电压 \(U_s=10V\) 时电流 \(I=2A\)，当 \(U_s=1V\) 时 ，此时的电流 \(I\) 等于多少？

【解答】将本题当中的 \(U_s\) 视为激励，电流 \(I\) 视为响应。当 \(U_s=1V\) 时相比原来的 \(U_s=10V\)，系数 \(k = \frac{1}{10}\)，因此其响应 \(I = \frac{1}{10} \times 2 = 0.2A\)。

【例题】假设电阻 \(R1\)、\(R2\)、\(R3\) 已知，试分析独立电压源 \(U_{s1}\) 和 \(U_{s2}\) 与电流 \(I\) 与电压 \(U_1\) 的关系？

【解答】该电路拥有 3 条支路 2 个网孔 2 个结点，采用结点电压法会比较简便。

将一个结点设置为参考结点，另外一个结点的电压就是待求变量 \(U_{N1}\)

\[ (\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3}) \cdot U_{N1} = \frac{U_{s1}}{R_1} + \frac{U_{s2}}{R_2} \]

将 \(\frac{1}{R_1}\)、\(\frac{1}{R_2}\)、\(\frac{1}{R_3}\) 用电导 \(G_1\)、\(G_2\)、\(G_3\) 进行表示，这样就可以求得结点电压 \(U_{N1}\)：

\[ U_{N1} = \frac{G_1}{G_1 + G_2 + G_3} U_{s1} + \frac{G_2}{G_1 + G_2 + G_3} U_{s2} \]

获取到 \(U_{N1}\) 的表达式以后，求解电流 \(I\) 就显得比较容易：

\[ I = \frac{U_{N1}}{R_3} = \frac{G_1G_3}{G_1 + G_2 + G_3} U_{s1} + \frac{G_2G_3}{G_1 + G_2 + G_3} U_{s2} \]

接下来就可以求解 \(U_1\)：

\[ U_1 + U_{s1} = U_{N1} \implies U_1 = U_{N1} - U_{s1} = \frac{-G_1-G_3}{G_1 + G_2 + G_3} U_{s1} + \frac{G_2}{G_1 + G_2 + G_3} U_{s2} \]

这样就求得了电流 \(I\) 和电压 \(U_1\) 与独立电压源 \(U_{s1}\) 和 \(U_{s2}\) 的关系表达式。将上面方程组当中的 \(\frac{G_1G_3}{G_1 + G_2 + G_3}\) 记为 \(k_{11}\)，\(\frac{G_2G_3}{G_1 + G_2 + G_3}\) 记为 \(k_{12}\)，\(\frac{-G_1-G_3}{G_1 + G_2 + G_3}\) 记为 \(k_{21}\)，\(\frac{G_2}{G_1 + G_2 + G_3}\) 记为 \(k_{22}\)，这样就可以得到：

\[ \begin{cases} I = K_{11}U_{s1} + K_{12}U_{s2} \\ U_1 = K_{21}U_{s1} + K_{22}U_{s2} \end{cases} \]

\(K_{11}U_{s1}\) 等于 \(U_{s2}\) 为零时的电流 \(I\)，记为 \(I^{'}\)；\(K_{12}U_{s2}\) 则等于 \(U_{s1}\) 为零时的电流 \(I\)，记为 \(I^{''}\)。\(K_{21}U_{s1}\) 等于 \(U_{s2}\) 为零时的电压 \(U_1\)，记为 \(U_{1}^{'}\)；\(K_{22}U_{s2}\) 则等于 \(U_{s1}\) 为零时的电压 \(U_1\)，记为 \(U_{1}^{''}\)。

\[ \begin{cases} I = K_{11}U_{s1} + K_{12}U_{s2} = I^{'} + I^{''} \\ U_1 = K_{21}U_{s1} + K_{22}U_{s2} = U_{1}^{'} + U_{1}^{''} \end{cases} \]

置换定理

置换定理也称为替代定理，对于由二端网络 \(N_1\) 和 \(N_2\) 连接构成的电路而言，如果已经求得 \(N_1\) 和 \(N_2\) 连接端口的电压 \(u_k\) 与电流 \(i_k\)，那么就可以采用一个电压等于 \(u_k\) 的电压源或电流等于 \(i_k\) 的电流源来替代其中一个网络，从而使得另外一个网络内部的电压、电流保持不变。

接下来对置换定理进行证明，对于上面二端网络 \(N_1\) 和 \(N_2\) 构成的电路，其端口电压为 \(u_k\) 端口电流为 \(i_k\)。在其底部连接线上串联 \(-u_k\) 和 \(+u_k\) 两个电压源，同时添加 A、B、C、D 四个测量点。由于 DC 为 \(-u_k\)，CB 为 \(+u_k\)，显然 DB 之间是等电位点，可以直接采用短路线进行连接。同样的，由于 AB 之间是正 \(+UK\)，BC 之间为 \(-u_k\)，所以 AC 也为等电位点，可以直接用短路线进行连接。

这样就得到了 \(N_1\) 这个二端网络与电压源 \(u_k\) 相连接的电路图，也就是说 \(N_2\) 可以被 \(u_k\) 所置换。

类似的，还可以在 AD 之间并联两个大小相等方向相反的电流源来置换二端网络 \(N_2\)。

【例题】已知下图电路，如果需要让负载电阻 \(R_L\) 的电流为电流源支路电流 \(I\) 的 \(\frac{1}{6}\)，求解 \(R_L\) 的值？

【解答】假设 \(R_L\) 两端的电压为 \(U_{R_L}\)，根据欧姆定律可以得到：

\[ U_{R_L} = R_L \times \frac{I}{6} \]

用电流值为 \(\frac{I}{6}\) 的独立电流源来置换 \(R_L\)，对于没有被置换部分的电路，其电压电流关系保持不变。此时电流源两端的电压就是 \(U_{R_L}\)。

上面的电路包含了两个电流源，可以考虑采用叠加定理进行分析。当仅考虑电流源 \(I\) 起作用时，\(\frac{I}{6}\) 的独立电流源应该开路置零，设置此处的开路电压为 \(U_{R_L}^{'}\)：

\[ U_{R_L}^{'} = 4Ω \times (\frac{8Ω}{8Ω+4Ω+4Ω} \times I ) = 2I \]

反过来，当仅考虑电流源 \(\frac{I}{6}\) 起作用时，独立电流源 \(I\) 开路置零，其余部分保持不变，\(\frac{I}{6}\) 处独立电流源两端的电压为 \(U_{R_L}^{''}\)：

\[ U_{R_L}^{''} = 4Ω \times (\frac{8Ω+4Ω}{8Ω+4Ω+4Ω} \times \frac{I}{6}) = -0.5I \]

这样根据叠加定理，就可以求解出总电压 \(U_{R_L}\) 的值：

\[ U_{R_L} = U_{R_L}^{'} + U_{R_L}^{''} = 2I - 0.5I = 1.5I \]

对于原电路没有被置换的部分，其电压电流关系保持不变。根据题意可以知道 \(U_{R_L}\) 与电流电压的关系为 \(\frac{U_{R_L}}{R_L} = \frac{I}{6}\)，因此负载电阻 \(R_L\) 的值为：

\[ R_L = \frac{U_{R_L}}{\frac{I}{6}} \xrightarrow{U_{R_L}=1.5I} \frac{1.5I}{\frac{I}{6}} = 9Ω \]

这样就完成了本题的求解，接下来对置换定理进行一个总结：

  • 置换定理适用于线性、非线性电路以及时变电路；
  • 没有被置换支路的相互连接以及参数不能发生改变；
  • 置换定理的应用必须满足：原电路与置换后电路必须有唯一解，被置换的支路和电路其它部分无耦合关系；

这里再比较一下置换定理与等效变换的区别，

由于上图电路当中 \(N_1\) 和 \(N_2\) 的伏安关系不同，所以两者并非等效关系。左图电流源的伏安关系为电流恒为 \(1A\)，右图电压源的伏安关系为电压恒为 \(2V\)。但是如果单就 \(2Ω\) 电阻而言，左图电流源 \(2Ω\) 电阻上产生的电压 \(u_1 = 2Ω \times 1A = 2V\)，右图电压源 \(2Ω\) 电阻上产生的电流 \(i_2 = \frac{2V}{2Ω} = 1A\)。

观察会发现两种情况下的电压与电流是相同的，也就是说可以采用一个 2V 的独立电压源置换 1A 独立电流源，而对于没有被置换的 \(2Ω\) 电阻而言，其电压和电流保持不变。因此，\(N_1\) 与 \(N_2\) 是置换的关系。

置换定理是指针对某个具体电路而言，被替代支路之外的电路拓扑结构、元件参数不能改变；等效变换则是对任意外电路都具有相同伏安关系，而与外电路的状态无关。

伏安法求解戴维南等效电阻

戴维南等效定律的内涵就是，任何一个线性含源二端网络都可以等效为一个电压源和电阻的串联，即戴维南等效电路。线性含源二端网络进行戴维南等效之后，对于任意相同的外电路，都具有相同的伏安关系。

求解线性含源二端网络的戴维南等效电路时，主要是求解电压源的 \(U_{oc}\) 以及戴维南等效电阻 \(R_o\)。如果该二端网络不包含有受控源，那么可以让含源二端网络中的独立源置零，此时利用电阻的串并联，以及三角形星形变换就可以很容易的求解到 \(R_o\) 的值。如果二端网络包含了受控源，此时戴维南等效电阻 \(R_o\) 的求解会比较困难，前一小节介绍了外加电源法和开路短路法两种求解戴维南等效电阻的方法：

令线性含源二端网络中的独立源置零，得到一个线性无源二端网络，设其等效电阻为 \(R_o\)。对于阻值为 \(R_o\) 的电阻而言，假设此时外加一个电流源或者一个电压源，根据欧姆定律都可以得到 \(R_o=\frac{U}{I}\)，这种求解含受控源二端网络的戴维南等效电阻的方法，就称为外加电源法。

如果已经求得线性无源二端网络的开路电压 \(U_{oc}\)，在此基础上继续求取线性含源二端网络的短路电流 \(I_{sc}\)，同样根据欧姆定律，戴维南等效电阻 \(R_o = \frac{U_{oc}}{I_{sc}}\)，这就是另外一种求解戴维南等效电阻的方法，称为开路短路法。

诺顿定理

由戴维南定理可以推导得到线性含源二端网络的戴维南等效电路， 即电压源与电阻的串联。而本节将要介绍的诺顿定理，则是将线性含源二端网络等效为电流源 \(I_{sc}\) 和电阻 \(R_o^{'}\) 的并联电路，也就是诺顿等效电路，其中的 \(R_o^{'}\) 称为诺顿等效电阻。

线性含源二端网络即可以等效为戴维南等效电路，也可以等效为诺顿等效电路。由等效的定义可以知道，诺顿等效电阻 \(R_o^{'}\) 实质就是戴维南等效电路中的 \(R_o\)。从而可以得到这样一个结论：诺顿等效电阻就是含源二端网络 N 当中，所有独立源置零时的等效电阻。

诺顿等效电路当中的 \(I_{sc}\) 实质就是由端子a流向端子b的短路电流，由于上图左右两个电阻相互等效，因此诺顿等效电路当中的 \(I_{sc}\) 就是线性含源二端网络端口上的短路电流。

综上所述，诺顿定理的内涵就是：任意的线性含源二端网络 N，等效为一个独立电流源与一个电阻的并联。其中 \(I_{sc}\) 就是线性含源二端网络端口上的短路电流，而等效电阻就是二端网络当中所有独立源为零时的等效电阻。

戴维南等效电路与诺顿等效电路的关系：戴维南等效电路当中的戴维南等效电阻就是诺顿等效电路当中的诺顿等效电阻， 此外由于 \(I_{sc} = \frac{U_{oc}}{R_o}\)，即之前介绍过的两种实际电源之间的等效变换，因此实际电源的这种等效变换也称为戴维南诺顿等效变换。

另一部分则是仅仅由独立电流源起作用，此时线性含源二端网络当中的所有独立源置零，从而得到一个无源的二端网络 \(N_0\)，此时a和b端口电压记为 \(U^{''}\)。由于任意线性无源二端网络 \(N_0\) 都可以等效为一个电阻，假设电阻为 \(R_o\)，那么端口电压 \(U^{''} = -IR_o\)。

最后，来对戴维南定理与诺顿定理进行一个总结：

  • 戴维南定理和诺顿定理都是采用叠加推导而来的，仅适用于线性电路，而不适用于非线性电路；
  • 诺顿定理可以视为戴维南定理的另外一种形式，两者之间可以互相转换；
  • 求解戴维南等效电阻或者诺顿等效电阻 \(R_o\) 时，如果二端网络当中含受控源，则必须考虑受控源的作用，将其保留在原电路当中一起进行计算；
  • 二端网络一定是明确的二端网络，与外电路之间无任何耦合联系，但是控制量可以是端口电压或者电流；(对于下图左侧的线性含源二端网络 \(N_1\) 而言，独立源 \(U_1\) 与外电路之间存在着耦合关系；而中间的 \(N_2\) 包含有受控源，其控制量为外电路上的独立源 \(U_1\)，也存在耦合关系；因此，两者都不能应用戴维南和诺顿定理进行分析计算。右侧的 \(N_3\) 包含的受控源，其控制量 \(U\) 是二端网络端口上的电压，因此可以采用戴维南和诺顿定理进行分析计算。)；

最大功率传输定律

戴维南定理和诺顿定理的一个典型应用，就是对于给定的含源二端网络，寻找到一个合适的负载，使得负载上获得的功率能够最大化。

线性含源二端网络与负载相连接，当负载发生变化的时候，负载两端所获得的功率也随之变化。那么对于给定的线性含源二端网络，负载电阻 \(R_L\) 的值为多大时，才能够获得最大功率？

线性含源二端网络可以进行戴维南或诺顿等效，这里选择戴维南等效，即电压源与电阻串联后，再接入外电路，此时负载 \(R_L\) 上功率 \(P_L\) 的表达式为：

\[ P_L = I^2R_L = (\frac{U_{oc}}{R_o + R_L})^2 \cdot R_L \]

接下来，将功率随着负载的变化趋势，在一个平面直角坐标系上面进行描述：

可以看到，当负载 \(R_L\) 的阻值较大的时候，由于电流 \(I\) 较小，因此负载得到的功率 \(I^2R_L\) 很小；而当负载 \(R_L\) 的阻值较小时，虽然电流 \(I\) 较大，但是得到的功率依然较小；如果对于上面方程当中的负载求一阶导数，可以得到如下表达式：

\[ \frac{dP_L}{dR_L} = \frac{U_{oc}^2 [(R_o + R_L)^2 - 2R_L(R_o + R_L)]}{(R_o+R_L)^4} = \frac{U_{oc}^2(R_o-R_L)}{(R_o-R_L)^3} \]

当上面表达式的值为零时，此时的功率可以有最大值；也就是说负载 \(R_L\) 等于 \(R_o\) 的时候，负载上面会有最大功率。这就是最大功率传输定律：由线性含源二端网络传递给可变负载 \(R_L\) 的功率为最大值的条件是负载 \(R_L\) 应当与戴维南或者诺顿等效电阻的阻值相等 \(R_L = R_o\)。此时负载电阻 \(R_L\) 上获得的最大功率 \(P_max\) 等于：

\[ P_{max} = \frac{U_{oc}^2}{4R_o} \]

如果当前进行的是诺顿等效，则还可以写成下面的形式：

\[ P_{max} = \frac{I_{sc}^2 R_o}{4} \]

由最大传输功率定理可以知道，对于给写的线性含源二端网络，当其连接至一个可变负载的时候，该可变负载的值与线性含源二端网络的戴维南等效电阻相等的时候 \(R_L = R_o\)，负载上获得的功率是最大的。

此时由于 \(R_L = R_o\)，它们消耗的功率也相等，当负载获得的功率来自内阻 \(R_o\) 的电压源时，就可以认为功率的传递效率为 50%。

【例题】如图所示桥式电路，\(U_S = 12V\)，中间支路是负载 \(R_L\)，四个桥臂电阻 \(R_1\)、\(R_2\)、\(R_3\)、\(R_4\) 阻值下图已经给定。（1）求解 \(R_L\) 获得最大功率时的电阻值？（2）计算此时 \(R_L\) 得到的功率？（3）计算此时电源产生的功率？

【解答】首先，求解除了 \(R_L\) 以外的线性含源二端网络的戴维南等效电路，之前例题已经计算出其开路电压 \(U_{oc} = 2V\)，戴维南等效电阻 \(R_o = 5.83Ω\)：

根据最大功率传输定律，当 \(R_L = R_o = 5.83Ω\) 的时候，负载上获得的功率最大。此时 \(R_L\) 得到的最大功率 \(P_{max}\) 为：

\[ P_{max} = \frac{U_{oc}^2}{4R_o} = \frac{2^2}{4 \times 5.83} = 0.17W \]

这样也就可以计算出负载电阻 \(R_L\) 此时的电流：

\[ I = \frac{U_{oc}}{R_o + R_L} = \frac{2V}{5.83Ω + 5.83Ω} \approx 0.17A \]

假设电源 \(U_s\) 上流过的电流由电源正极流向负极，记为 \(I_s\)。求解 \(I_s\) 需要得到 \(R_2\) 和 \(R_4\) 上流过的电流，或者是 \(R_1\) 和 \(R_3\) 上流过的电流，这里将这四个电阻上流过电流的参考方向都标记出来：

接下来，对由 \(R_3\) 和 \(R_4\) 构成的回路列写 KVL 方程 \(10Ω \cdot I_3 + 5Ω \cdot I_4 = 12V\)；以及对结点 b 列写 KCL 方程 \(I_3 + I = I_4\)，联立两个方程进行求解：

\[ \begin{cases} 10Ω \cdot I_3 + 5Ω \cdot I_4 = 12V \\ I_3 + I = I_4 \end{cases} \implies I_4 = 0.91A \]

然后，再对由 \(R_1\) 和 \(R_2\) 构成的回路列写 KVL 方程 \(5Ω \cdot I_1 + 5Ω \cdot I_2 = 12V\)；同样对结点 a 列写 KCL 方程 \(I_2 + I = I_1\)，联立方程求解：

\[ \begin{cases} 5Ω \cdot I_1 + 5Ω \cdot I_2 = 12V \\ I_2 + I = I_1 \end{cases} \implies I_2 = 1.15A \]

根据叠加定理就可以求解得到 \(I_s\) 的值：

\[ I_s = I_2 + I_4 = 2.06A \]

最后，就可以求解出电源的功率 \(P_{US}\) 的值：

\[ P_{US} = -U_s \cdot I_s = -12V \times 2.06A = -24.72W \]

这里 \(P_{US}\) 的值为负值，说明是在提供功率，此时负载电阻 \(R_L\) 所得到功率的百分比为 \(0.69%\)，这个结果远远小于 \(50%\)：

\[ \frac{P_{max}}{P_{US}} = \frac{0.17W}{24.72W} = 0.69% \]

非线性电阻电路

前面的电路分析方法所涉及的都是线性电路，实际上所有电路严格意义上都是非线性的，只是工程上会将非线性程度较低的电路元件进行线性化处理；但是对于非线性化程度较为明显的元件，必须要单独进行讨论。由非线性元件构成的电路称为非线性电路，本小节只讨论非线性电阻电路。

之前介绍过所有电路的分析都是基于元件约束（伏安关系）和拓扑约束（基尔霍夫定律）两类约束，非线性电路的分析依据同样是这两类约束；这里，首先讨论非线性电阻元件的伏安关系。

对于线性电阻而言，其伏安特性曲线是过原点的一条直线，其元件约束满足欧姆定律：

而对于非线性电阻元件，其伏安关系不再是一条过原点的直线，而是符合某种特定的非线性函数。下图所示的非线性电阻元件称为压控型非线性电阻，其表达式为一个电流关于电压的函数 \(i = f(u)\)：

对于下图所示这种伏安关系曲线的非线性电阻而言，其电阻两端的电压都是关于电流的一个单值函数。即对于任何一个电流，都有一个电压与此相对应，这样的线性电阻就称为流控型的非线性电阻，其解析表达式为一个电压关于电流的函数 \(u = f(i)\)：

还有一种非线性电阻元件，其伏安关系特性曲线为单调上升或者单调下降的（例如普通二极管），称为单调性非线性电阻；其即属于电压控制类型也属于电流控制型，因此 \(i = f(u)\) 和 \(u = f(i)\) 两种解析表达都可以适用。

无论哪种类型的非线性电阻元件，其伏安关系曲线关于原点都是不对称的，这样的电阻元件也称为单向性电阻。而线性电阻都属于双向电阻，使用时并不需要区分端钮。

下图当中的电路由非线性电阻元件二极管、电阻、直流电压源构成，其中二极管两端的电压和电流一定是伏安关系曲线上的某个点，这里假设为 Q 点，即静态工作点，该点对应的电压为 \(U_D\) 电流为 \(I_D\)，两者的比值称为静态电阻 \(R=\frac{U_D}{I_D}\)。

如果在上面电路的直流电压源基础之上，再添加一个正弦交流电压 \(\Delta u\)；此时二极管两端的工作电流，就会沿着静态工作点做上下移动。正弦交流电压 \(\Delta u\) 会使得二极管两端的 \(U_D\) 产生一个变化量，记为 \(\Delta U_D\)，其对应的电流记为 \(\Delta I_D\)，两者的比值则称为动态电阻 \(r=\frac{\Delta U_D}{\Delta I_D} = \frac{du}{di}\)。

接下来，以隧道二极管为例来说明非线性电阻的正负性。当隧道二极管两端加的是正向压降时，伏安关系曲线位于第一象限，显然静态电阻总是为正值，而动态电阻处于伏安关系的下降部分时为负值，处于上升部分时为正值。换而言之，隧道二极管加正向电压的情况下，静态电阻总是为正值，而动态电阻可能为正值也可能为负值。

解析法

【例题】设某个非线性电阻元件的伏安特性为 \(u = f(i) = 10i + i^2\)。（1）当\(i_1＝1mA\) 且 \(i_2＝5A\) 时对应的 \(u_1\)、\(u_2\) 的电压？（2）如果 \(u_3 = f(i3)\) ，且 \(i_3 = i_1 + i_2\) 时，等式 \(u_3 = u_1 + u_2\) 是否成立？

【解答（1）】当电流 \(i_1 = 1mA\) 的时候，此时的电压 \(u\) 是 10 倍的 \(i\) 加上 \(i\) 的平方，这样就可以计算出对应的 \(u_1\) 的值：

\[ u_1 = 10 \times 1mA + (1mA)^2 = 11mV \]

当电流 \(i_2 = 5A\) 时，同样将其代入上面的表达式，可以得到对应的 \(u_2\) 的值：

\[ u_2 = 10 \times 5A + (5A)^2 = 75V \]

由此可以看出，如果将非线性电阻近似为一个 \(10Ω\) 线性电阻，当电流 \(i = 1mA\) 比较小的时候，所得到电压的误差较小。如果当 \(i_2 = 5A\) 的时候，如果线性电阻依然保持 \(10Ω\)，就会计算得到 \(u_2\) 的值为 \(50V\)，这个值与实际得到的 \(75V\) 误差较大。

【解答（2）】当 \(i_3 = i_1 + i_2\) 的时候，代入到前面的解析表达式，就可以得到如下方程：

\[ u_3 = 10(i_1 + i_2) + (i_1 + i_2)^2 = i_1^2 + i_2^2 + 2i_1i_2 + 10i_1 + 10i_2 \]

由于 \(u_1 = i_1^2 + 10i_1\) 且 \(u_2 = i_2^2 + 10i_2\)，所以上面的方程可以继续推导为下面形式：

\[ u_3 = u_1 + u_2 + 2i_1i_2 \]

由此可见 \(u_3 \neq u_1 + u_2\)，说明叠加定理仅适用于线性电路，并不适用于非线性电路。这个例题的求解是在已知非线性电阻伏安特性解析表达式的情况下完成的，这种分析方法称为解析法。

【解答】上面电路当中，仅包含一个非线性电阻元件，剩余部分则是一个含源的线性二端网络，这里将该网络进行戴维南等效，分别求解其开路电压 \(U_{oc}\) 和戴维南等效电阻 \(R_o\)。

在非线性电阻开路的情况下，开路电压 \(U_{oc}\) 等于 \(6Ω\) 和 \(3Ω\) 电阻对 \(15V\) 电压源的分压：

\[ U_{oc} = \frac{6Ω}{3Ω + 6Ω} \times 15V = 10V \]

戴维南等效电阻 \(R_o\) 等于含源二端网络当中独立源置零时的等效电阻：

\[ R_o = \frac{3Ω \times 6Ω}{3Ω+6Ω} = 2Ω \]

分段线性法

分段线性法指的是将非线性电阻的伏安特性曲线分段的进行线性化处理，然后再分段进行求解。

【例题】非线性电阻元件和 \(2Ω\) 线性电阻以及 \(10V\) 电压源构成了一个回路，非线性电阻的伏安关系曲线经过线性化以后如下图所示，求解非线性电阻元件两端流过的电流 \(i\) 以及电压 \(u\) ？

【解答】根据非线性电阻经过线性化处理以后的伏安关系曲线，可以分别写出电压和电流的分段函数：

\[ u = \begin{cases} 2i，0 \le i \le 1A \\ 1+i，i \ge 1A \end{cases} \]

根据上面这个解析表达式，就可以对电路进行求解。第一种情况是电流 \(0 \le i \le 1A\)，此时非线性电阻元件的伏安关系为 \(u = 2i\)，即可以将其等效为一个 \(2Ω\) 线性电阻：

对上面这个电路列写 KVL 方程可以得到：

\[ 2Ω \cdot i + 2Ω \cdot i = 10V \implies i = 2.5A \implies u = 2i = 2Ω \times 2.5A = 5V \]

第二种情况是电流 \(i \ge 1A\) 时，此时非线性电阻元件的伏安关系为 \(u = 1 + i\)：

类似的，对上面电路列写 KVL 方程：

\[ 2i + (1 + i) = 10V \implies i = 3A \implies u = i + 1 = 3A + 1 = 4V \]

接下来对上面求解的两组结果进行校验，电流 \(i = 2.5A\) 和电压 \(u = 5V\) 这组解是基于 \(u = 2i，0 \le i \le 1A\) 这个条件求得的，显然这里电流 \(i = 2.5A\) 已经超出了定义域范围，因而这组解并非所需要的结果。而电流 \(i = 3A\) 和电压 \(u = 4V\) 这组解是在 \(u = 1+i，i \ge 1A\) 条件下求得的，电流 \(i = 3A\) 显然满足 \(i \ge 1A\) 这个条件，这组解才是本题的正确结果。
