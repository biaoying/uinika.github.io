现代电路分析涉及的理论与原理

发表于 2020-08-19 分类于 Electronics 阅读次数： 本文字数： 24k 阅读时长 ≈ 22 分钟

《电路分析》主要介绍电路理论的基本知识，也是后续模拟、数字、高频电路的先导基础内容，其主要用于分析电路当中的电磁现象，并采用电流、电压、电量、磁通等物流量描述其过程，从而研究电路的基本规律及其分析方法。其理论与工程实际紧密相连，需要注意电路模型与实际电路的差别与联系，以及测量值与计算值的差别；由于每个电路模型分别具有特定的工作条件，根据工作场景的不同，还会选取不同的电路模型。

本篇文章主要包含了电路模型与电路定律、电阻电路分析、动态电路的时域分析、正弦稳态电路分析等方面的概念，由于电路分析方法本质是数学在电路当中的应用，因此学习过程需要注重将电路分析方法与数学知识相结合，例如向量法当中的复数。全文撰写过程当中，参考并借鉴了国外大学普遍采用的《Electric Circuits》第 10 版教材。

电路的基本概念和分析方法

集总参数电路

电荷在电场力作用下定向移动形成电流，电流通过的路径称为电路。描述电路特征的基本物理量有电流（\(i\)）、电压（\(u\)）、电荷（\(q\)）、磁通（\(\varPhi\)）。

当电路几何尺寸远小于电路当中的信号波长时，电路周围的电磁辐射效应可以忽略不计，此时电流的能量完全分布在金属导线与电路元件上面，这样的电路称为集总参数电路。反之，当电路几何尺寸与信号波长差距较大，电磁辐射效应明显，电路当中各处的电流不一致，此时电流的能量在空间当中离散分布，这样的电路则称为分布参数电路。

例如：假设一个音频放大电路的最高工作频率 \(f＝25kHz\)，其波长：

\[ \lambda = \frac{c}{f} = \frac{3 \times 10^8 m/s}{25 \times 10^3 Hz} = 12km \]

这种情况下，音响设备和音频放大电路的几何尺寸远小于该波长，因此可以被视为集总参数电路。如果这里将工作信号的频率无限提高，则波长会迅速减小。当波长减小至比电路尺寸要小得多时，就需要改用分布参数电路来进行分析。

电流、电压、电功率

电流

电荷（单位为库仑C）的定向运动形成电流，其大小值等于单位时间内通过导体横截面的电荷，采用的国际单位为安培A。

\[ i = \frac{dq}{dt} \]

电路分析时，习惯上将正电荷流动的方向定义为电流方向，然而事实上，电流是由电子的运动所形成。而电流的参考方向则是假定的电流方向，如果假定方向和实际方向一致，则电流量值为正，否则为负。

如果采用双下标方式表达电流的参考方向，就可以写为\(i_{ab}=2A\) 或者 \(i_{ba}=-2A\)。

电压

电场力将单位正电荷从电场中的a点移动到b点所做的功，称为a与b两点之间的电压，可以用数学公式表达如下：

\[ u=\frac{dw}{dq} \]

习惯上，在电场力的作用下，如果正电荷从a点移动至b点失去电场能量，则a点为正极b点为负极。电压分析时，同样引入了参考方向的概念，电压的参考方向采用一对正负号进行表示。

关联参考方向

电流参考方向的流入端为电压参考方向的正极性，称为关联参考方向；反之则称为非关联参考方向。

电功率

单位时间内某一段电路消耗或者提供的能量称为功率，其数学表达式如下所示：

\[ p = \frac{dw}{dt} = \frac{w}{q} \cdot \frac{q}{t} = u \cdot i \]

电压与电流均为代数量，因此作为其乘积的功率也是代数量。而功率正负号的物理意义在于：如果电路中的电压、电流参考方向任意假定，则正负号所代表的物理意义没有共性可言。而在关联参考方向下，功率为正号+时表示吸收功率，为负号-时表示发出功率。

上面的电路图当中，流入 \(1Ω\) 电阻的电流为 \(1A\)，这个电流由 \(1V\) 的电压源提供，则此时流入该电压源的电流就为 \(-1A\)。根据功率的定义，\(1Ω\) 电阻在电压、电流处于关联参考方向的时候 \(P_{1Ω} = ui = 1W\)；而 \(1V\) 电压源在电压、电流处于关联参考方向的时候 \(P_{1V} = ui = -1W\)。

基尔霍夫定律

基尔霍夫定律由德国物理学家 G.R.基尔霍夫在 1845 年提出，分为基尔霍夫电流定律（KCL，Kirchhoff’s Current Law），基尔霍夫电压定律（KVL，Kirchhoff’s Voltage Law）。

  • 支路（Branch）：一个二端元件就可以视为一条支路，其电流和电压分别称为支路电流和支路电压，支路数通常使用字母b表示，上图当中一共拥有 6 条支路。
  • 结点（Node）：两条以及两条以上支路的连接点称为结点，结点数通常使用n表示。上图当中，a、b、c点是结点，d点和e点之间由理想导线连接为 1 个结点，因而该电路一共拥有 4 个结点。
  • 回路（Loop）：由支路所组成的闭合路径，上图当中，包括{1,2}、{1,3,4}、{1,3,5,6}、{2,3,4}、{2,3,5,6}、{4,5,6}在内，一共拥有 6 个回路。
  • 网孔（Mesh）：内部不含有支路的回路，网孔数通常使用m表示，上图包含了 3 个网孔。
  • 广义结点：将电路中的一部分围成一个封闭的平面（高斯面），该封闭平面就称为广义结点。

基尔霍夫电流定律 KCL

对于集总参数电路中的任意一个结点，在任意时刻流出该结点的电流代数和等于零，可以记为如下表达式：

\[ \sum_{k=1}^n i_k = 0 \]

根据定义，可以规定流出结点的支路电流取正号+，流入结点的支路电流取负号-。

\[ -i_1 + i_2 - i_3 + i_4 - i_5 = 0 \]

结点只是理想导线的汇合点，既不可能积累电荷，也不可能消灭电荷。根据电荷守恒原理，结点处的 \(\frac{dq}{dt}\) 必须等于零。

【例题】：已知 \(i_1=1A\)、\(i_7=-2A\)、\(i_{10}=3A\)，试求出下面电路当中 \(i_4\) 的值。

  • 根据第 ① 个结点列写 KCL 方程：\(i_{10} - i_{11} = 0\)
  • 根据第 ② 个结点列写 KCL 方程：\(-i_5 + i_{11} - i_7 = 0\)
  • 根据第 ③ 个结点列写 KCL 方程：\(i_5 - i_2 = 0\)
  • 根据第 ④ 个结点列写 KCL 方程：\(-i_1 + i_4 + i_2 = 0\)

【求解】综上所述，可以求得表达式 \(i_{10} - i_7 + i_4 - i_1 = 3A + 2A + i_4 - 1A = 0\)，进而可以推导出 \(i_4 = -4A\)。

基尔霍夫电压定律 KVL

对于集总参数电路当中的任意一个回路，在任意时刻沿闭合回路绕行一周，各支路电压的代数和等于零。

\[ \sum_{k=1}^n u_k = 0 \]

电荷沿着回路绕行一圈回到原点，其电势能不变，KVL 是电路中能量守恒的体现。

\[ u_1 + u_2 - u_3 = 0 \]

【例题】：已知 \(u_1=1V\)、\(u_2=-5V\)，试求解下面电路当中支路电压 \(u_3\) 的值。

【求解】应用 KVL 定律可以得到 \(u_1 + u_2 - u_3 = 1V - 5V - u_3 = 0\)，从而可以进一步求得 \(u_3 = -4V\)。

【例题】：已知 \(u_2=2V\)、\(u_4=-4V\)、\(u_5=-5v\)，试求解下图中支路电压 \(u_1\) 的值。

【求解】首先，对 \(\{ u_3, u_4, u_5 \}\) 回路列写 KVL 方程 \(u_4 - u_5 - u_3 = -4V + 5V - u_3 = 0\)，从而可以求得 \(u_3 = 1V\)。然后，对 \(\{ u_1, u_3, u_2 \}\) 回路列写 KVL 方程 \(u_1 + u_3 - u_2 = u_1 + 1V - 2V = 0\)，最终可以求得 \(u_1 = 1V\)。

电阻

电阻器在实际电路当中的应用极为广泛，电路分析过程中，电阻元件是根据实际电子元器件特性抽象而来的一种理想化电路模型。例如下图是将三极管视为电阻元件时的特性曲线：

如果一个二端元件在任意时刻的电压 \(u\) 与电流 \(i\) 之间的关系，可以由 U-I 平面上的一条曲线来确定，则该二端元件称为二端电阻元件，其数学表达式为 \(f(u, i)=0\)，并采用下面的电路符号进行表示：

电阻元件根据自身的不同特点，可以划分为不同的类型：

  • 线性电阻与非线性电阻：特性曲线为通过坐标原点直线的电阻称为线性电阻，否则称为非线性电阻。
  • 时变电阻与时不变电阻：特性曲线随时间变化的电阻称为时变电阻，否则称为时不变电阻。

线性非时变电阻的电压电流关系特性曲线是一条过原点，并且不随时间变化的直线。其电压电流关系（VCR，Voltage Current Relation）可以表达为 \(u = Ri\)。

习惯上使用电阻参数 \(R\) 来表示上图当中直线的斜率，其国际单位为欧姆Ω：

\[ R = \frac{u}{i} \]

除此之外，电导参数 \(G\) 也可以用来表达这条直线的斜率，其国际单位为西门子S：

\[ G = \frac{i}{u} \]

根据上述电阻、电导以及功率的定义，可以得到在关联参考方向下，电阻所吸收的功率为：

\[ p = ui = Ri^2 = \frac{u^2}{R} = Gu^2 = \frac{i^2}{G} \]

当正电阻 \(R>0\) 时有 \(p \geq 0\) 表明电阻总是吸收功率，而当负电阻 \(R<0\) 时有 \(p \leq 0\) 表明电阻总是发出功率。这里出现的负电阻是过 U-I 平面第 2、4 象限的一条直线，表示对外发出功率，也称为有源电阻。

注意：虽然实际电阻总是耗能元件，但实际电路当中的一些元件的某个端口可能会呈现负电阻的特性，比如由三极管或运放构成的一些电路，所以才会在电路分析当中引入负电阻的模型。

\[ \begin{cases} p_{1v} = ui = 1A \times 1V = 1W \\ p_{2v} = ui = -1A \times 2V = -2W \end{cases} \]

独立电流源

无论二端元件的电压为何值，其电流保持常量 \(I_S\) 或者按照指定的时间函数 \(i_s(t)\) 变化，则该二端元件称为独立电流源，简称为电流源。

独立电流源的元件 VCR 电压电流关系如下所示：

\[ x = \begin{cases} 端口电流恒定，i = i_s(t) = I_s \\ 端口电压 u 为任意值，由外电路确定。 \end{cases} \]

【例题】求出下面电路图当中，流过 \(1A\) 电流源的电压 \(u\) 的值？

【求解】上图中的 \(1A\) 电流源连接了一个 \(1Ω\) 的电阻，根据欧姆定律该电阻两端的电压为 \(1V\)；根据 KVL 定律，流过电压源两端的电压 \(u\) 应为 \(-1V\)。如果此时将图中 \(1Ω\) 的电阻更换为 \(100Ω\)，此时电流依然为 \(1A\)，但是电阻两端的电压降为 \(100V\)，则相应的 \(1A\) 电流源两端的电压为 \(-100V\)。由此可见，电流源的 \(1A\) 电流始终恒定不变，而两端的电压降随外电路的变化而变化。

【例题】求出下面电路图当中，电流源和电压源吸收的功率 \(p_{1A}\) 和 \(p_{1V}\) 分别为多少？

【求解】根据回路的 KVL 方程，\(1A\) 两端的电压降为 \(-1V\)，由于 \(1Ω\) 电阻两端的电压降为 \(1V\)，则流过该电阻的电流为 \(1A\)。根据电阻上面的结点列写 KCL 方程，可以求出流过 \(1V\) 电压源的电流为 \(0A\)。

\[ \begin{cases} p_{1A} = ui = -1V \times 1A = -1W \\ p_{2v} = ui = 1V \times 0A = -2W \end{cases} \]

元件/拓扑约束与 2b 方程

由不同元件构成的电路所表现出的特性不同，而相同元件不同的连接关系也会呈现出不同的特性。因此，电路包含有元件约束、连接/拓扑约束两类约束关系。

  • 元件约束：由元件的电压电流关系（VCR，Voltage Current Relation）进行描述，反应了电子元件对其所在支路电压、电流所起到的约束关系。
  • 拓扑约束：通过基尔霍夫电流和电压定律，分别反应电路结构和连接特性对支路电流、电压的约束关系。

应用元件和拓扑约束建立的电路方程称为 2b方程，即对于含有b条支路的集总参数电路，以b条支路电压与b条支路电流作为变量，可以建立2b个方程。2b方程是电路分析最为基本的方法，也是后续其它电路分析方法的重要依据。

支路电流法

对于仅仅由独立电压源和线性二端电阻构成的电路，可以将欧姆定律 \(u=Ri\) 代入 KVL 方程，从而消除全部电阻的支路电压，转换为以支路电流为变量的 KVL 方程。叠加原有的 KCL 方程，最终得到以b个支路电流为变量的b个线性无关的方程组，称为支路电流方程。

【求解】以b条支路的电流作为变量列写方程，其依据依然是上一小节中提到的元件和拓扑两类约束关系。

首先，同样针对结点 ①、②、③ 列写 3 个 KCL 方程：

\[ \begin{cases} i_1 + i_4 = 0 \\ -i_1 + i_2 + i_3 = 0 \\ -i_2 + i_5 = 0 \end{cases} \]

然后，同样针对 2 个网孔的列写 2 个 KVL 方程；通过代入 VCR 方程，将上式中的 5 个电压变量替换为 5 个电流变量：

\[ \begin{cases} u_1 + u_3 - u_4 \implies R_1i_1 + R_3i_3 - u_{s1} = 0 \\ u_2 + u_5 - u_3 \implies R_2i_2 + u_{s2} - R_3i_3 = 0 \end{cases} \]

将上述方程整理以后可以得到：

\[ \begin{cases} R_1i_1 + R_3i_3 = u_{s1} \\ R_2i_2 - R_3i_3 = -u_{s2} \end{cases} \]

联立上述 5 个方程，即可以求解 5 个支路电流。如果还需求解某个支路电压变量，可以将其代入欧姆定律，即可求解出相应的电压变量。

支路电压法

对于由线性二端电阻和独立电流源构成的电路，可以采用支路电压作为变量来建立电路方程。

在 2b 方程的基础上，将电阻元件的 VCR 方程 \(i=Gu\) 代入 KCL 方程，将支路电流转换为支路电压，就可以得到(n-1)个以支路电压作为变量的 KCL 方程，叠加原有的(b-n+1)个 KVL 方程，最终可以构成b个以支路电压作为变量的方程，这组方程就称为支路电压方程。

\[ \begin{cases} u_1 + u_{is1} = 0 \\ -u_1 + u_3 + u_2 = 0 \\ -u_2 + u_{is2} = 0 \end{cases} \]

\[ \begin{cases} G_1u_1 + G_3u_3 = i_{s1} \\ G_2u_2 - G_3u_3 = -i_{s2} \end{cases} \]

联立上述方程，即可求解出全部的电压变量。如果当前需要求解的是支路电流，则相应的代入欧姆定律即可。

含源线性电阻单口等效

仅有两个端钮与外部电路相连接的网络，称为二端网络。如果只强调二端网络的端口特性，而不关心网络内部的情况，则称二端网络为单口网络，通常简称为单口。

单口电阻的特性由端口电压电流关系 VCR 来进行表征（表现为 U-I 平面上的一条曲线）。当两个单口网络的 VCR 关系完全相同时，则称这两个单口相互等效。

\[ \begin{aligned} & u_1 = \frac{R_{31}(R_{12}+R_{23})}{R_{12}+R_{23}+R_{31}} i_1 + \frac{R_{23} R_{31}}{R_{12} + R_{23} + R_{31}} i_2 \\ & u_2 = \frac{R_{23} R_{31}}{R_{12} + R_{23} + R_{31}} i_1 + \frac{R_{23}(R_{12}+R_{31})}{R_{12}+R_{23}+R_{31}} i_2 \end{aligned} \]

观察电阻三角形联接和星形联接的 VCR 方程，如果要求它们等效，那么 2 个 VCR 方程的对应系数必须分别相等：

\[ \begin{aligned} & R_1 + R_3 = \frac{R_{31}(R_{12}+R_{23})}{R_{12}+R_{23}+R_{31}} \\ & R_3 = \frac{R_{23}R_{31}}{R_{12}+R_{23}+R_{31}} \\ & R_2 + R_3 = \frac{R_{23}(R_{12}+R_{31})}{R_{12}+R_{23}+R_{31}} \end{aligned} \]

整理上述方程，可以得到如下关于 \(R_1\)、\(R_2\)、\(R_3\) 的等式：

\[ \begin{aligned} & R_1 = \frac{R_{31}R_{12}}{R_{12} + R_{23} + R_{31}} \\ & R_2 = \frac{R_{12}R_{23}}{R_{12} + R_{23} + R_{31}} \\ & R_3 = \frac{R_{23}R_{31}}{R_{12} + R_{23} + R_{31}} \end{aligned} \]

综上所述，电阻三角形联结等效变换为电阻星形联结的公式为：

\[ R_i = \frac{连接至 i 端 2 个电阻的乘积}{三角形连接 3 个电阻之和} \]

注意：当三角形网络 3 个电阻相等的时候 \(R_{12} = R_{23} = R_{31} = R_{\Delta}\) ，就可以得到星形网络 3 个电阻的阻值也是相等的 \(R_1 = R_2 = R_3 = R_Y = \frac{1}{3}R_{\Delta}\)。

同样的，由上面 VCR 方程的等效系数等式，可以推导出星形联结方式转换为三角形连接方式的电阻阻值公式。

\[ \begin{aligned} & R_{12} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_3} \\ & R_{23} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_1} \\ & R_{31} = \frac{R_1R_2 + R_2R_3 + R_3R_1}{R_2} \end{aligned} \]

或者，也可以书写为电导的形式：

\[ \begin{aligned} & G_{12} = \frac{G_1G_2}{G_1+G_2+G_3} \\ & G_{23} = \frac{G_2G_3}{G_1+G_2+G_3} \\ & G_{31} = \frac{G_3G_1}{G_1+G_2+G_3} \end{aligned} \]

综上所述，电阻星形联结等效变换为电阻三角形联结的公式为：

\[ R_{mn} = \frac{每两个星形网络的电阻乘积之和}{不与该电阻 mn 端连接的电阻的阻值} \]

注意：当星形网络 3 个电阻相等（对称）的时候 \(R_{1} = R_{2} = R_{3} = R_{Y}\) ，可以得到三角形网络电阻的阻值等于 3 倍星形网络电阻的阻值 \(R_{\Delta} = 3 R_Y\)。

本小节所描述的等效，是针对外部电路的等效，而并不针对内部电路，其主要用途是用于简化电路分析过程。

网孔分析法

前面章节内容已经讨论了b条支路n个结点的一般求解方法，我们可以采用两类约束列写 2b 方程，从而求解出全部的支路电压和电流。为了减少联立方程组的数目，也讨论了以b个支路电流为变量的支路电流法，进一步减少了联立方程组数量。如果要更进一步减少联立方程个数，则需要考虑使用本小节将要介绍的网孔分析法。

下面电路当中有5条支路3个结点，可以列写2个独立的 KCL 方程。如果知道某3个支路的电流变量，那么剩余2条支路的变量，就可以通过2个 KCL 方程进行求解。比如先求解出 \(i_1\)、\(i_2\)、\(i_4\)，那么通过结点 KCL 方程就可以得到 \(i_3\) 和 \(i_5\)；再比如 \(i_1\)、\(i_2\)、\(i_5\) 同样就可能会求解出 \(i_3\) 和 \(i_4\)。因此，可以认为这组数量为 \(b-n+1\) 个的电流变量是独立（自动满足 KCL 约束）和完备（通过该组变量可以求解剩余所有支路电流变量）的。

接下来，本小节将要讨论的问题是如何找出b-n+1个完备、独立的支路电流变量，找到以后依据什么来建立方程，并且进一步求解出这组变量？

网孔电流方程

网孔电流是一种沿网孔边界闭合流动的假想电流，对于拥有b条支路n个结点的平面连通电路而言，其总共拥有(b-n+1)个网孔电流，它们是能够确定全部支路电流的独立电流变量。

网孔电流与支路电流密切相关，比如外围支路电流 \(i_1\) 与网孔电流 \(i_m\) 相同（参考方向不一致数值可能相反），而公共支路 \(i_5\) 可以由两个网孔共同来确定。下面的等式列写出了上图中支路电流与网孔电流之间的关系：

\[ \begin{aligned} i_1 = i_{1m} \\ i_2 = i_{2m} \\ i_3 = i_{3m} \\ i_4 = i_{1m} + i_{3m} \\ i_4 = i_{1m} + i_{2m} \\ i_4 = i_{2m} + i_{3m} \end{aligned} \]

根据上面的等式，即然外围支路可以由网孔电流唯一确定，那么就没有必要去增设一个新的变量来表征这个电路的变量，而可以直接使用外围支路的电流 \(i_1\)、\(i_2\)、\(i_3\) 来表征网孔电流 \(i_{1m}\)、\(i_{2m}\)、\(i_{3m}\) ，这样就找到了一组独立完备的电流变量，接下来讨论如何建立方程的问题。

图示网孔电流方向为绕行方向，写出三个网孔的 KVL 方程分别为：

\[ \begin{cases} R_1i_1 + R_5(i_1+i_2)+R_4(i_1+i_3)=u_{s1} \\ R_2i_2 + R_5(i_1+i_2)+R_6(i_2-i_3)=u_{s2} \\ R_4(i_1+i_3)+R_3i_3-R_6(i_2-i_3)=-u_{s3} \end{cases} \]

对上述 3 个方程做数学变换，将电流变量前面的系数提取出来，整理成为如下形式：

\[ \begin{cases} (R_1+R_5+R_4)i_1 + R_5i_2 + R_4i_3=u_{s1} \\ R_5i_1 + (R_2+R_6+R_5)i_2 - R_6i_3=u_{s2} \\ R_4i_1 - R_6i_2 + (R_4+R_6+R_3)i_3=-u_{s3} \end{cases} \]

上面方程左侧的系数是有规律可循的，\(i_1\)、\(i_2\)、\(i_3\) 前面的系数 \((R_1+R_5+R_4)\)、\((R_2+R_6+R_5)\)、\((R_4+R_6+R_3)\) 称为各自网孔的自电阻，其余变量前面的系数则是 2 个相邻网孔的互电阻，即两个网孔公共电阻的阻值。如果两个相邻的网孔电流以相同方向流过交界处电阻，则互电阻系数取正值+，如果以相反方向则取负值-。上述方程右侧的\(u_{s1}\)、\(u_{s2}\)、\(-u_{s3}\) 表示该网孔电压源电压升的代数和。

将上面的网孔方程进一步改写为更加一般的形式：

\[ \begin{cases} R_{11}i_1 + R_{12}i_2 + R_{13}i_3 = u_{s11} \\ R_{21}i_1 + R_{22}i_2 + R_{23}i_3 = u_{s22} \\ R_{31}i_1 + R_{32}i_2 + R_{33}i_3 = u_{s33} \end{cases} \]

上式当中 \(R_{11}\)、\(R_{22}\)、\(R_{33}\) 称为网孔的自电阻，将对角线上的 \(R_{kj}\) 称为互电阻，如果网孔绕行方向全部取一致，那么互电阻系数为负值。等式右边的 \(u_{s11}\)、\(u_{s22}\)、\(u_{s33}\) 则是各自网孔全部电压源电压升的代数和，如果电压降就取负号-，如果电压升则取正号+。

最后，将网孔分析法的计算步骤总结如下：

  1. 在电路图上标注网孔电流及其参考方向，如果全部网孔电流的参考方向一致，即全部选择顺时针或者逆时针方向，则网孔方程所有互电阻项全部取负号-；
  2. 观察电路图，依据上述总结的系数规律，直接列写各个网孔方程；
  3. 联立求解网孔方程，得到所有的网孔电流变量；
  4. 如果需要求解其它支路电流，可以假设支路电流的参考方向，然后根据支路电流与网孔电流的线性组合关系进行求解；
  5. 如果还需要求解支路电压，则根据元件电压电流关系的 VCR 方程，求解得到各条支路的电压；

\[ \begin{cases} (\frac{1}{1Ω} + \frac{1}{2Ω})u_1 - \frac{1}{2Ω} \cdot u_2 = 6A \\ -\frac{1}{2Ω}v_1 + (\frac{1}{2Ω} + \frac{1}{1Ω} + \frac{1}{2Ω})u_2 - \frac{1}{2Ω}u_3 = 0 \\ -\frac{1}{2Ω}u_2 + (\frac{1}{2Ω}+\frac{1}{2Ω}+\frac{1}{2Ω})u_3 = 9A \end{cases} \implies \begin{cases} (1+0.5)u_1 - 0.5u_2 = 6A \\ -0.5u_1 + (0.5+1+0.5)u_2 - 0.5u_3 = 0 \\ -0.5u_2 + (0.5+0.5+0.5)u_3 = 9A \end{cases} \]

【求解】这种情况需要增设一个流过该电压源的电流变量 \(i\)，结点 1 与参考结点之间有一个电压源支路，所以结点 1 的电压就已知等于 \(u_1 = 14V\)；结点 2 拥三条支路，其中两条为电导支路，其中一条电导支路跨接在结点 1 和 2 之间，所以对结点 2 建立的方程为 \(-u_1 + 1.5u_2 = 3A - i\)；结点 3 同样拥有三条支路，其中一条为跨接在结点 1 和 3 之间的公共支路，所以列写出的方程为 \(-0.5u_1 + 1.5u_3 = i\)。

\[ \begin{cases} u_1 = 14V \\ -u_1 + 1.5u_2 = 3A - i \\ -0.5u_1 + 1.5u_3 = i \\ u_2 - u_3 = 8V \end{cases} \]

\[ \begin{cases} 0.2u_1 = 10A - i_1 \\ (0.5+0.25)u_2 = i_1 + 5A \\ u_1 - u_2 = 3i \\ i = 0.5u_2 \end{cases} \]
