电路分析涉及的理论与原理

发表于 2020-05-21 分类于 Electronics 阅读次数： 本文字数： 46k 阅读时长 ≈ 42 分钟

《电路分析》主要介绍电路理论的基本知识，也是后续模拟、数字、高频电路的先导基础内容，其主要用于分析电路当中的电磁现象，并采用电流、电压、电量、磁通等物流量描述其过程，从而研究电路的基本规律及其分析方法。其理论与工程实际紧密相连，需要注意电路模型与实际电路的差别与联系，以及测量值与计算值的差别；由于每个电路模型分别具有特定的工作条件，根据工作场景的不同，还会选取不同的电路模型。

本篇文章主要包含了电路模型与电路定律、电阻电路分析、动态电路的时域分析、正弦稳态电路分析等方面的概念，由于电路分析方法本质是数学在电路当中的应用，因此学习过程需要注重将电路分析方法与数学知识相结合，例如向量法当中的复数。全文撰写过程当中，参考并借鉴了国外大学普遍采用的《Electric Circuits》第 10 版教材。

基本概念与规律

电路模型

实际电路：由实际电子元器件（电阻器、电容器、电感线圈、晶体管、集成电路等）互相连接而构成的物理系统，主要包括如下部分：

  • 电源：产生电能和电信号的设备，例如：电池、信号发生器等；
  • 负载：吸收电能和电信号的设备，例如：电阻、灯泡等；
  • 中间环节：连接电源和负载并对电路的工作状态进行控制，例如：传输线、开关、继电器等；

电路模型：由理想电路元件（电阻元件、电容元件、电感元件、独立源、受控源等）与理想导线相互连接而成。

不同工作条件下，实际电子元件可以被表达为不同的电路模型。例如下的简单照明电路，如果只考虑灯泡对于电能的消耗，则可以将其表现为一个电阻元件R；如果还考虑到交流电路当中的灯泡还具有电感效应，则还需要同时将其考虑为电感元件R + L。

电路分类

集总参数假设

集总参数条件：如果实际电路尺寸 < 电路当中电磁信号的波长，即可以认为电磁过程集中在电子元件内部，而与空间位置无关。

集总参数电路：满足集总参数条件的实际电路，其电路模型就可以被认为是集总参数电路。

例如：我国电力系统工频交流电的频率为50Hz，那么其波长根据光速计算可得6000km；由于实际电路尺寸远远小于波长，可以认为满足集总化条件。

\[ \lambda = \frac{c}{f} = \frac{3×10^{8}}{50} = 6000km \]

分布参数电路：不满足集中化条件的实际电路，其物理量取值与时间、空间位置相关。

例如：远距离输电当中，上千公里的输电线就不可能满足集中化条件，因而属于分布参数电路。

其它分类方式

  1. 线性与非线性电路： 线性电路：仅由线性元件（元件电磁特性可以采用线性约束进行描述，例如：理想的电阻、电感、电容元件）组成的电路； 非线性电路：电路中至少含有一个非线性元件；
  2. 时变与非时变电路： 非时变电路：所有元件的参数都不随时间变化； 时变电路：至少有一个元件的参数随时间变化；
  3. 无源与有源电路： 无源电路：电路中所含元件均为无源元件（自身不消耗电能，只需要输入信号，无需外加电源即可正常工作）； 有源电路：电路含有源元件（自身消耗电能，除输入信号以外，必须使用外加电源才能正常工作）；

电流与电压

电路变量用于对电路模型进行描述，电路分析的主要任务就是通过求解特定变量进而了解电路特性。进行电路分析时，常见的电路变量有电流、电压、能量、功率、电荷、磁链等。

电流

电流：单位时间t内通过导体某横截面的电荷量q。

\[ i = \frac{dq}{dt} \]

电流采用安培A作为标准单位，将正电荷移动的方向作为电流的方向。

直流电流：电流的大小和方向都不会随时间变化，记为大写字母I。

时变电流：电流的大小或者方向随着时间变化，记为i(t)，简写为小写字母i。

电压

单位正电荷\(q\)在电路两点之间运动时，电场力所做的功\(w_{ab}\)。

\[ v_{ab} = \frac{dw_{ab}}{dq} \]

电压采用伏特V作为标准单位，方向由高电位流向低电位。

电位：指某点至参考点之间的电压，其取值与参考点的选择有关系，记为\(v_x\)。

a点到零电位参考点c的电压为\(v_a = \frac{d(w_{ab} + w_{bc})}{dq}\)，而b点到c点的电压为\(v_b = \frac{dw_{bc}}{dq}\)，那么a点到b点的电位差则为\(v_{ab} = \frac{dw_{ab}}{dq} = v_a -v_b\)。

直流电压：电压的大小与方向不随时间变化，记作大写字母V。

交流电压：电压的大小或者方向会随时间变化，记作小写字母v(t)，简写为v。

理想情况下，a点到c点之间没有电流，但是由于电压是两点之间电场力做的功，因此两点之间可以认为是存在电压的。面对集总参数电路中的能量是守恒的，因此以上电路中abc和abdc两条不同路径上，电场力所做的功是相同的，所以\(v_{ac}\)也就是唯一的。

参考方向

当难以确定电压和电流的实际方向时，可以任意假定其方向，并采用正负号或者下标法进行表示。如果电流或电压的数值为正，则实际方向与参考方向一致，如果为负，则实际方向与参考方向相反。参考方向概念的引入，主要是为了便于将数学运算引入到电路分析当中。

注意：一旦指定电压和电流的参考方向之后，列写电路方程时不能再进行改变！

一个元件的电流、电压的参考方向可以独立的任意指定，如果指定流过元件的电流参考方向是从标有电压正极性+的一端流向负极性-一端，即电压电流的参考方向一致，那么就称为关联参考方向，反之则称为非关联参考方向。例如下图当中，\(v\)的参考方向与\(i_1\)的参考方向关联，而与\(i_2\)的参考方向非关联。

功率与能量

功率：电场力单位时间t所做的功，用符号p表示，单位为瓦特W。

\[ p(t) = \frac{dw}{dt} \]

能量：指的是功率的积分，用符号w表示，单位为焦耳J。

\[ w(t) = \int_{t_0}^{t}p({\tau})d{\tau} \]

下图当中，对于元件的a与b两端，电压参考方向由a指向b，分别根据电流\(i=\frac{dq}{dt}\)和电压\(v=\frac{dw}{dq}\)的定义列写方程，从而推导出功率的定义\(v \times i = \frac{dw}{dq}\times\frac{dq}{dt} = \frac{dw}{dt} = p\)。

下图左侧元件的电压v和电流i为关联参考方向，则该元件吸收的功率为\(p = v \times i\)；右侧元件的电压v和电流-i为非关联参考方向，则该元件吸收的功率为\(p = v \times -i\)。

对于下图当中的电阻而言，电压与电流为关联参考方向，此时电阻吸收的功率为\(p_{1Ω} = v \times i = 1瓦\)；而对于电压源而言，电压与电流为非关联参考方向，因此电阻吸收的功率为\(p_{1v} = v \times -i = -1瓦\)，即向外释放功率。

基尔霍夫电流定律

基尔霍夫电流定律（Kirchhoff’s Current Law, KCL），电路上任意一个节点在任意时刻，流出/流入的电流代数和为零，列写为KCL 方程如下：

\[ \sum_{k=1}^{N} {i_k} = 0 \]

通常情况下，将流出节点的电流取为+，流入节点的电流取为-。例如下面的示意图里，根据 KCL 方程可知\(i_1 + (-i_2) + (-i_3) + (-i_4) + i_5 = 0\)，由此可推导出未知电流\(i_5 = -i_1 + i_2 + i_3 + i_4 = -6+2+(-3)+1 = -6A\)。

接下来继续列写出下面电路图对应的全部 KCL 方程，并进一步判断其是否都是相互独立的。

上图当中，\(i_3\)、\(i_4\)、\(i_5\)对应的节点可以合并处理，最终得到如下 3 个节点：

  • 节点 1：\(i_1 + i_2 = 0\);
  • 节点 2：\(-i_1 + i_3 + i_4 - i_5 = 0\);
  • 节点 3：\(-i_2 - i_3 - i_4 + i_5 = 0\);

其中，节点2和节点3相加并且化简以后，得到的依然是节点1的方程式，因此可以认为 3 个方程线性相关，去掉其中 1 个才能成为独立方程组。由此可以得出结论，对于具有n个节点的电路，独立的 KCL 方程将会有n-1个。

广义 KCL：KCL 不但对于节点成立，对于任意闭合的曲面也能够成立。即在任何时刻，所有流出/流入电路任意闭合曲面的电流代数和为零，即\(\sum i_{流入} = \sum i_{流出}\)。

由上面的示意图，可以进行如下符合广义 KCL 原理的推导，即流入曲面的电流等于流出的电流：

\[ i_1 + i_2 – i_3 + i_4 – i_5 = 0 \implies i_1 + i_2 + i_4 = i_3 + i_5 \]

如果把电流的流出视为正流入视为负，根据 KCL 方程可知下图中节点a的流入电流为\(i_3 = 7 - 2 = 5A\)。

根据广义 KCL，将图中黄色标注区域视为一个曲面。根据 KCL 定律\(i_1 + i_2 + i_3 = 0\)，可以得到\(i_1 = -7A\)。

基尔霍夫电压定律

基尔霍夫电压定律（Kirchhoff’s Voltage Law, KVL），任意时刻沿着电路的任意回路，所有相邻节点之间的电压代数和为零。

\[ \sum_{k=1}^{M} {v_k} = 0 \]

电压参考方向与环绕路径方向一致取+，否则取-。

根据 KVL 定义，整理上图当中各个支路的电压，并列写为下面的推导等式：

\[ v_1 - v_2 + v_3 + v_4 – v_5 = 0 \implies v_1 + v_3 + v_4 = v_2 + v_5 \]

对于含有b条支路和n个节点的电路，独立 KVL 方程有b-n+1个，也即网孔的个数。接下来，写出下图所示电路上所有的 KVL 方程，并判断它们是否相互独立。

  • 回路 1 ：\(– v_3 – v_2 + v_1 + v_4= 0\)；
  • 回路 2 ：\(– v_4 + v_5 + v_6 = 0\)；
  • 回路 3 ：\(– v_3 – v_2 + v_1 + v_5 + v_6 = 0\)；

将回路 1与回路 2相加，可以得到回路 3对应的 KVL 方程。由于每个方程都可以由另外两个方程线性推导出来，所以上述 3 个方程并不是互相独立的。

广义 KVL：KVL 不仅适用于支路组成的回路，还可以适用于虚拟回路。由于a和c两点之间无支路连接，因此也就不存在电流。根据前面所学的内容，两点之间可以存在电压，因此如果a和c两点之间电压为\(v_0\)，则可以假想它们之间存在一个电压为\(v_0\)的元件，那么包含有\(v_0\)的闭合路径就被称为虚拟回路。

下图当中，已知\(v_0=1V\)、\(v_4=2V\)、\(v_5=3V\)，求其它的电压值。

对于abca回路，由 KVL 定律推导可得：

\[ v_1 + v_5 – v_0 = 0 \implies v_1 = v_0 - v_5 = 1 - 3 = -2v \]

对于abda回路，由 KVL 定律推导可得：

\[ v_1 + v_2 – v_4 = 0 \implies v_2 = -v_1 + v_4 = 2 + 2 = 4v \]

对于acd和abdc这两个虚拟回路，分别通过 KVL 定律推导\(v_3\)：

\[ v_4 + v_3 – v_0 = 0 \implies v_3 = -v_0 + v_4 = -1 + 2 = 1v \]

\[ v_1 + v_2 – v_3 – v_0 = 0 \implies v_3 = -v_0 + v_1 + v_2 = 1v \]

由此可见，无论以哪个闭合路径的回路列写方程并求解\(v_3\)，其结果都是唯一的，与所选取的路径没有关系，这个结论体现了能量的守恒。

理想电源

理想电源是对实际电源的一种理想化抽象，其定义特性与外电路无关，因而具有独立性，也被称为独立源。

理想电压源：任意时刻，元件两端的电压为一个确定值，经过的电流取决于外电路。

理想电流源：任意时刻，流过元件的电流为一个确定值，产生的电压同样取决于外电路。

接下来计算下面各图当中，各个元件所吸收或者释放的功率。

根据 KVL 方程可知下图中的\(v = 10V\)，由于电阻的v和i是关联参考方向，因此\(i = \frac{10v}{5Ω} = 2A\)；求解出所有电压和电流以后，由此就可以推导出每个元件吸收的功率。电压源的电压、电流处于非关联参考方向，因此其输出的功率为\(p_{10V} = -v \times i = -20W\)，电阻的电压、电流处于关联参考方向，因此其吸收的功率为\(p_{5Ω} = v \times i = 20W\)。

根据 KVL 方程以及电流的方向，可以首先得出\(v = 10V\)并且\(i = -2A\)；此时，左侧10V电压源的电压与电流处于非关联参考方向，因此其消耗的功率为\(p_{10V} = -v \times i = -10 \times (-2) = 20W\)；而右侧2A电流源的电压、电流处于关联参考方向，因此输出功率应为\(p_{5Ω} = v \times i = 10 \times -2 = -20W\)。

上图左侧电压源、中间的电阻、右侧的电流源相互并联在一起，这里需要首先求出每个元件的电压与电流，然后再得到每个元件的功率。由于10v电压源与5Ω电阻相互并联，因此可知\(v = 10V\)。而该电阻的电压电流是关联参考方向，因此\(i_2 = \frac{v}{5} = 2A\)；根据 KCL 方程当中流入节点的电流等于流出该结点的电流，可以求解出\(i_1 + 2 = 2 \implies i_1 = 0A\)；

接着求出上图当中每个元件的功率，对于左侧电压源而言，因为\(i_1 = 0A\)，所以\(p_{10V} = -v \times i_1 = 0W\)；而中间电阻的功率\(p_{5Ω} = v^2 / 5 = 20W\)；由于并联关系，右侧电流源的电压也为10v，而其与2A电流为非关系参考方向，所以\(p_{2A} = -v \times 2 = -20W\)。综上所述，电路中向外输出功率的是2A的电流源（由于其功率为负值），消耗功率的是5Ω的电阻元件，而10v电压源既不消耗也不输出能量。

理想电压源的串并联

理想电压源是可以串联的，串联之后可以形成一个新的电压源，其输出电压为各个电压源之和\(v_s = v_{s1} + v_{s2}\)。

理想电压源是不可以并联的，理想电压源并联之后，就会构成一个回路，如果此时\(v_{s1} \not = v_{s2}\)，那么就违反了 KVL 定律；而如果\(v_{s1} = v_{s2}\)则会导致两个电压源的电流解不唯一。

理想电流源的串并联

理想电流源是可以并联的，并联之后会形成一个新的电流源，其输出电流等于各电流源之和\(i_s = i_{s1} + i_{s2}\)。

理想电流源是不可以串联的，串联后形成下图的红色结点，如果此时\(i_{s1} \not = i_{s2}\)，由于输入总电流不等于输出总电流，所以违反了 KCL 定律；而如果\(i_{s1} = i_{s2}\)则会导致两个电流源的电压解不唯一。

实际电源

实际电压源

实际电压源的外部特性曲线是一条倾斜的直线，端电压\(V_s\)随着端电流\(i\)的增加而逐步衰减。

由 KVL 定律可知，当两个元件相串联时，总电压等于两个元件电压的代数和，由此可以推断出一个实际电压源的模型，是由理想电压源\(V_s\)和理想电阻\(R_0\)（称为内阻，该值越小，电压源就越接近理想特性）串联组合而成。

如果将实际电压源的端口短路时，此时上图中\(v = V_s - R_0 \times i = 0\)，由此可以推导出\(i = \frac{V_s}{R_0}\)。如果此时\(R_0 \Rightarrow 0\)，那么\(i \Rightarrow \infty\)，由此可以看出实际电压源内阻较小，不可以进行短路，否则会烧毁电源设备。

实际电流源

实际电流源的外部特性曲线同样是一条倾斜的直线，端电流\(I_s\)随着端电压\(V\)的增加而逐渐减小。

当两个元件并联时，电阻\(R_0\)的电流等于\(\frac{V_s}{R_0}\)，基于 KCL 方程可得\(i = I_s - \frac{V_s}{R_0}\)，即实际电流源的模型是由理想电流源\(I_s\)和理想电阻\(R_0\)（称为内阻，越大实际电流源就越接近理想电流源特性）并联构成。

实际电压源的并联

当\(V_{s1} \not = v_{s2}\)的时候该回路不符合 KVL 定律，所以理想电压源是不能并联的。

但是分别由\(V_{s1}\)和\(R_1\)、\(V_{s2}\)和\(R_2\)组成的实际电压源并联结构里，即使该回路里\(V_{s1} \not = v_{s2}\)，两个电压之差也将会由\(R_1\)和\(R_2\)两个电阻进行分担，因而并不违反 KVL 定律，所以实际电压源是可以并联的。

注意：为了接近理想电压源的效果，通常会将\(R_1\)和\(R_2\)尽量变得更小，越是接近理想电压源，就越需要避免将其进行并联。

实际电流源的串联

理想电流源是不可以进行串联的，如果中间的指定结点上\(i_{s1} \not = i_{s2}\)不相等就违反了 KCL 定律。

但是分别由\(i_{s1}\)和\(R_1\)、\(i_{s2}\)和\(R_2\)组成的实际电流源串联结构里，即使\(i_{s1} \not = i_{s2}\)不相等，两个电流的差会由\(R_1\)和\(R_2\)两个电阻所承担，因此并不违返 KCL 定律，所以实际电流源是可以串联的。

注意：为了接近理想电流源效果，通常会将\(R_1\)和\(R_2\)尽量变得更大，越是接近理想电流源，就越需要避免将它们进行串联。

开路与短路

开路：无论两点之间的电压为多少，电流都恒等为零，可以分别采用阻值无穷大的电阻、0 值电流源、断开的理想开关进行表示。。

短路：无论两端通过的电流是多少，其电压都恒等为零，可以分别采用阻值为 0 的电阻、0 值电压源、闭合的理想开关进行表示。

注意：实际开关在使用过程当中，闭合状态下存在有限的导通电阻，断开状态则存在有限的截断电阻，与理想状态下的开路和短路存在一定差别。

受控源

通过前面学习到的电阻元件、电压源、电流源三个理想元件，可以构造出诸如实际电压源或者实际电流源等模型。

但是对于大规模集成电路上广泛使用的MOSFET（金属-氧化物场效应晶体管），由于是具备控制功能的三端元件，现有的理想元件无法有效体现其实际模型，因此这里引入能够体现控制关系的受控源作为元件模型。

受控源是表示电路中两个端口变量之间控制关系的元件模型，其电压/电流受到其它支路上的电压/电流控制，其中电压/电流和控制量成比例关系的就称为线性受控源。

被控端模型

受控源的输出可以是电压也可以是电流，如果输出的是电压（输出与端口电流无关），则被称为受控电压源；其符号只是将独立电压源符号当中的圆圈更换为菱形。

如果输出的是电流（输出与端口电压无关），则被称为受控电流源，其符号同样只是将独立电流源符号上的圆圈更换为菱形。

受控端模型

受控源的控制关系是单方向的， 当控制变量是某个支路上的电压时，可以在该支路两端上引出 2 根线并标出 2 个端子，这两个端子构成的端口就是控制端，端子之间的电压\(v_1\)就是控制电压。由于此时端口是断开的，因此控制端口的电流\(i_1\)等于0。

如果控制变量是某个支路上的电流时，可以将该条支路上的理想导线拐弯以后，再标注 2 个端子从而构成控制端口，端口电流\(i_1\)就是控制电流。 由于两个端子是通过一条理想导线进行连接的，因此因此控制端口的电压\(v_1\)就等于0。

受控源完整模型

由于受控源的控制端和受控端可以是电压或者电流，因而可以具体将受控源分为 4 种：

电压控制电压源 VCVS

此时受控源是受控电压源，输出电压是\(v_2\)，控制量是某条支路上的电压\(v_1\)，此时\(v_2 = \mu \times v_1\)，其中\(\mu\)为电压增益倍数。

电压控制电流源 VCCS

控制量依然是某一条支路上的电压\(v_1\)，受控源更换为受控电流源，输出电流为\(i_2\)，此时\(i_2 = g \times v_1\)，由于\(i_2\)与\(v_1\)并非同一端口上的变量，因此这里将\(g\)称为转移电导。

电流控制电压源 CCVS

受控源为受控电压源，控制量是某一条支路上的电流\(i_1\)，此时\(v_2 = r \times i_1\)，同样由于两者并非相同端口的变量，这里将\(r\)称为转移电阻。

电流控制电流源 CCCS

受控源为受控电流源，控制量依然是某一条支路上的电流\(i_1\)，输出电流\(i_2 = \alpha \times i_1\)，其中\(\alpha\)称为电流增益。

注意：由于受控源的控制端口，要么是开路，要么是短路，因而电路绘制时通常不会画出控制端口，使得受控源看起来像一个二端元件一样，但实质上它是一个四端元件。

构建 NMOSFET 模型

根据受控源的模型，就可以构建出 NMOSFET 的模型，该模型当中，\(r_{gs}\)表示栅缘电阻，\(r_{ds}\)表示漏缘电阻，漏极电流\(I_d\)由\(I_{ds}\)以及电压控制电流源的\(g_m V_{gs}\)共同控制。

受控源与独立源的区别

受控源的电源值依赖于其它变量，一般不能单独作为电路的激励源，因此受控源也称为非独立源。但是，受控源与独立源一样，电压源的电压与受控端口电流无关，电流源的电流与受控端口的电压无关。

支路变量法

如果一个电路包含n个节点b条支路，由于每条支路都会含有电压和电流变量，因此电路中一共会含有 \(2 \times b\) 个变量。对于含有 n 个节点的电路而言，可以列写 n-1 个独立 KCL 方程以及 \(b-(n+1)\) 个（同时也是网孔的个数）；由于一个元件就是一条支路，那么b条支路就含有b个元件的伏安关系。

正如上图黑色虚线框中所描绘的，一个电路总共可以列写 \(\lbrace n-1 \rbrace + \lbrace b-(n-1) \rbrace + \lbrace b \rbrace = 2b\) 个独立方程，利用两类约束关系可以完全确定所有支路上的电压和电流，为了减小变量个数，可以只将b个支路上的电压作为求解变量，这种方法称为支路电压法；此外也可以将b条支路上的电流作为求解变量，这种方法称为支路电流法。

根据 KVL 定律：\(v_3 = v_1 + v_2 = 6V + 6V ＝ 12V\)；根据 KCL 定律：\(i_3 = 3A - i_2 = 1.5A\); 由此得到电阻 \(R_3 = \frac{v_3}{i_3} = 8Ω\)。

惠斯顿电桥

电阻分压应用到了惠斯顿电桥当中，英国科学家查尔斯·惠斯顿于 1843 年利用其精确的测量电阻，下图当中，\(R_1\)和\(R_2\)为已知阻值电阻，\(R\)是用精密电位器实现的可变电阻（其阻值可以准确读出），\(R_x\)是待测量的未知电阻，\(G\)为检流计。

当电桥达到平衡的时候，检流计\(G\)的读数为0，这意味该支路没有电流流过，根据开路的定义，可以将该支路断开。此时将会发现\(R_1\)和\(R\)以及\(R_2\)和\(R_x\)皆为串联结构，这里假定\(R_1\)和\(R_2\)的电压分别为\(V_1\)和\(V_2\)，根据电阻分压公式，可以直接写出\(V_1\)的表达式\(v_1 = \frac{R_1}{R_1 + R} \times v_s\)，以及\(V_2\)的表达式\(v_2 = \frac{R_2}{R_1 + R_x} \times v_s\)。

受控源与独立源的区别

受控源与独立源的相同点：

  • 受控电压源的电流为任意值；
  • 受控电流源的电压为任意值；
  • 受控源可以吸收或放出能量；

受控源与独立源的不同点：

  • 受控源的电源值依赖于其它变量，并不能独立向外提供能量；
  • 受控源在特定条件下具有电阻的特性；

线性电路分析方法

线性电路的齐次性

如果研究的电路只包含有线性元件和独立源，那么该电路就可以认为属于线性电路。

由于线性元件对电路变量施加着线性约束，根据电路定律和元件线性约束关系所列写的方程，就可以认为是线性方程，具体可以表现为代数方程、微分方程、积分方程。

\[ y = L(x) \]

线性电路性质

  1. 齐次性：电路的输入 \(x\) 变化 \(k\) 倍时，电路的输出同样变化 \(k\) 倍；

    \[ L(kx) = kL(x) \]

  2. 叠加性：当信号 \(x_1\) 和 \(x_2\) 同时输入时，电路的输出为两者分别单独输入时，电路的输出之和，也称为可加性；

    \[ L(x_1 + x_2) = L(x_1) + L(x_2) \]

将线性电路的齐次性和叠加性综合起来，可以得到更为简洁的线性电路特性表示形式，称为线性性：

\[ L(k_1x_1+k_2x_2)=k_1L(x_1)+k_2L(x_2) \]

齐次定理

当线性电路只有一个输入的时候，电路当中各处的电流与电压均与激励输入呈现相同的线性比例变化。一种较为特殊的情况是，如果当前讨论的线性电路属于电阻性电路（仅包含电阻与受控源）时，则会呈现一种较为特殊的比例关系。这就是齐次定理的内容，具体可以采用如下公式（常数 \(k\) 是比例系数）进行表示：

\[ y = kx \]

接下来通过一个由 3 个电阻和 1 个受控电压源组成的电路分析实例，对齐次定理的有效性进行验证。

  • 根据左侧回路列写 KVL 方程：\(4Ω \times i_1 + v - v_s = 0\)；
  • 根据右侧回路列写 KVL 方程：\(-3v + 12Ω \times i_2 - v = 0\)；
  • 根据欧姆定律：\(v = 6Ω \times i_3\)；
  • 对上图顶部中间的节点列写 KCL 方程：\(i_3 = i_1 - i_2\)；

联立求解上述方程以后，可以得到任意支路上的电压电流：\(i_1 = \frac{1}{6} v_s\)、\(i_2 = \frac{1}{9} v_s\)、\(v_1 = \frac{2}{3} v_s\)、\(v_2 = \frac{4}{3} v_s\)、\(v = \frac{1}{3} v_s\)；

观察以后可以发现，上述的每个电压与电流都与外加的激励 \(V_s\) 有关系，通通都可以采用 \(支路变量 y = k \cdot 外加激励x\) 进行表示，其中比例系数 \(k\) 取决于电路的结构与参数，与外加激励的输入没有关系。

齐次定理的应用

继续以上面的电路图为例，来求解 \(v_2\) 和 \(v_s\) 之间的关系。

齐次定理告诉我们：对于电阻性电路而言，当电路只外加一个激励 \(v_s\) 的时候，电路中任意支路的电压和电流都是与 \(v_s\) 呈比例关系，即 \(i_s = k \cdot v_s\)。由此可以知道，如果能首先求出比例系数 \(k\) ，就可以得到外加激励为 \(v_s\) 的时候，\(i_2\) 对应的值。

\[ i_s = k \times v_s \]

由上面的式子，我们可以假定 \(i_2\) 的值，然后倒推出对应的激励 \(v_s\) 的值，最后求得比例系数 \(k\)。

首先，假设 \(i_s = 1A\)，根据欧姆定律 \(v_2 = 12V\)。然后对上图右侧的回路列写 KVL 方程可得 \(3v + v = v_2 \implies v = \frac{v_2}{4Ω} = 3V\)，根据欧姆定律\(i = \frac{v}{6Ω} = \frac{3V}{6Ω}= 0.5A\)。

对上图顶部中间的节点列写 KCL 方程 \(i_1 = i + i_2 = 0.5A + 1A = 1.5A\)，再由欧姆定律可以得到 \(v_1 = 4Ω \times i_1 = 6V\)。对左侧回路列写 KVL 方程 \(v_s = v_1 + v = 9V\)，那么根据齐次定理可以推导出比例系数 \(k_{i_2} = \frac{i_2}{v_s} = \frac{1}{9}\)，即：

\[ i_s = \frac{1}{9} \times v_s \]

根据上述分析过程可以发现，运用齐次定理可以简化电路变量的求解过程，通过简单的口算就能得出电路的响应。以上内容就是\(i_2\)的求解，运用同样方法可以求解出其它电路变量与\(v_s\)之间的关系，而无需再度推导比例系数 \(k\) 的值，因为前面已经将各个支路的变量推导出来了。

例如：当 \(v_s = 9V\) 的时候，\(i_1 = 1.5A\)，而两者的比例关系 \(k_{i_1} = \frac{i_1}{v_s} = \frac{1.5A}{9V} = \frac{1}{6} \cdot v_s\)；同样的，当 \(v_s = 9V\) 的时候，\(v = 6V\)，则两者之间的比例系数为\(\frac{6V}{9V} = \frac{2}{3} \implies v_1 = \frac{2}{3} \cdot v_s\)。依照相同的方法，可以继续求得 \(v = \frac{3V}{9V} \cdot v_s\) 、\(v_2 = \frac{12V}{9V} \cdot v_s = \frac{4}{3} \cdot v_s\)。

当电路中，除了电压源 \(v_s\) 之外，还包含有一个电流源 \(i_s\) 作为激励。

此时根据 KCL 定理可得 \(i_2 + i_s = i_3\)，然后分别对左侧的回路列写 KVL 方程 \(4Ω \cdot i_1 + v = v_s\)，以及中间的回路列写 KVL 方程 \(-v - 3v + 12Ω \cdot i_3 = 0\)，最后根据欧姆定律得到 \(v = 6 \times (i_1 - i_2)\)。

联立求解之后可以观察到，无论是支路的电流还是电压，电路的响应都是由 \(v_s\) 和 \(i_s\) 两个分量构成。

此时，电路的响应可以分解为如下的 \(y_1\) 和 \(y_2\) 两个部分：

这里会发现 \(y_1\) 等于电压源 \(v_s\) 单独作用时电路的响应，而 \(y_2\) 等于电流源 \(i_s\) 单独作用时电路的响应，即电路中的响应等于电压源和电流源分别单独作用时所产生响应分量的叠加，将这个概念进行更为一般化的推广以后就得到了叠加定理：在含有多个独立源的线性电路里，每条支路上的电压或者电流，都可以看成是各个独立源分别单独作用时该支路上产生的电压或者电流的代数和。

叠加定理在应用的时候需要注意以下几点：

  1. 叠加定理只适用于线性含独立源的电路；
  2. 叠加定理中的独立源单独作用，是指保留一个独立源，让剩余的独立源置0。由于电压源的电压值等于0相当于短路，电流源的电流值等于0相当于开路。所以，这里的独立源置0是指电压源短路，电流源开路；
  3. 当电路中含有受控源的时候，由于受控源依赖于独立源存在，所以受控源只能作为无源元件处理，而不能参予叠加。受控源不能单独作用于电路，也不能置0。
  4. 叠加定理只对电压和电流变量成立，功率并不服从叠加定理。

接下来通过一个例子，重新阐述一下叠加定理。下面电路当中，分别拥有电压源 \(V_s\) 和电流源 \(I_s\)，而 \(N_0\) 则是一个内部不含独立源的线性电路，利用叠加定理表示流过电阻的电流 \(I\)。

由前面讲解的叠加定理可以知道，电流 \(I\) 是由 \(I^{'} = k_1 \cdot V_s\)（电压源单独作用电流源置零，即开路）和 \(I^{''} = k_2 \cdot I_s\)（电流源单独作用电压源置零，即短路）组成。

\[ I = I^{'} + I^{''} = k_1 \cdot V_s + k_2 \cdot I_s \implies y = \sum_{i=1}^{n} {k_i} {x_i} \]

如果一个电路含有 n 个独立源，每条支路上的电压或者电流，都可以表示为各个独立源电源值的加权和。

等效变换分析法

利用等效思想，可以只关注电路的外部特性，忽略其内部细节，从而简化电路的求解过程，正式讲解之前先介绍几个基本概念：

二端子电路

电路分析过程中，为了方便求解，经常需要将一个复杂的电路分解为若干个简单的子电路，然后再通过导线将这些子电路连接起来。比如下图左侧的电路，利用 KCL 定律求解电流 \(I_1\) 的时候，需要将该电路分为右侧的 2 个子电路 \(N_1\) 和 \(N_2\)，两者通过 3 条导线进行连接，这些用于连接各个子电路的导线就被称为端子。

实际电路当中，较为常用的是具有 2 个端子的子电路，称为二端电路或者二端网络。比如下图左侧所示电路的蓝色部分，可以通过子电路 \(N\) 进行表示，此时 \(N\) 可以通过 2 个端子与外加激励 \(i_s\) 相连，成为了一个二端电路。此时，在端子上标注出端口电压和电流的参考方向，就构成了一个标准的二端电路符号。又由于二端电路的 2 个端子构成了 1 个端口，所以二端电路有时也被称为单口电路或者单口网络。

电路当中，二端电路通过其端口的伏安特性关系，影响剩余电路，比如上图左侧电路里，二端电路 \(N\) 与外加激励 \(i_s\) 直接相连，此时 \(N\) 就会强制 \(i_s\) 的电压与电流满足其自身的端口伏安关系。某些电路分析问题当中，比如说在求解 \(i_s\) 的电压的时候，可以将 \(N\) 视作一个黑箱而不必关注其内部电路，只需要得到 \(N\) 的端口伏安关系，再将端口电流等于 \(i_s\) 这个条件，代入至 \(N\) 的伏安关系中，就可以求得 \(N\) 的端口电压，即 \(i_s\) 的电压。

串联等效变换

两个二端元件或者二端电路串联以后构成一个新的二端网络。

根据串联电路的特点，流过二端元件或者电路的电流均与端口电流 \(i\) 相等，而端口电压 \(v\) 就等于各二端元件或者电路的端电压之和：

\[ \begin{cases} i_1 = i_2 = i \\ v = v_1 + v_2 \end{cases} \]

端口电压 \(v\) 等于端口电流 \(i\) 乘以n个电阻的阻值之和，观察这个伏安特性关系可以知道，端口电压与电流的比值为一个常数，符合电阻的特点。因此，n个电阻串可以等效为一个电阻，其阻值就等于n个电阻的阻值之和。

电流源与任意子电路串联

另一种情况是串联元件当中包含了一个电流源，由于是串联电路，所以下图中 \(N\) 的电流与端口电流均应该等于电流源的电流 \(i_s\)，电流源的电压则是任意的，需要由外电路来确定。这些伏安关系正好符合电流源的特点，因此电流源与任意其它元件串联就可以等效为一个电流源，其中的 \(N\) 就被称为无效伴随网络。

并联等效变换

两个二端元件或者二端电路并联以后构成一个新的二端网络。

根据并联电路的特点，各个二端元件或者电路的端电压等于端口电压 \(v\)，端口电流 \(i\) 就等于各二端元件或者电路的端电流之和：

\[ \begin{cases} v_1 = v_2 = v \\ i = i_1 + i_2 \end{cases} \]

电阻的并联等效

n个电阻的并联构成一个新的二端电路，其端口电流等于n个电阻的电流之和，由于各并联电阻的电压都等于端口电压 \(v\)，那么每个电阻的电流就等于其电导乘以电压 \(v\)，那么电阻并联后端口的伏安特性关系可以表示为：端口电流 \(i\) 等于端口电压 \(v\) 乘以各个电阻的电导之和：

观察这个伏安特性关系可知，端口电压与端口电流的比值为一个常数，符合电阻的特点。所以，n个电阻并联就可以等效为一个电阻，等效后电阻的电导值等于这n个电阻的电导之和。

电流源的并联等效

当并联的元件都是独立源时，前面提到过理想电压源是不能并联的，所以这里只讨论电流源并联。n 个电流源并联构成一个新的二端网络，其端口电流为 n 个电流源的和，由于电流源的端口电压 \(v\) 是任意的，主要由外电路来决定。这里端口的伏安特性关系，符合电流源的定义，即n个电流源并联可以等效为一个电流源，其电流值就等于n个电流源电流的代数和。

注意：之所以这里强调是代数和，是因为这n个电流源的方向有可能不同，此时就需要相减。

电压源与任意子电路并联

如果并联元件里有一个是电压源，即该电压源与任意子电路 \(N\) 相并联（由于理想电压源不能并联，所以 \(N\) 不能为电压源）。由于是并联连接，\(N\) 的电压和端口电压都应该等于电压源电压 \(v_s\)。由于电压源的电流是任意的，需要由外电路来决定，所以端口电流 \(i\) 等于 \(N\) 的电流加上电压源的电流，其结果也是任意的，同样需要外电路来进行确定。这一伏安特性关系符合电压源的特性。

当电压源与任意一个电压源以外的元件或者电路并联的时候，就可以等效为一个电压源，其中 \(N\) 也被称为无效伴随网络。假设等效前后电压源电流分别为 \(i_1\) 和 \(i_2\)，由于等效前后电压源的电流发生了变化（等效前 \(i_1 = i - i_N\)，但在等效后 \(i_2 = i\)），所以 \(i_1 \ne i_2\)。

直接列写二端网络的端口伏安关系

加压求流法

如果在一个二端网络端口接入电压源 \(v\)，就会产生一个端口电流 \(i\)，从而求解出电压电流关系 \(v = ki + b\)，根据这个v-i关系可以画出对应的等效电路，即一个电压源串联上一个电阻。其中电压源的电压值就是该v-i关系的常数b，电阻的阻值就是v-i关系当中的比例系数k，这样的方法就被称为加压求流法。

加流求压法

如果在一个二端网络端口接入电流源 \(i\)，从而产生一个端口电压 \(v\)，也可以求解出两者的关系 \(i = kv + b\)，根据这个关系也可以绘制出对应的等效电路，即一个电流源并上一个电阻。 其中，电流源的电流值b等于v-i关系中的常数b，电阻的阻值等于v-i关系中的比例系数k的倒数，这种方法就被称为加流求压法。

输入电阻

齐次定理小节曾经留下过一道思考题，即一个不含独立源，而只包含受控源与电阻的二端电路，可以等效为什么元件？

这里采用加压求流法来求解二端电路的等效电路，在端口添加电压源 \(v\)，从而产生电流 \(i\)。

由线性电路的齐次定理可以知道，当电路中只含有一个独立源 \(v\) 的时候，端口电流 \(i\) 就会与该电压源 \(v\) 成比例关系：

\[ i = k \cdot v \implies R_i = \frac{v}{i} = \frac{v}{k \cdot v} = \frac{1}{k} \]

戴维南与诺顿定理

戴维南定理

前面的内容学习了线性电路的等效变换，任意一个线性含独立源的二端电路，都总是可以通过等效变换将其化简为一个电压源与电阻的串联或者一个电流源与电阻的并联。

这样结论的得出，是必然还是偶然的呢？本小节就将从理论层面揭示这偶然背后的必然性，从而推导出线性电路的普适性等效简化方法，这个普适性的方法就是戴维南定理：任意一个线性含独立源的二端网络 \(N\) 均可等效为一个电压源 \(V_{oc}\) 与一个电阻 \(R_0\) 相串联的支路，其中 \(V_{oc}\) 为该网络的开路电压，\(R_0\) 为该网络中全部独立源置零后的等效电阻。

上图就是俗称的戴维南等效电路，接下来讨论如何证明戴维南定理：根据等效的定义，如果下图左右两边电路的伏安关系是相同的，就可以认为这两个电路是等效的，对于右侧电路列写 KVL 方程 \(v = V_{oc} + R_0 \cdot i\)，接下来的任务就是要证明左侧网络的端口伏安关系也可以通过这个等式来表达。

前面小节内容提到过，求解一个网络的端口伏安关系，可以采用加压求流或者加流求压的方法，这里采用加流求压法，即在二端网络 \(N\) 的端口添加电流源 \(i\) 并求端口电压 \(v\)。二端网络 \(N\) 的内部可能含有多个电源，此外还有外加的电流源 \(i\)，对于含有多个电源的电路，通常会采用叠加定理来求解变量。

首先让 \(N\) 单独作用，外加的激励源开路置零，假设此时电压 \(v\) 的分量为 \(v_1\)，然后再让外加的电流源 \(i\) 单独作用，此时 \(N\) 置为 \(N_0\)，且端口电压为 \(v_2\)。

根据叠加定理可以知道 \(v = v_1 + v_2\)，而 \(v_1\) 是二端网络 \(N\) 端口断开时的端口电压，其实也就是戴维南等效电路里的开路电压 \(V_{oc}\)；再来看 \(v_2\)，利用线性电路齐次定理，\(N_0\) 的端口电压 \(v_2\) 与外加激励 \(i\) 呈比例关系，表明 \(N_0\) 可以等效为一个电阻，根据欧姆定律可以得到其阻值 \(R_0 = \frac{v_2}{i}\)，从而可以推导出 \(v_2 = R_0 \cdot i\)。

\[ v = v_1 + v_2 = V_{oc} + R_0 \cdot i \]

由此可见，上面左右两个电阻的伏安关系相同，也就同时意味着两个二端网络等效，戴维南定理也就是成立的。对于任意一个二端网络，只要它是线性的，就会存在叠加，存在叠加就会存在戴维南定理，因而其具备较强的普适性。

诺顿定理

戴维南定理定理告诉我们：任意一个线性含源的二端网络，都可以等效为一个电压源 \(V_{oc}\) 与一个电阻 \(R_0\) 的串联，观察这个等效电路，会发现这正好是一个实际电压源的模型，实际电压源模型与实际电流源模型之间可以相互转化，压源串阻可以等效为流源并阻。等效后电流源的电流 \(I_{sc} = \frac{V_{oc}}{R_0}\)，这个并联形式的等效电路就是诺顿等效。

诺顿定理是指任意线性含独立源的二端网络均可等效为一个电流源 \(I_{sc}\) 与一个电阻 \(R_0\) 相并联的支路，其中 \(I_{sc}\) 为该网络的短路电流，\(R_0\) 为该网络中全部独立源置零后的等效电阻。

诺顿定理说明了诺顿等效所涉及的两个参数的求解方式，其中 \(R_0\) 与戴维南等效一样，指的是二端网络内部独立源全部置零以后的等效输入电阻，\(I_{sc}\) 为该网络端口的短路电流，之所以是短路电流，加入上图右侧的蓝色方框表示任意线性含源的二端网络，那么通过诺顿定理可知：

该二端网络可以采用一个电流源 \(I_{sc}\) 与一个电阻 \(R_0\) 的并联来等效，如果再在端口添加一条短路线，显然在端口短路的时候，诺顿等效电阻 \(R_0\) 的电压与电流皆为零，因此电流源的电流 \(I_{sc}\) 将会全部流入到端口上添加的短路线上面，因此端口短路电流就等于 \(I_{sc}\)，而方向与电流 \(i\) 的方向完全相反。这说明当端口短路时，端口短路电流就是诺顿等效中的 \(I_{sc}\)，由电源模型的等效变换可知，戴维南等效与诺顿等效之间可以相互转化，对于同一个二端网络，当可以求解其戴维南等效时，也就同时可以轻松的确定其诺顿等效，反之亦然。

注意：这里存在一种特殊情况，当求解出二端网络的等效输入电阻等于零或者无穷的时候，即 \(R_0 = 0\)（理想电压源）或者 \(R_0 = \infty\)（理想电流源）时，就不能进行这样的等效。因为当 \(R_0 = 0\) 的时候，该二端网络是理想电压源，所以只会存在戴维南等效。当 \(R_0 = \infty\) 的时候，该二端网络为理想电流源，所以只存在诺顿等效。

可见，并不需要复杂的理论计算，只需要两步简单的测量方法，就可以求出 \(V_{oc}\) 和 \(R_0\) 两个等效参数。

最大功率传输定理：对于任意给定的线性有源二端网络，其负载获得最大功率的条件是负载电阻等于二端电路戴维南等效内阻，此时也称为最大功率匹配。

求解当负载实现最大功率传输时，电源功率的传输效率将会等于多少？接下来看下面的例子：求解图示电路负载可能获得的最大功率，以及匹配时电源功率的传输效率？

首先，对 \(V_{oc}\) 进行求解，这里需要断开负载电阻，于是下面电路 ab 两端的电压（即10Ω电阻的电压）就是 \(V_{oc}\)。

这个电路由 1 个电压源和 3 个电阻构成一个简单的单回路电路，由分压原理可以求解10Ω电阻的电压：

\[ V_{oc} = \frac{10Ω}{2Ω+8Ω+10Ω} \times 20V = 10V \]

然后，求解 \(R_0\) 的值，需要将二端网络内部独立源全部置零，这里将20V电压源置零，此时电路变为2Ω串联8Ω然后并联10Ω电阻的电路，于是可以求解：

\[ R_0 = \frac{(2Ω + 8Ω)\times{10Ω}}{(2Ω + 8Ω) + {10Ω}} = 5Ω \]

根据最大功率传输定理，当负载 \(R_L = R_0 = 5Ω\) 时所获得的功率最大，就等于：

\[ P_{Lmax} = \frac{ {V_{oc}}^2 }{4 \times R_0} = 5W \]

接下来研究最大功率传输的时候，电源功率的传输效率。先来计算在最大功率传输的时候，20V电压源所发出的功率。为了计算这个功率，首先应该计算出其电流 \(i_1\)。当最大功率匹配的时候，负载电阻等于 \(5Ω\)，此时 \(i_1\) 就等于总电压20V除以总电阻\(\frac{(2Ω+8Ω+10Ω) \times 5Ω}{2Ω+8Ω+10Ω+5Ω} = 4Ω\) 等于

\[ I_1 = \frac{20V}{10 + 10 / 3} \approx 1.5A \]

于是可以求出电源发出的功率 \(P_s\)：

\[ P_s = 20V \times I_1 = 30W \]

最后可以求出在最大功率匹配的时候，电源功率的传输效率 \(\eta\)：

\[ \eta = \frac{P_L}{P_S} = \frac{5W}{30W} = 16.7% \]

可见在最大功率传输条件下，电源功率的利用率不高，有83.3%被负载以外的电阻消耗掉。因此，最大功率传输定理，并不适用于电力系统，而更加适用于电子信息领域，因为该领域常常着眼于从微弱信号中获得最大功率，而不看重效率的高低。

节点分析法

前面的小节学习了线性电路的一些分析方法，这些方法可以有效求解电路里各支路的变量，然而随着电路结构越来越复杂，电路元件越来越多，电路方程的列写与求解也会变得越来越困难，这种背景下就需要采用更为高效与更具有规则的电路求解方法。接下来的内容，就会介绍电路理论当中较为规范的两种分析方法：节点分析法、网孔分析法，本小节主要介绍节点分析法。

节点分析法选取节点电压为变量，所谓节点电压是指电路中各节点相对于参考节点的电压，对于一个含有n个节点的电路而言，其节点电压的个数为n-1个。

为了克服已有方法的缺点，节点分析法应该具备如下 3 个特点：

  1. 完备性：新方法可以求解任意电路的任意支路的电流、电压变量；
  2. 独立性：所选取的变量之间应该相互独立，以确保变量的个数最少，从而使得方程的个数最少；
  3. 规范化：新方法的方程列写应当有固定规律可循，可用于计算机的辅助分析；

节点电压是否具有完备性，如果能求解出电路中所有节点电压，可否求解出电路中各支路的电压与电流？以上面电路为例，当求解出各个节点电压的时候，就可以求出各个元件的电压（各元件的电压等于元件两端的节点电压之差）。根据欧姆定律可以求出电阻的电流，而电流源的电流又是已知的，因此节点电压可以求出所有支路的电压与电流，从而能够满足完备性要求。

如果节点电压不具备独立性，就会出现诸如电压相加为零的情况，但是会得到这种结果吗？这里来分析一下，电压求解通常采用 KVL 定律对图中任意回路列写 KVL 方程，例如 \(G_2\)、\(G_3\)、\(I_{s1}\)、\(I_{s2}\) 构成的回路，可以得到如下 KVL 方程：

\[ (V_a - V_b) + (V_b - V_c) + (V_c - V_d) + (V_d - V_a) = 0 \]

  3. \(I_{sii}\): 方程右侧为电流，其值等于所有流入到该节点的电流源电流之和，流入为正，流出为负。
  4. 节点方程可以采用矩阵的形式进行表示，左侧关于节点电压的系数，可以用系数的矩阵进行表示。如果电路当中不存在受控源，那么这个系数矩阵就是对称的 \(G_{ij} = G_{ji}\)。

运放电路分析

运算放大器简称运放，是一种由多个晶体管组成的多端有源器件，具有高增益、高输入电阻、低输出电阻等特点。可以用于放大、模拟计算机的数学计算、滤波、电压电流转换、比较器等功能。下面是德州仪器公司生产的μA741运算放大器的实物与功能引脚图。

运算放大器

运算的电路符号采用一个三角形来表示，其拥有 5 个端口，其中a和b称为输入端（+a称为同向输入端，-b称为反向输入端，因此运放的输入 \(v_d = v_+ - v_-\)）；c称为输出端，其输出电压 \(V_o = A \cdot v_d\)，其中 \(A\) 称为开环增益，表示的是运放的放大倍数，一般可以达到 \(10^5 ~ 10^8\) 之间。d和e则是两个电源端，分别用 \(VCC+\) 与 \(VCC-\) 进行表示，运放属于有源器件，没有电源供电就无法正常工作；这两个电源端口可以有多种不同的供电模式，为了简化电路结构通常省略不画，从而得到只包含 3 个端子的运放电路符号。

注意：运算放大器的电压都是指对地电压。

外部特性

如果采用运放的输入 \(v_d\) 表示横坐标，输出 \(v_o\) 表示纵坐标。根据 \(v_o = A \cdot v_d\) 可以采用一条过原点的倾斜直线来表示运放的外部特性曲线，而这个倾斜直线的斜率就是 \(A\)。因此，运放的输出 \(v_o\) 随着 \(v_d\) 的增加而线性增长，当 \(v_d > +d\) 的时候，\(v_o\) 就不再伴随 \(v_d\) 的增加线性增长，而是变成一条平行于横轴的直线，即 \(v_o = V_{s1}\)，这里 \(V_{s1}\) 被称为正向饱和电压。

同样，运放的输出 \(v_o\) 随着 \(v_d\) 的增长而线性减小，当 \(v_d < \-d\) 的时候，\(v_o\) 也变成了一条平行于横轴的直线，即 \(v_o = -V_{s2}\)，这里 \(-V_{s2}\) 通常称为反向饱和电压。

运算放大器的外部特性曲线可以分为 3 个区域：

  • 线性工作区 \(v_d < d, v_o = A \cdot v_d\)；
  • 正向饱和区 \(v_d > d, v_o = v_{s1}\)；
  • 反向饱和区 \(v_d < -d, v_o = -v_{s2}\)；

运放的负反馈

直接放大方式

利用运放实现输入信号的放大，最直观的方式是采用直接放大，即将运放的反向输入端接地，同向输入端与输入信号 \(v_i\) 相连，如果运放工作于线性放大区，运放的输出 \(v_o\) 就等于 \(A \cdot v_d = A \cdot v_i\)。

但是采用这种直接放大的方式会带来诸多问题，运放的开环增益 \(A\) 如果很大，从而导致其线性工作区间很窄，例如一个运放正向饱和电压 \(V_{s1} = 10V\)，开环增益 \(A = 10^6\)，在上图中间的运放外部曲线上取线性放大区间的边界点作为观察点，由于该边界点依然处于线性放大区，所以在该边界点上 \(v_o = A \cdot v_d\)，其中 \(v_d\) 就是线性放大区的最大值 \(+d\)，而 \(v_o\) 恰好等于正向饱和电压 \(V_{s1}\)，于是可以推导出 \(d = \frac{V_{s1}}{A}\)，即 \(d = 10uV\)。该结果意味着输入信号 \(v_i\) 如果位于 \(+/-10uV\) 之外的话，运放就会脱离线性放大区，进入正向饱和区，无法实现放大功能。

假定这里的 \(v_i\) 是一个正弦信号，当 \(v_i\) 的值稍微大于10uV时，运放工作于正向饱和区，运放的输出 \(v_o\) 就恒等于 \(+V_{s1}\)。同样，当 \(v_i\) 的值稍微小于10uV时，运放工作于反向饱和区，运放的输出 \(v_o\) 就恒等于 \(-V_{s2}\)，这样以来就无法正常实现信号的放大，因为将输入信号的变化幅度限制到mV数量级以下即不合理也不现实，所以直接放大这种方式在现实生活当中是无法使用的。

其次，直接放大的放大倍数与 \(A\) 直接相关，而 \(A\) 容易受到温度等其它因素的影响，从而导致整个放大电路的放大倍数不稳定，这是直接放大方式所带来的另外一个问题。解决这些问题，通常采用的方法就是负反馈。

负反馈方式

负反馈是通过电子元件将运算放大器的输出引入至反相输入端，其产生的反馈电压会抵抗运放输出电压的变化趋势。例如下面电路就采用了 2 个电阻构成的分压电路作为负反馈，运算放大器的输出电压 \(v_o\) 经过分压电路反馈输入到其反向输入端 \(v-\)。

假设 \(t<0\) 的时候，运放是没有输入的，理想情况下输出也同样为零，即 \(v_d = v_o = 0\)。在 \(t = 0\) 的时刻，电路接入了直流电压源的输入信号 \(V_{in}\)，即运放的输入 \(v_d = V_{in}\)。

由于运放的开环增益 \(A\) 非常大，运放的线性工作区间非常窄，当 \(V_{in}\) 加入到运放的同向输入端以后，运放的输入电压 \(v_d\) 将会位于线性区间以外，此时运放有进入到正向饱和区的趋势，因此当 \(t > 0\) 的时候，\(v_o\) 将从零朝着正向饱和电压进行变化。

前一节曾经提到，运放的两个输入电流通常被近似认为等于零，对于两个电阻的公共节点向左的电流为零，两个电阻可以认为是串联结构，又由于两个电阻总电压为 \(v_o\)，根据电阻分压原理，每个电阻所分得的电压就是 \(v_ = \frac{v_o}{2}\)，由于 \(v_d = v_+ - v_-\)，其中 \(v_+\) 恒等于 \(v_{in}\) 是一个常量，而 \(v_- = \frac{v_o}{2}\)，这意味 \(v_-\) 会伴随 \(v_o\) 的增加而增加，这也就导致随着 \(v_o\) 的增加 \(v_d\) 逐渐减小。

总结一下，对于上面的运放而言，左侧的输入端 \(v_d\) 将从 \(v_{in}\) 逐渐减小，而右侧的输出端 \(v_o\) 将从零逐渐增加。一边减小一边增加，那么是否存在一个时刻 \(v_o = A \cdot v_d\)？如果这个等式满足，则称 \(t_0\) 为新的平衡点。

接下来求证一下等式是否满足，假设存在 \(t_0\) 时刻，该时刻下 \(v_o = A \cdot v_d = A(V_{in} - \frac{v_o}{2})\)，将 \(A\) 换算到等号左边，由于 \(A\) 的数值非常巨大，因此可以认为 \(\frac{v_o}{A}\)，这就意味着 \(V_{in} - \frac{v_o}{2} \approx 0\)，由此可以推导出 \(v_o \approx 2 \cdot V_{in}\)。

下面回到电路当中，在新的平衡点上，运放的输入端电压 \(v_d\) 应该等于多少，由于 \(v_d = v_{in} - \frac{v_o}{2}\)，而 \(v_o \approx 2 \cdot V_{in}\)，那么 \(v_+ - v_- \approx 0\)，由上述分析过程可知，采用负反馈电路，不但可以实现输入信号的放大，还能保证运算放大器工作于线性放大区。避免将输入信号的幅度限制到uV微伏以下。

此外，负反馈还可以抵抗输出电压的变化趋势，使输出电压维持在平衡点上面。这里依然以上面的电路为例，上面已经推导出在新的平衡点 \(v_o = 2 \cdot v_i\)。假设在某个时刻，由于某种原因，运放在输出电压 \(v_o\) 略微高于 \(2 \cdot v_i\)，即 \(v_o\) 出现增加的趋势。当 \(v_o\) 增加时，反馈电压 \(v_-\) 就会增加，由于 \(v_d = v_+ - v_-\)，所以 \(v_d\) 也会减小。由于运放的开环增益非常巨大，此时其开环增益会立刻做出调整，使得输出电压 \(v_o\) 下降，直至输入端满足 \(v_+ \approx v_-\)。此时，运放就相当于一个高灵敏的电压比较器，时刻检查输出的反馈电压与输入电压之间的误差，并通过调整输出电压来消除这种误差，从而实现将输出电压保持输入电压的特定比值附近。

接下来讨论下一个问题，利用负反馈能否解决直接放大所带来的另一个问题，即电路放大倍数与 \(A\) 直接相关。首先求解下图左侧电路当中，输出电压 \(v_o\) 与 \(v_i\) 的比值 \(\frac{v_o}{v_i}\)，该比值通常被称为闭环增益，这里我们将电路中的运放利用受控原模型进行等效，可以得到下图右侧的电路。

在右侧电路当中，\(v_o\) 就等于 \(A \cdot v_d\)，运放的两个输入电流 \(i_+\) 和 \(i_-\) 可以认为等于零，于是两个电阻具有相同的电流，可以认为它们是串联结构。由于总电压是 \(v_o\)，因此每个电阻所分得的电压就是 \(\frac{v_o}{2}\)，于是可以求解出 \(v_d = v_i - \frac{v_o}{2}\)，将其代入至 \(v_o = A \cdot v_d\) 等式当中，就可以求解出 \(v_o\) 与 \(v_i\) 的关系，

上图中间的关系式当中，由于 \(A\) 的取值较大，因此可以近似的将 \(\frac{2}{A} \approx 0\)，于是最后推导出 \(v_o \approx 2 \cdot v_i\)。由这个推导过程可见，只要 \(A\) 足够大，电路的闭环增益就与 \(A\) 没有关系，该闭环增益只取决于负反馈网络两个电阻的阻值，该特点对于实际电路的分析与设计非常有利。

综上分析可知，负反馈电路不但可以保证运放工作于线性放大区，而且还会使电路的闭环增益与 \(A\) 无关。

理想运放模型

前面学习的运放称为有限增益运放，虽然其开环增益 \(A\) 取值较大，但依然具有一个具体值，如果假设这个开环增益 \(A\) 为无穷大，则该运放可以被称为理想运放。两者在符号上的区别，主要就是少了字母 A。

有限增益运放的线性范围是一条倾斜的直线，斜率是开环增益 \(A\)，当将 \(A\) 设置为无穷大的时候，理想运放的外部特性曲线就如上图右下侧所示，其线性放大区是一条位于Y轴上的直线，这表明当理想运放工作于线性放大区时，理想运放的输入电压 \(v_d\) 就等于零，即 \(v_+ - v_- = 0\)。实现这个等式需要依靠负反馈电路，拥有负反馈以后，运放的输出就与其开环增益无关，而取决于外部的反馈电路，因而理想运放对于电路变量的约束仅体现在输入端。下面就会介绍理想运放在输入端的电路特性：

虚断：如果输入电阻 \(R_i\) 为无穷大，则运放的两个输入端电流 \(i_+\) 和 \(i_-\) 等于零，就好像运放的两个输入端子是断开的一样，这个特性就称为虚断。 虚短：当理想运放工作于线性放大区的时候，理想运放的输入电压 \(v_d = \frac{v_o}{A}\)，而 A 被视为无穷大，因此 \(v_+ \cdot v_- = 0\)，即 \(v+ = v-\)，此时运放的两个输入端子电压恒为零，这一特性被称为虚短。

进行含有理想运放的电路分析时，通常就会采用 虚短 + 虚断 的分析方法，下面采用一个例子来展示分析过程：

上面电路，由虚断的特性可知 \(i_+ = i_- = 0\)，表明负反馈电阻网络向左的端子电流为零，即两个电阻流过的电流相同，由于总的电压为 \(v_o\)，根据串联分压原理，每个电阻所分得的电压 \(v_- = \frac{v_o}{2}\)。又由虚短定义可知 \(v_+ = v_- = v_i\)，联立两个方程可以求得 \(v_o = 2 \cdot v_i\)。

对于理想运放而言，我们认为输入端的电流为零。但是当理想运放与外部电路连接时，正常工作的运放输出端会产生电流 \(i_o\)。此时，由于输入电流为零，而输出的电流不等于零，看起来并不满足 KCL 定律。

但是事实并不是这样，由于之前为了简化电路省略了运放模型的 \(V_{cc+}\) 与 \(V_{cc-}\) 两个电源端子，在运放正常工作的时候，电源会对运放供电，此时电源端子上会有电流，这样运放电路当中总输入电流除了 \(i_+\) 和 \(i_-\)，还有 \(i_1\) 和 \(i_2\)。

将电源端线纳入考虑之后，运放的电流就满足了 KCL 定律 \(i_o = i_+ i_- + i_1 + i_2\)。

注意：本门课程后续出现的运放，如果没有特别说明，均是指理想运放。

对于复杂的含有运放的电路，通常采用节点分析法会较为简单。

例题 1

以下面的电路为例，利用节点分析法证明其为加法器电路：

根据理想运放的虚断特性，其两个输入端电流为零，这意味着运放的输入端对两个电阻节点的方程列写不造成影响，而运放的输出端结点在正常工作时会产生输出电流，其值无法基于欧姆定律并利用节点电压进行表示，所以不列写运放输出端节点的节点方程。此外，由于理想运放具有虚短特性，这里需要补充虚短路方程。

首先，将 \(v_1\)、\(v_2\)、\(v_o\) 视为已知电压，然后分别对图中的两组并联电阻与运放的连接点列写节点方程：

\[ (\frac{1}{R} + \frac{1}{R}) \cdot v_+ - \frac{v_1}{R} - \frac{v_2}{R} = 0 \\ (\frac{1}{R} + \frac{1}{R}) \cdot v_- - \frac{v_o}{R} = 0 \\ \]

然后再补充一个虚短方程 \(v_+ = v_-\)，由这 3 个方程联立可以推导出 \(v_o = v_1 + v_2\)，从而表明该电路为一个加法器电路。

例题 2

证明当下面电路中的 \(\frac{R_4}{R_3} = \frac{R_5}{R_6}\) 时，输出电压为 \(v_o = k \cdot (v_2-v_1)\)，其中 \(k\) 为常数。

这里同样采用节点分析法来完成证明，上面电路当中 \(v_1\) 和 \(v_2\) 分别通过电阻 \(R_1\) 和 \(R_2\) 与运放的同向输入端相连，由理想运放的虚断特性可知，两个电阻所在支路是没有电流流过的，因此两个运放的同向输入端是不需要列写节点方程的。

对于剩下的 \(a\)、\(b\)、\(c\)、\(v_o\) 四个节点，其中 \(b\) 和 \(v_o\) 是运放的输出端节点，对于这两个节点不需要列写节点方程。所以，这里仅仅只需要对 \(a\) 和 \(c\) 这两个节点列写节点方程即可。

首先，对 \(a\) 节点列写节点方程：\((\frac{1}{R_5} + \frac{1}{R_6}) \cdot v_a - \frac{v_b}{R_6} = 0\)。接下来对 \(c\) 节点列写节点方程：\((\frac{1}{R_3} + \frac{1}{R_4}) \cdot v_c - \frac{v_b}{R_3} - \frac{v_o}{R_4} = 0\) ，最后再补充两个虚短方程 \(v_a = v_1\)，\(v_c = v_2\)。由这 4 个方程就可以推导出，如果满足 \(\frac{R_4}{R_3} = \frac{R_5}{R_6}\) 时，\(v_o = (1+\frac{R_4}{R_3}) \cdot (v_2-v_1) = k \cdot (v_2-v_1)\)。

运放的应用

本小节通过一些例子，展示如何使用运放来实现不同的功能。

反相与同相放大器

下图所示电路，输入电压 \(v_i\) 和输出电压 \(v_o\) 分别通过两个电阻与运放的反向输入端相连，下面就采用理想运放电路的虚短路特性，求解 \(v_i\) 与 \(v_o\) 的关系。

首先，由虚短路特性可以知道 \(v_+ = v_- = 0\)，由于上图运放的同向输入端接地，所以 \(v_+ = 0\) 且 \(v_- = 0\)，这个特点被称为虚地，即运放的反向输入端好像也与地相连一样。又由虚断可知，运放的两个输入端子的电流为零，即 \(i_- = 0\)，如果对该节点列写 KCL 方程，那么两个电阻的电流 \(i_1\) 和 \(i_2\) 必然会相等 \(i_1 = i_2\)。其中，\(i_1\) 可以表示为 \(\frac{v_i - v_-}{R_1}\)，\(i_2\) 则可以表示为 \(\frac{v_- - v_o}{R_f}\)，于是得到上图右侧的等式，由该结果可以看出：电路输出 \(v_o\) 等于输入 \(v_i\) 乘以一个比例系数，其中 \(R_f\) 与 \(R_1\) 的比值就是输入电压的放大倍数，通过改变两个电阻的阻值而实现不同的放大倍数，而前面的负号表示输入与输出电压的极性相反，因此该电路被称为反向放大器。

上面电路当中，输入电压 \(v_i\) 与运放的同向输入端相连，输出电压 \(v_o\) 通过负反馈电路与运放的反向输入端相连。同样由虚短路特性可以知道 \(v_+ = v_- = v_i\)，而由虚断可知运放两个输入端子的电流为零，因此 \(i_- = 0\)，如果对该节点列写 KCL 方程流过这两个电阻的电流必然相等 \(i_1 = I_2\)。其中，\(i_1\) 可以表示为 \(\frac{v_o - v_-}{R_f}\)，电流 \(i_2\) 可以表示为 \(\frac{v_- 0}{R_1}\)，于是得到上图等式右边的结果。由这个结果可以了解，电路的输出等于电路的输入前面乘以一个系数，该系数即是放大倍数，可以通过改变电阻阻值实现不同放大倍数。此外，由于这个系数前面为正号，表示输入输出电压的极性相同，因此该电路被称为同向放大器。

由这两个例子可以看出来，当将输入电压分别与运放的反向输入端和同向输入端相连的时候，可以分别实现反向与同向放大。
