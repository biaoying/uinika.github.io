现代电路分析涉及的理论与原理

发表于 2020-08-15 更新于 2020-08-16 分类于 Electronics 阅读次数： 本文字数： 13k 阅读时长 ≈ 12 分钟

《电路分析》主要介绍电路理论的基本知识，也是后续模拟、数字、高频电路的先导基础内容，其主要用于分析电路当中的电磁现象，并采用电流、电压、电量、磁通等物流量描述其过程，从而研究电路的基本规律及其分析方法。其理论与工程实际紧密相连，需要注意电路模型与实际电路的差别与联系，以及测量值与计算值的差别；由于每个电路模型分别具有特定的工作条件，根据工作场景的不同，还会选取不同的电路模型。

本篇文章主要包含了电路模型与电路定律、电阻电路分析、动态电路的时域分析、正弦稳态电路分析等方面的概念，由于电路分析方法本质是数学在电路当中的应用，因此学习过程需要注重将电路分析方法与数学知识相结合，例如向量法当中的复数。全文撰写过程当中，参考并借鉴了国外大学普遍采用的《Electric Circuits》第 10 版教材。

电路的基本概念和分析方法

集总参数电路

电荷在电场的作用下定向移动形成电流，电流所通过的路径则称为电路。描述电路特征的基本物理量有电流、电压、电荷、磁通，通常使用 \(i\)、\(u\)、\(q\)、\(\varPhi\) 表示。

当电路的几何尺寸远小于电路当中的信号波长时，电路周围的电磁辐射效应可以忽略不计，此时电流的能量完全分布在金属导线与电路元件上面，这样的电路称为集总参数电路。反之，当电路几何尺寸与信号波长差距较大，电磁辐射效应明显，电路当中各处的电流不一致，此时电流的能量在空间当中离散分布，这样的电路则称为分布参数电路。

例如：假设一个音频放大电路的最高工作频率 f ＝ 25kHz，其波长：

\[ \lambda = \frac{c}{f} = \frac{3 \times 10^8 m/s}{25 \times 10^3 m/s} = 12km \]

这种情况下，音响设备和音频放大电路的几何尺寸远小于该波长，因此可以被视为集总参数电路。如果这里将工作信号的频率无限提高，则波长会迅速减小。当波长减小至比电路尺寸要小得多时，就需要改用分布参数电路来进行分析。

电流、电压、电功率

电流

电荷（单位为库仑C）的定向运动形成电流，其大小值等于单位时间内通过导体横截面的电荷，采用的国际单位为安培A。

\[ i = \frac{dq}{dt} \]

电路分析时，习惯上将正电荷流动的方向定义为电流方向，然而事实上，电流是电子的运动所形成。而电流的参考方向则是假定的电流方向，如果假定方向和实际方向一致，则电流量值为正，否则为负。

例如下图的二端元件当中，每秒有 \(2C\) 正电荷由a点移动至b点，当规定电流参考方向由a点指向b点时，则该电流 \(i=2A\)：

反之，如果规定电流参考方向由b点指向a点时，则电流 \(i=-2A\)：

如果采用双下标方式表达电流的参考方向，可以写为\(i_{ab}=2A\) 或者 \(i_{ba}=-2A\)。

电压

电场力将单位正电荷从电场中的a点移动到b点所做的功，称为a与b两点之间的电压，可以用数学公式表达如下：

\[ u=\frac{dw}{dq} \]

习惯上，在电场力的作用下，如果正电荷从a点移动至b点失去电场能量，则a点为正极b点为负极。电压分析时，同样引入了参考方向的概念，电压的参考方向采用一对正负号进行表示。

关联参考方向

电流参考方向的流入端为电压参考方向的正极性，称为关联参考方向；反之则称为非关联参考方向。

下面的电路图里，\(I_1、U_1\) 和 \(I_2、U_2\) 以及 \(I_4、U_4\) 为关联参考方向，\(I_3、U_3\)、\(I_5、U_5\) 为非关联参考方向。

下图当中，二端元件电流、电压的参考方向，(a)和(b)为关联参数方向，(c)和(d)为非关联参考方向。

电功率

单位时间内某一段电路消耗或者提供的能量称为功率，其数学表达式如下所示：

\[ p = \frac{dw}{dt} = \frac{w}{q} \cdot \frac{q}{t} = u \cdot i \]

电压与电流均为代数量，因此作为其乘积的功率也是代数量。而功率正负号的物理意义在于：如果电路中的电压、电流参考方向任意假定，则正负号所代表的物理意义没有共性可言。而在关联参考方向下，功率为正号+时表示吸收功率，为负号-时表示发出功率。

上面的电路图当中，流入 \(1Ω\) 电阻的电流为 \(1A\)，这个电流由 \(1V\) 的电压源提供，则此时流入该电压源的电流就为 \(-1A\)。根据功率的定义，\(1Ω\) 电阻在电压、电流处于关联参考方向的时候 \(P_{1Ω} = ui = 1W\)；而 \(1V\) 电压源在电压、电流处于关联参考方向的时候 \(P_{1V} = ui = -1W\)。

基尔霍夫定律

基尔霍夫定律由德国物理学家 G.R.基尔霍夫在 1845 年提出，分为基尔霍夫电流定律（KCL，Kirchhoff’s Current Law），基尔霍夫电压定律（KVL，Kirchhoff’s Voltage Law）。

  • 支路（Branch）：一个二端元件就可以视为一条支路，其电流和电压分别称为支路电流和支路电压，支路数通常使用字母b表示，上图当中一共拥有 6 条支路。
  • 结点（Node）：两条以及两条以上支路的连接点称为结点，节点数通常使用n表示。上图当中，a、b、c点是节点，d点和e点之间由理想导线连接为 1 个节点，因而该电路一共拥有 4 个节点。
  • 回路（Loop）：由支路所组成的闭合路径，上图当中，包括{1,2}、{1,3,4}、{1,3,5,6}、{2,3,4}、{2,3,5,6}、{4,5,6}在内，一共拥有 6 个回路。
  • 网孔（Mesh）：内部不含有支路的回路，网孔数通常使用m表示，上图包含了 3 个网孔。
  • 广义结点：将电路中的一部分围成一个封闭的平面（高斯面），此封闭平面称为广义节点。

基尔霍夫电流定律 KCL

对于集总参数电路中的任意一个节点，在任意时刻流出该节点的电流代数和等于零，可以记为如下表达式：

\[ \sum_{k=1}^n i_k = 0 \]

根据定义，可以规定流出节点的支路电流取正号+，流入节点的支路电流取负号-。

\[ -i_1 + i_2 - i_3 + i_4 - i_5 = 0 \]

结点只是理想导线的汇合点，即不可能积累电荷，也不可能消灭电荷。根据电荷守恒，结点处的 \(\frac{dq}{dt}\) 必须等于零。

【例题】：已知 \(i_1=1A\)、\(i_7=-2A\)、\(i_{10}=3A\)，试求出下面电路当中 \(i_4\) 的值。

  • 根据第 ① 个节点列写 KCL 方程：\(i_{10} - i_{11} = 0\)
  • 根据第 ② 个节点列写 KCL 方程：\(-i_5 + i_{11} - i_7 = 0\)
  • 根据第 ③ 个节点列写 KCL 方程：\(i_5 - i_2 = 0\)
  • 根据第 ④ 个节点列写 KCL 方程：\(-i_1 + i_4 + i_2 = 0\)

【求解】综上所述，可以求得表达式 \(i_{10} - i_7 + i_4 - i_1 = 3A + 2A + i_4 - 1A = 0\)，进而可以推导出 \(i_4 = -4A\)。

基尔霍夫电压定律 KVL

对于集总参数电路当中的任意一个回路，在任意时刻沿闭合回路绕行一周，各支路电压的代数和等于零。

\[ \sum_{k=1}^n u_k = 0 \]

电荷沿着回路绕行一圈回到原点，其电势能不变，KVL 是电路中能量守恒的体现。

\[ u_1 + u_2 - u_3 = 0 \]

【例题】：已知 \(u_1=1V\)、\(u_2=-5V\)，试求解下面电路当中支路电压 \(u_3\) 的值。

【求解】应用 KVL 定律可以得到 \(u_1 + u_2 - u_3 = 1V - 5V - u_3 = 0\)，从而可以进一步求得 \(u_3 = -4V\)。

【例题】：已知 \(u_2=2V\)、\(u_4=-4V\)、\(u_5=-5v\)，试求解下图中支路电压 \(u_1\) 的值。

【求解】首先，对 \(\{ u_3, u_4, u_5 \}\) 回路列写 KVL 方程 \(u_4 - u_5 - u_3 = -4V + 5V - u_3 = 0\)，从而可以求得 \(u_3 = 1V\)。

然后，对 \(\{ u_1, u_3, u_2 \}\) 回路列写 KVL 方程 \(u_1 + u_3 - u_2 = u_1 + 1V - 2V = 0\)，最终可以求得 \(u_1 = 1V\)。

电阻

电阻器件在实际电路当中使用极为广泛，电路分析过程中，电阻元件是根据实际电子元器件特性抽象而来的一种理想化电路模型。例如下图是将三极管视为电阻元件时的特性曲线：

而下图当中，黑色的线条描述的是实际电阻元件，而红色的线条则是将二极管视为电阻元件时的特性曲线：

如果一个二端元件在任意时刻的电压 \(u\) 与电流 \(i\) 之间的关系，由 U-I 平面上的一条曲线来确定，则此二端元件称为二端电阻元件，其数学表达式为 \(f(u, i)=0\)，并采用下面的电路符号进行表示：

电阻元件根据自身的不同特点，可以划分为不同的类型：

  • 线性电阻与非线性电阻：特性曲线为通过坐标原点直线的电阻称为线性电阻，否则称为非线性电阻。
  • 时变电阻与时不变电阻：特性曲线随时间变化的电阻称为时变电阻，否则称为时不变电阻。

线性非时变电阻的电压电流关系特性曲线是一条过原点，并且不随时间变化的直线。其电压电流关系（VCR，Voltage Current Relation）可以表达为 \(u = Ri\)。

习惯上使用电阻参数 \(R\) 来表示上图当中直线的斜率，其国际单位为欧姆Ω：

\[ R = \frac{u}{i} \]

除此之外，电导参数 \(G\) 也可以用来表达这条直线的斜率，其国际单位为西门子S：

\[ G = \frac{i}{u} \]

根据上述定义的电阻和电导参数以及功率的定义，可以得到关联参考方向下，电阻吸收的功率：

\[ p = ui = Ri^2 = \frac{u^2}{R} = Gu^2 = \frac{i^2}{G} \]

当正电阻 \(R>0\) 时有 \(p \geq 0\) 表明总是吸收功率，而当负电阻 \(R<0\) 时有 \(p \leq 0\) 表明总是发出功率。

这里出现的负电阻是过 U-I 平面二四象限的一条直线，表示对外发出功率，也称为有源电阻。

注意：虽然实际电阻总是耗能元件，但实际电路当中的一些元件的某个端口可能会呈现负电阻的特性，比如由三极管或运放构成的一些电路，所以才会在电路分析当中引入负电阻的模型。

独立电源

下图当中，红色的曲线绘制的是 U-I 平面上电池的端口曲线，由于实际电池具有内阻，因此该特征曲线伴随电流的增大而略有下降。当电流 \(i\) 大至一定程度的时候，电压 \(U_s\) 会陡降，这个电流的拐点就称为实际电源的最大工作电流 \(I_{max}\)。

独立电压源

无论二端元件的电流为何值，其电压总是保持常量 \(U_s\) 或按指定的时间函数 \(u_s(t)\) 变化，则该二端元件就称为独立电压源，简称为电压源。其电路符号和特征曲线如下图所示：

独立电压源的元件 VCR 电压电流关系如下所示：

\[ x = \begin{cases} 端口电压恒定，u = u_s(t) = U_s \\ 端口电流 i 为任意值，由外电路确定。 \end{cases} \]

【例题】求出下面电路图当中，流过 \(1V\) 电压源的电流 \(i\) 的值？

【求解】下图左侧 \(1V\) 电压源连接了一个 \(1Ω\) 的电阻，其两端电压降为 \(1V\)，根据欧姆定律流过该电阻的电流为 \(1A\)；根据 KCL 定律，流过该电压源的电流 \(i\) 应为 \(-1A\)。右侧的电路当中，将 \(1Ω\) 的电阻更换为 \(10Ω\)，则流过该电阻的电流为 \(0.1A\)，相应的流过该电压源的电流 \(i\) 应为 \(-0.1A\)。

【例题】求取下面电路图里面，每个电压源吸收的功率分别为多少？

【求解】关联参考方向下，电压源吸收的功率 \(P\) 等于 \(p = u_si\)，其结果为正+时表示吸收功率，为负-时表示发出功率。

\[ \begin{cases} p_{1v} = ui = 1A \times 1V = 1W \\ p_{2v} = ui = -1A \times 2V = -2W \end{cases} \]

独立电流源

无论二端元件的电压为何值，其电流保持常量 \(I_S\) 或者按照指定的时间函数 \(i_s(t)\) 变化，则该二端元件称为独立电流源，简称为电流源。

独立电流源的元件 VCR 电压电流关系如下所示：

\[ x = \begin{cases} 端口电流恒定，i = i_s(t) = I_s \\ 端口电压 u 为任意值，由外电路确定。 \end{cases} \]

【例题】求出下面电路图当中，流过 \(1A\) 电流源的电压 \(u\) 的值？

【求解】上图中的 \(1A\) 电流源连接了一个 \(1Ω\) 的电阻，根据欧姆定律该电阻两端的电压为 \(1V\)；根据 KVL 定律，流过电压源两端的电压 \(u\) 应为 \(-1V\)。如果此时将图中 \(1Ω\) 的电阻更换为 \(100Ω\)，此时电流依然为 \(1A\)，但是电阻两端的电压降为 \(100V\)，则相应的 \(1A\) 电流源两端的电压为 \(-100V\)。由此可见，电流源的 \(1A\) 电流始终恒定不变，而两端的电压降随外电路的变化而变化。

【例题】求出下面电路图当中，电流源和电压源吸收的功率 \(p_{1A}\) 和 \(p_{1V}\) 分别为多少？

【求解】根据回路的 KVL 方程，\(1A\) 两端的电压降为 \(-1V\)，由于 \(1Ω\) 电阻两端的电压降为 \(1V\)，则流过该电阻的电流为 \(1A\)。根据电阻上面的节点列写 KCL 方程，可以求出流过 \(1V\) 电压源的电流为 \(0A\)。

\[ \begin{cases} p_{1A} = ui = -1V \times 1A = -1W \\ p_{2v} = ui = 1V \times 0A = -2W \end{cases} \]

元件/拓扑约束与 2b 方程

由不同元件构成的电路所表现出的特性不同，而相同元件不同的连接关系也会呈现出不同的特性。因此，电路包含有元件约束、连接/拓扑约束两类约束关系。

  • 元件约束：由元件的电压电流关系（VCR，Voltage Current Relation）进行描述，反应了电子元件对其所在支路电压、电流所起的约束关系。
  • 拓扑约束：通过基尔霍夫电流和电压定律，分别反应电路结构和连接特性对支路电流、电压的约束关系。

应用元件和拓扑约束建立的电路方程称为 2b方程，即对于含有b条支路的集总参数电路，以b条支路电压与b条支路电流作为变量，可以建立2b个方程。2b方程是电路分析最为基本的方法，也是后续其它电路分析方法的重要依据。

对于含有b条支路n个结点的电路，可以列出如下线性无关的方程：

【例题】下面电路图具备 5 条支路和 4 个结点，请列写出由 10 个变量组成的 10 个方程，以求解其中的任意一个变量。

【求解】首先，针对结点 ①、②、③ 列写 3 个 KCL 方程：

\[ \begin{cases} i_1 + i_4 = 0 \\ -i_1 + i_2 + i_3 = 0 \\ -i_2 + i_5 = 0 \end{cases} \]

然后，针对 2 个网孔的列写 2 个 KVL 方程：

\[ \begin{cases} u_1 + u_3 - u_4 = 0 \\ u_2 + u_5 - u_3 = 0 \end{cases} \]

最后，列出 b 条支路的 5 个 VCR 方程：

\[ \begin{cases} u_1 = R_1i_1 \\ u_2 = R_2i_2 \\ u_3 = R_3i_3 \\ u_4 = u_{s1} \\ u_5 = u_{s2} \\ \end{cases} \]

支路电流法

对于仅仅由独立电压源和线性二端电阻构成的电路，可以将欧姆定律 \(u=Ri\) 代入 KVL 方程，从而消除全部电阻的支路电压，转换为以支路电流为变量的 KVL 方程。叠加原有的 KCL 方程，最终得到以b个支路电流为变量的b个线性无关的方程组，称为支路电流方程。

【例题】下面电路图具备 5 条支路和 4 个结点，请列写出由 10 个变量组成的 10 个方程，以求解其中的任意一个变量。

【求解】以b条支路的电流作为变量列写方程，其依据依然是上一小节中提到的元件和拓扑两类约束关系。

首先，同样针对结点 ①、②、③ 列写 3 个 KCL 方程：

\[ \begin{cases} i_1 + i_4 = 0 \\ -i_1 + i_2 + i_3 = 0 \\ -i_2 + i_5 = 0 \end{cases} \]

然后，同样针对 2 个网孔的列写 2 个 KVL 方程；通过代入 VCR 方程，将上式中的 5 个电压变量替换为 5 个电流变量：

\[ \begin{cases} u_1 + u_3 - u_4 \implies R_1i_1 + R_3i_3 - u_{s1} = 0 \\ u_2 + u_5 - u_3 \implies R_2i_2 + u_{s2} - R_3i_3 = 0 \end{cases} \]

将上述方程整理以后可以得到：

\[ \begin{cases} R_1i_1 + R_3i_3 = u_{s1} \\ R_2i_2 - R_3i_3 = -u_{s2} \end{cases} \]

联立上述 5 个方程，即可以求解 5 个支路电流。如果再需要求解某个支路电压变量，可以将其代入欧姆定律，即可以求解出相应的电压变量。

支路电压法

对于由线性二端电阻和独立电流源构成的电路，可以采用支路电压作为变量来建立电路方程。

在 2b 方程的基础上，将电阻元件的 VCR 方程 \(i=Gu\) 代入 KCL 方程，将支路电流转换为支路电压，可以得到以(n-1)个以支路电压作为变量的 KCL 方程，叠加原有的(b-n+1)个 KVL 方程，可以构成b个以支路电压作为变量的电路方程，这组方程称为支路电压法方程。

【例题】

【求解】根据元件和拓扑两类约束，可以列写出 3 个线性无关的 KVL 方程 2 个线性无关的 KCL 方程。列写 KCL 方程的时候，不以 \(i_1\) 到 \(i_3\) 的支路电流作为变量进行列写，而是直接以电压作为变量进行列写，即代入欧姆定律方程，从而得到带有电压变量的 KCL 方程，并最终得到 5 个支路电压作为变量的 5 个方程：

\[ \begin{cases} u_1 + u_{is1} = 0 \\ -u_1 + u_3 + u_2 = 0 \\ -u_2 + u_{is2} = 0 \end{cases} \]

\[ \begin{cases} G_1u_1 + G_3u_3 = i_{s1} \\ G_2u_2 - G_3u_3 = -i_{s2} \end{cases} \]

联立上述方程，即可求解出全部的电压变量。如果当前需要求解的是支路电流，则相应的代入欧姆定律即可。

网络等效简化电路分析

线性无源单口网络等效

线性无源单口网络是指不含有独立源的单口网络，可以由电阻和受控源或者纯电阻组成。对于完全由电阻组成的无源单口网络，根据电阻的连接方式分为串联、并联、电桥等几种形式。

线性电阻串联

电阻串联是指每个电阻元件依次首尾相接，流过每个电阻的电流相等，下图就是一个由串联电阻组成的无源单口网络：

由于端口电压 \(U\) 等于每个串联电阻上的电压之和，对于该单口网络可以列写其 VCR 方程：

\[ u = u_1 + u_2 + u_3 + ... + u_n = R_1i_1 + R_2i_2 + R_3i_3 + ... + R_ni_n = (R_1 + R_2 + R_3 + ... + R_n)i \]

再看下面电路图当中的这个由 1 个电阻构成的单口网络：

其端口的 VCR 方程非常容易列写：

\[ u = Ri \]

当满足 \(R\) 等于所有串联电阻之和的时候，上述两个电路的 VCR 表达式完全相同，说明两个端口网络是等效的。由此可以得出结论：n个线性电阻串联的单口网络，其端口特性等于一个线性二端电阻，其电阻值为各串联电阻之和。

\[ R = \sum_{k=1}^n R_k \]

线性电阻并联

电阻并联是指每个电阻元件分别首尾相接，每个电阻两端的电压分别相等，下图为电阻并联情况下的无源端口网络：

端口电流 \(i\) 等于每个并联电阻流经的电流之和，由此可以列写 VCR 方程：

\[ i = i_1 + i_2 + i_3 + ... + i_n = G_1u_1 + G_2u_2 + G_3u_3 + ... + G_nu_n = (G_1 + G_2 + G_3 + ... + G_n)u \]

再看下面电路图当中的这个由 1 个电阻构成的单口网络：

其端口的 VCR 方程非常容易列写：

\[ i = Gu \]

比较上述两个电路的 VCR 关系，可以看出满足电导 \(G\) 等于所有并联电导之和的时候，两个 VCR 表达式完全相同，说明两个端口网络是等效的。由此可以得出结论：n个线性电阻并联的单口网络，其端口特性等效于一个线性二端电阻，其电导值为各并联电阻的电导之和。

\[ G = \sum_{k=1}^n G_k \]

平衡电桥电路等效

电阻之间的关系，除了串联和并联之外，还有一种情况是电桥电路。下面的电桥电路中，\(R_1\)、\(R_2\)、\(R_3\) 称为桥臂，而 \(R_5\) 称为对角线电阻。

假设对角线支路 \(R5\) 不存在， 则电路变换为如下所示：

根据电阻的分压公式，可以得到如下等式：

\[ u_{cb} = \frac{R_3}{R_1 + R_3} u_{ab} \]

\[ u_{db} = \frac{R_4}{R_2 + R_4} u_{ab} \]

当 \(R_1 \times R_4 = R_2 \times R_3\) 时，将上面的两个等式相除可以得到：

\[ \frac{u_{cb}}{u_{db}} = \frac{R_3}{R_1 + R_3} \times \frac{R_2 + R_4}{R_4} = 1 \implies u_{cb} = u_{db} \]

此时，如果将 \(b\) 点视为零参考点，则得到C点的电位等于D点的电位 \(v_c = v_d\)。

因此无论c和d两点之间是开路、短路还是接入任意电阻，都不会影响其它支路变量，这种情况就称为电桥平衡，电桥平衡的条件就是 \(R_1 \times R_4 = R_2 \times R_3\)。

【求解】上述电路当中，不相临两个桥臂电阻的乘积等于另外两个桥臂电阻的乘积，\(12Ω \times 3Ω = 6Ω \times 6Ω\)，所以这是一个电桥平衡电路。

将对角线支路视为开路，则 \(12Ω\) 和 \(6Ω\)、\(3Ω\) 和 \(6Ω\) 电阻分别串联，然后两部分再并联，可以求得等效电阻为：

\[ R_{ab} = \frac{(12Ω+6Ω) \times (3Ω+6Ω)}{(12Ω+6Ω) + (3Ω+6Ω)} = 6Ω \]

将对角线支路视为短路，则 \(12Ω\) 和 \(6Ω\)、\(3Ω\) 和 \(6Ω\) 电阻分别并联，然后两部分再串联，可以求得等效电阻为：

\[ R_{ab} = \frac{12Ω \times 6Ω}{12Ω + 6Ω} + \frac{3Ω \times 6Ω}{3Ω + 6Ω} = 6Ω \]

含源线性电阻单口等效

仅有两个端钮与外部电路相连接的网络，称为二端网络。如果只强调二端网络的端口特性，而不关心网络内部的情况，则称二端网络为单口网络，通常简称单口。

单口电阻的特性由端口电压电流关系 VCR 来进行表征（表现为 U-I 平面上的一条曲线）。当两个单口网络的 VCR 关系完全相同时，则称这两个单口相互等效。

根据单口 VCR 方程得到的最简电路，称为单口等效电路。电路分析计算过程当中，可以使用等效电路代替原来的单口网络，以便简化分析计算过程。

求解单口等效电路的数学方法是求解单口网络的 VCR 关系，具体方法是通过外加电源法，让单口网络中的电压变量 \(u\) 或者电流变量 \(i\) 已知，然后用已知的量求解未知的量，从而得到单口的 VCR 方程。

如果外加电压源，就表示当前电压是已知的，由此就可以列写出如下数学表达式：

\[ i = f(u) = G_0u - i_{sc} \]

上面的数学表达式是一个 KCL 方程，可以绘制出下面两条支路并联的电路：

如果外加电流源，就表示当前电流是已知的，同样可以由此列写如下数学表达式：

\[ u = f(i) = R_0i - u_{oc} \]

上面的数学表达式是一个 KVL 方程，可以绘制出下面的串联电路：

即然外加电压和电流源以后的方程，表征的都是同一个单口网络，因此两个方程从数学角度而言必然是相同的，对应的两种形式等效电路之间显然也是等效的。

对这两个形式进行数学变换，分别用电压来表征电流，以及用电流来表征电压：

\[ u = R_0i + u_{oc} \implies i = \frac{u}{R_0} - \frac{u_{oc}}{R_0} \]

\[ i = G_0u + i_{sc} \implies u = \frac{i}{G_0} - \frac{i_{sc}}{G_0} \]

由上面的等式，可以进一步获得这 3 个参量的数学表达形式：

\[ \begin{cases} u_{oc} = R_0i_{sc} \\ i_{sc} = u_{oc}G_0 \\ R_0 = \frac{1}{G_0} \end{cases} \]
